## 핵심 상수 매핑

```
n=6          : 시스템 모듈(냉방/난방/환기/제습/가습/공기청정)
tau=4        : 기본 사이클(증발/압축/응축/팽창)
sopfr=5      : 냉매 회로 단(흡입→압축→토출→응축→팽창)
n/phi=3      : 3-way 댐퍼 위치(외기/환기/배기), 3상 인버터
phi=2        : 이중 열교환기 패스, 듀얼 코어
sigma=12     : 제어 단계(센싱→판단→PWM→밸브→팬→...→피드백)
sigma-phi=10 : 존(zone) 분할 최대
J_2=24       : 시간당 환기 횟수 기준(ACH×시간), 최저소음 dB
```

import math
sigma = 12
tau = 4
phi = 2

total = 6
