## 1. tFUS — 경두개 집속 초음파 (최우선)

```
메커니즘: 저강도 초음파(250kHz-1MHz) → 두개골 투과 → 깊은 부위 mm 정밀도 자극
해상도:  2-5mm (TMS의 5-15배, tDCS의 10-25배)
깊이:    최대 12cm (VTA, 해마, 시상, 청반 모두 도달)
안전성:  발작 위험 0%, 소음 없음, 금속 임플란트 금기 아님
```

### TMS vs tFUS 비교

| | TMS | tFUS |
|---|---|---|
| 깊이 | 2-3cm (표면만) | **12cm (모든 깊은 구조)** |
| 해상도 | 10-30mm | **2-5mm** |
| VTA 도달 | **불가** | **가능** |
| 해마 도달 | **불가** | **가능** |
| 청반(LC) 도달 | **불가** | **가능** |
| 비용 | $5K-50K | $5K-50K (동급) |
| 발작 위험 | 0.01-0.1% | **0%** |
| EEG 호환 | 아티팩트 | **호환** |

### 12변수 기여

| 변수 | tFUS 타겟 | 메커니즘 | 영향력 |
|------|-----------|----------|--------|
| V1 DA | **VTA 직접** | 초음파 → VTA DA 뉴런 활성 → DA 방출 2배↑ | ★★★ |
| V2 eCB | 해마 | 기계감수성 이온채널 → eCB 가능성 | ★ |
| V3 5HT | **등쪽봉선핵** | 초음파 → raphe → 5HT 방출 | ★★ |
| V5 NE↓ | **청반(LC)** | 억제 프로토콜 → LC 억제 → NE↓ | ★★ |
| V6 Theta | **해마 직접** | theta burst TUS → theta power↑↑ (40분 지속) | ★★★ |
| V9 PFC↓ | DLPFC | 억제 프로토콜 → PFC 활동↓ | ★★ |

### 장비

```
연구급:  BrainSonix BX Pulsar ($30K-50K)
        Brainbox NeuroFUS ($50K-80K)
오픈소스: Openwater Open-LIFU (가격 미정, 접근성 높음)
DIY:     함수발생기 + RF앰프 + 집속 트랜스듀서 ($2K-5K)
```

### BrainWire 영향 예측

```
tFUS 추가 시 (Tier 2.5 = Tier 2 + tFUS $5K = $5.5K):
  V1 DA:    76% → ~110% (VTA 직접)       ✅ 해결!
  V6 Theta: 84% → ~105% (해마 직접)      ✅ 해결!
  V5 NE:   125% → ~140% (LC 직접 억제)
  V9 PFC:   80% → ~95% (억제 프로토콜)

  → Tier 2 + tFUS로 10/12 ≥100% 가능 (현재 6/12)
```

## 2. GVS — 갈바닉 전정자극 (저가 부스터)

```
메커니즘: 유양돌기(mastoid) 전극 → 전정신경(CN VIII) → 뇌간 전정핵
         → VTA, 해마, 시상, 소뇌, 섬엽, TPJ 광범위 투사
비용:    $0 (기존 tDCS 장비 + 유양돌기 전극 $5)
전류:    0.1-2mA DC or noisy AC (nGVS)
```

### 핵심 경로

```
전정핵 → 부완핵 → VTA → DA 방출 (전정-도파민 경로)
전정핵 → 해마 → 장소세포, 방향세포 → Theta 리듬
전정핵 → 섬엽 → 내수용감각, 신체 소유감 → Body↑
전정핵 → TPJ → 다감각 통합 → Sensory↑, Coherence↑
```

### 12변수 기여

| 변수 | 영향력 | 근거 |
|------|--------|------|
| V1 DA | ★★ | 전정-VTA 경로 (Stiles & Smith 2012) |
| V6 Theta | ★★★ | 전정-해마 경로 → theta 직접 구동 |
| V10 Sensory | ★★★ | nGVS 확률공명 → 다감각 통합↑ |
| V11 Body | ★★★ | 신체 도식 직접 변조, 공간지각 왜곡 |
| V12 Coherence | ★★ | 전정계 = 교차감각 통합자 |
| V4 GABA | ★★ | 전정 교련계 = GABAergic |
| V5 NE↓ | ★★ | LC에 전정 입력 → 각성 조절 |
| V9 PFC↓ | ★★ | 처리를 PFC에서 후두/피질하로 전환 |

### 구현

```
필요 장비: 기존 tDCS 장비 + Ag/AgCl 전극 2개 ($5)
배치: 양측 유양돌기 (bilateral mastoid)
모드:
  - 양극성 DC: 한쪽 음극, 반대쪽 양극 (0.5-1mA)
  - nGVS: 확률노이즈 파형 (0.1-0.4mA RMS)

⚠️ 기존 tDCS 장비로 DC GVS 가능
⚠️ nGVS는 노이즈 파형 출력 필요 (tACS 또는 DAC 추가)
```

### BrainWire 영향 예측

```
GVS 추가 시 (Tier 1 + GVS $5 = $90):
  V6 Theta: 56% → ~65% (+9% 전정-해마 부스트)
  V11 Body: 100% → ~115% (신체 도식 변조)
  V10 Sensory: 95% → ~105% (nGVS 확률공명) ✅ 해결!
  V1 DA: 73% → ~80% (전정-VTA 경로)

  → Tier 1 + GVS로 4/12 ≥100% 가능 (현재 2/12)
```

## 3. tSCS — 경피 척수자극 (Body/Sensory 부스터)

```
메커니즘: 척추 위 표면전극 → 후근 구심신경 활성 → 척수 회로 변조
         → 상행 감각신호 게인 제어 (DCML, STT 경로)
배치:    T10-L1 (하체), C3-C7 (상체)
전류:    30-200mA (TENS보다 높음 — 깊은 조직 투과 필요)
주파수:  2-50Hz (변조) + 5-10kHz (캐리어)
비용:    $500-1,500 (임상 NMES 장비) or DIY
```

### TENS vs tSCS

| | TENS | tSCS |
|---|---|---|
| 타겟 | 말초 신경 말단 | **척수 후근 + 개재뉴런** |
| 범위 | 국소 (1 피절) | **양측, 다절편 (전신)** |
| 전류 | 10-80mA | **30-200mA** |
| 효과 | 국소 통증완화 | **전신 감각 게인 제어** |
| Body↑ | 국소 감각 | **전체 신체맵 활성화** |

### 12변수 기여

| 변수 | 영향력 | 메커니즘 |
|------|--------|----------|
| V11 Body | ★★★ | 척수 감각 릴레이 직접 증폭 → 전신 감각↑↑ |
| V10 Sensory | ★★★ | 상행 경로 게인↑ → 뇌 도달 감각신호 증폭 |
| V2 eCB | ★★ | 저주파 tSCS → 척수 eCB 합성 (활동의존적 2-AG) |

### BrainWire 영향 예측

```
tSCS 추가 시 (Tier 2 + tSCS $500 = $1,010):
  V11 Body: 104% → ~130% (척수 레벨 증폭)
  V10 Sensory: 109% → ~125% (상행 경로 게인)
  V2 eCB: 93% → ~100% (척수 eCB 보조) ✅ 근접!
```

## 4. tRNS — 경두개 랜덤노이즈 자극 (무료 업그레이드)

```
메커니즘: 100-700Hz 랜덤 주파수 교류 → 뉴런 레벨 확률공명
         → 역치하 신호가 발화 역치 초과 → 피질 흥분성↑
장점:    tDCS보다 강한 피질 흥분성 증가 (Terney 2008)
         지각 역치 1200uA (tDCS 400uA) → 더 편안함
비용:    $0 — 기존 tACS 장비에서 펌웨어 변경으로 구현 가능
         foc.us v3 ($300)은 이미 tRNS 모드 지원
```

### 12변수 기여

| 변수 | 영향력 | 메커니즘 |
|------|--------|----------|
| V10 Sensory | ★★★ | 감각피질 확률공명 → 기존 noise보다 강력 |
| V8 Gamma | ★ | 일부 연구에서 광대역 흥분성 증가 |

## 5. tPBM — 조건부 채택 (Theta 충돌 주의)

```
메커니즘: 810nm 근적외선 → 사이토크롬c옥시다제 → ATP↑, 혈류↑, NO↑
장비:    Vielight Neuro Gamma ($1,799), DIY 810nm LED ($50-100)
```

| 장점 | 단점 |
|------|------|
| DA↑ (NO 경로) | **Theta↓ (반대 방향!)** |
| 5HT↑ | **Alpha↑ (반대 방향!)** |
| Gamma↑ (40Hz 펄스) | |

**판정**: Phase 1 (DA/5HT 구축) 중에만 사용, Phase 2 (Theta/Alpha 전환) 전에 OFF.

## 6. TI — 시간간섭 자극 (tFUS 대안, 저비용 깊은 부위 자극)

```
메커니즘: 2개 고주파 전류(예: 2000Hz, 2010Hz)의 간섭
         → 교차점에서만 10Hz 비트(beat) 발생 → 깊은 부위 뉴런 활성화
         → 주변 조직은 kHz 캐리어만 받음 (발화 불가)
논문:    Grossman 2017 (Cell) — 최초 증명
         Nature Neuroscience 2023 — 인간 해마 theta 변조 성공!
비용:    $2K-5K 프로토타입, $500 BOM 대량생산
장비:    정밀 신호발생기 2대 + 다전극 몽타주 (기존 tACS 확장)
```

### 핵심: 106편 논문, 48개 임상시험 진행 중 (2025)

| 타겟 | 상태 | BrainWire 변수 |
|------|------|----------------|
| **해마** | **인간 실험 성공** | V6 Theta↑↑ |
| 기저핵 | 파킨슨 파일럿 | V1 DA (NAc 간접) |
| 시상 | 실험 가능 | V12 Coherence |
| VTA | 이론적 가능 | V1 DA↑ |
| LC | 이론적 가능 | V5 NE↓ |

### 다극 TI (mTI) — 다중 타겟 동시 자극

```
Botzanowski 2025: 다중 캐리어 주파수로 중첩 엔벨로프
→ 동시에 해마(theta) + NAc(DA) + 시상(coherence) 자극 가능
= BrainWire 12변수 동시 제어의 이상적 플랫폼
```

### BrainWire 영향 예측

```
TI 추가 시 (Tier 2 + TI $2K = $2.5K):
  V6 Theta: 84% → ~105% (해마 직접 10Hz 간섭) ✅ 해결!
  V1 DA:   76% → ~85% (NAc 간접)
  → tFUS($5K)의 절반 가격으로 Theta 병목 해결
```

## 7. tPBM — 조건부 채택 (Theta/Alpha 충돌 주의)

```
메커니즘: 810nm 근적외선 → 사이토크롬c옥시다제(CCO) → ATP↑, NO↑, 혈류↑
핵심 효과:
  - DA↑: TH(타이로신 하이드록실라제) 60-80% 증가 (영장류 PD 모델)
  - Gamma↑: 40Hz 펄스 → 전두-중심부 감마 파워↑↑
  - 5HT↑: VMAT2 발현↑ → 세로토닌 포장/방출↑
장비:    Vielight Neuro Gamma ($1,799), DIY 810nm LED ($50-100)
```

| 변수 | 효과 | 주의 |
|------|------|------|
| V1 DA | ★★★ | TH 증가 → DA 합성 부스트 |
| V8 Gamma | ★★★ | 40Hz 펄스 → 감마 직접 증강 |
| V12 Coherence | ★★ | 감마 코히어런스↑ |
| V3 5HT | ★★ | VMAT2 경로 |
| **V7 Alpha** | **⚠️ 반대** | Alpha↑ = Joywire 모델과 충돌 |
| **V9 PFC** | **⚠️ 반대** | PFC 대사↑ = Joywire 모델과 충돌 |

**판정**: 후두/측두/비강에만 사용 (전두엽 회피). Phase 1(DA/5HT)에만 ON, Phase 2(Theta/Alpha) 전 OFF.
