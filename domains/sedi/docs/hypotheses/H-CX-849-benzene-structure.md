# H-CX-849: Benzene Structure

> **Hypothesis**: Benzene C₆H₆ is the archetype of aromaticity with P₁=6 carbons in a ring, P₁=6 hydrogens, and P₁=6 π electrons satisfying the Hückel rule 4n+2 at n=1.

## Grade: 🟩 CONFIRMED

## Results

### The Structure

```
Benzene: C₆H₆
  Carbon atoms in ring:  6 = P₁
  Hydrogen atoms:        6 = P₁
  Total atoms:          12 = σ

Hückel aromaticity rule: 4n+2 π electrons
  For n=1: 4(1)+2 = 6 = P₁ π electrons
  Benzene is the FIRST aromatic compound (n=1)

The first aromatic ring:
  P₁ π electrons delocalized over P₁ carbon atoms
  Total atom count = σ(P₁) = σ(6) = 12
```

### n=6 Constants

```
σ = 12, τ = 4, φ = 2, sopfr = 5, n = P₁ = 6, M₃ = 7
P₂ = 28, P₃ = 496, σφ = 24, σ-τ = 8, T(6) = 21
```

### Verification

```
Benzene formula:
  C₆H₆: 6 C + 6 H = 12 total atoms = σ ✓
  Carbon count = P₁ = 6                   ✓
  Hydrogen count = P₁ = 6                 ✓

Hückel rule (4n+2 for n=1):
  π electrons = 6 = P₁                    ✓
  The rule's φ coefficient (4 = τ) and offset (2 = φ):
    τ·n + φ = 4(1) + 2 = 6 = P₁

Symmetry: D₆ₕ point group
  6-fold rotational symmetry = P₁          ✓
```

### Texas Sharpshooter Check

Benzene is the foundational aromatic molecule and its P₁=6 structure is well-known. The Hückel rule 4n+2 does decompose as τ·n+φ, which is clean. The total atom count C₆H₆=12=σ adds to the coherence. This is not a numerical fit but a structural identity — the first aromatic is built from P₁ in multiple independent ways.
