```

    4. Esteem (respect, recognition)

    7. Self-actualization
```

### Structural Analysis

```
Level groupings:
  Deficiency needs (D-needs): 4 = τ levels (1-4)
    (Absence causes anxiety; satisfaction removes motivation)
  Growth needs (B-needs):     1 = level 5 in original
    (Pursuit continues even when satisfied)
  Extended growth needs:      4 = τ levels (5-8)

  D-needs: τ = 4 (deficiency-driven)
  B-needs: τ = 4 in extended version (growth-driven)
  Total: 2τ = σ - τ = 8

  Original split:
    D-needs: τ = 4
    B-needs: 1
    Total: sopfr = 5

Hierarchy as binary tree:
  φ categories (D-needs / B-needs)

```
