# H-CX-678: Proton Charge Radius r_p ≈ sopfr/P₁ = 5/6 fm

> **Hypothesis**: The proton charge radius r_p = 0.8414 fm ≈ sopfr/P₁ = 5/6 = 0.8333 fm (0.96% error). The n=6 Egyptian fraction 5/6 predicts the proton size.

## Grade: 🟧★ (sub-1% from simplest n=6 ratio)

## Results

### The Prediction

```
r_p = 0.8414(19) fm   (muonic hydrogen, Pohl et al. 2010; PRad 2019)

sopfr/P₁ = sopfr(6)/P₁ = 5/6 = 0.8333 fm

Error: |0.8414 − 0.8333|/0.8414 = 0.96%
```

### The Proton Radius Puzzle

The proton radius was controversial for a decade:
```
CODATA 2010 (e-p scattering): 0.8768(69) fm
Muonic hydrogen (Pohl 2010):  0.84184(67) fm  — 7σ discrepancy!
PRad (JLab, 2019):            0.831(14) fm
CODATA 2018 (resolved):       0.8414(19) fm

sopfr/P₁ = 5/6 = 0.8333 fm — consistent with the smaller (muonic) value
```

### Egyptian Fraction Structure

```
5/6 = 1 − 1/6 = 1 − 1/P₁

This is the "unit fraction complement" of the first perfect number.
The proton radius in fm is within 1% of (1 − 1/P₁).
```

### Higher Precision

With σ correction: 5/6 + 1/(σ²·φ) = 5/6 + 1/288 = 0.8368  (0.55%)
