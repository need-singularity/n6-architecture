# H-CX-544: Up-Type Quark Koide Q(u,c,t) ≈ sopfr/P₁ = 5/6

> **Hypothesis**: The Koide-like formula for up-type quarks gives Q(u,c,t) ≈ 5/6 = sopfr(6)/P₁.

## Grade: 🟧 (1.9% match; extends Koide lepton result)

## Results

### The Prediction

```
Q(u,c,t) = (m_u + m_c + m_t) / (√m_u + √m_c + √m_t)²

Using PDG masses (MS-bar at 2 GeV for u,c; pole for t):
  m_u = 2.16 MeV, m_c = 1270 MeV, m_t = 172,760 MeV

Q = (2.16 + 1270 + 172760) / (1.470 + 35.64 + 415.6)²
  = 174032 / (452.7)²
  = 174032 / 204937
  = 0.849

Predicted: sopfr/P₁ = 5/6 = 0.8333
Error: 1.9%
```

### Comparison with Lepton Koide

| Sector | Koide Q | n=6 Prediction | Error |
|---|---|---|---|
| Leptons (e,μ,τ) | 0.66666 | φ/σ/τ²·σ² = 2/3 | 0.0009% |
| Up quarks (u,c,t) | 0.849 | sopfr/P₁ = 5/6 | 1.9% |

### n=6 Pattern

```
Q_leptons = 2/3 = φ·τ²/σ² = 2×16/144     (Koide δ=2/9)
Q_up      ≈ 5/6 = sopfr/P₁                 (this hypothesis)

Difference: 5/6 - 2/3 = 1/6 = δ⁺ = SEDI channel frequency!
```

The gap between quark and lepton Koide values is exactly 1/6 = 1/P₁ = δ⁺.

### Caveat

The 1.9% error and running mass scheme dependence make this a partial match. The lepton Koide (0.0009%) is far more precise. However, the pattern Q_up - Q_lepton = 1/6 is striking if confirmed.
