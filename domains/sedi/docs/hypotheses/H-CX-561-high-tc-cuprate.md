# H-CX-561: Cuprate High-T_c Optimal Doping p* = 1/P₁ = 0.16

> **Hypothesis**: The optimal hole doping for cuprate superconductors p* ≈ 0.16 = 1/P₁ = 1/6 (approximately).

## Grade: 🟧 (approximate; 4% error)

## Results

### The Prediction

```
p* = 1/P₁ = 1/6 = 0.1667

Observed optimal doping (cuprates): p* ≈ 0.16 (YBCO, BSCCO, Hg-1201)
Error: ~4%
```

### Universal Dome Shape

The superconducting dome T_c(p) peaks at p* ≈ 0.16 across ALL cuprate families. This universality suggests a fundamental origin.

### T_c Maximum (YBCO)

```
T_c,max(YBCO) = 93 K

93 = σ(σ-τ) - σ/τ = 12×8 - 3 = 96-3 = 93    EXACT integer
```

### Phase Diagram Boundaries

| Doping | Value | n=6 |
|---|---|---|
| Underdoped boundary | ~0.05 | 1/C(6,3) = 1/20 |
| Optimal p* | ~0.16 | ≈ 1/P₁ |
| Overdoped boundary | ~0.27 | ≈ σ/τ / σ = 1/4 = 1/τ |

### Caveat

p* = 0.16 is not exactly 1/6 = 0.1667. The 4% discrepancy is significant. However, the universality of p* ≈ 0.16 across cuprate families is unexplained, and 1/P₁ is the simplest n=6 expression in this range.
