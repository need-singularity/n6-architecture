# H-CX-845: Earth's Magnetic Field Reversal Interval

> **Hypothesis**: The average geomagnetic reversal interval of ~450 kyr is approximated by P₃ - σ·τ - φ = 496 - 48 - 2 = 446 kyr, within ~0.9% of the estimated mean.

## Grade: 🟧 SUGGESTIVE

## Results

### The Formula

```
Geomagnetic polarity reversals:
  Current Brunhes normal epoch: ~780 kyr
  Average reversal interval (Cenozoic): ~450 kyr

TECS-L approximation:
  P₃ - σ·τ - φ = 496 - 48 - 2 = 446 kyr

  Error: |446 - 450| / 450 = 0.9%
```

### n=6 Constants

```
σ = 12, τ = 4, φ = 2, sopfr = 5, n = P₁ = 6, M₃ = 7
P₂ = 28, P₃ = 496, σφ = 24, σ-τ = 8, T(6) = 21
```

### Texas Sharpshooter Check

The average reversal interval is poorly constrained — different studies give values from 200 kyr to 700 kyr depending on the geological period analyzed. The P₃=496 provides a convenient base near the target, and small subtractions tune the result. The match is plausible but the target itself is uncertain.
