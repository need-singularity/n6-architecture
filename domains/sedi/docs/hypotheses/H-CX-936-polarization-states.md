# H-CX-936: Polarization States = φ

> **Hypothesis**: Light has φ = 2 independent polarization states. This holds for any massless spin-1 boson (photon) and extends to the graviton (spin-2, also φ = 2 helicity states).

## Grade: 🟩 EXACT

## Results

### The Count

```
Photon (spin-1, massless):
  Helicity states: +1, -1  → count = 2 = φ
  (Spin-0 longitudinal mode forbidden by gauge invariance)

Polarization bases (all equivalent, all φ-dimensional):
  Linear:    H, V           → 2 = φ states
  Circular:  L, R           → 2 = φ states
  Diagonal:  +45°, -45°     → 2 = φ states
```

### n=6 Constants

```
σ = 12, τ = 4, φ = 2, sopfr = 5, n = P₁ = 6, M₃ = 7
P₂ = 28, P₃ = 496, σφ = 24, σ-τ = 8
```

### Why Not 3?

```
A spin-s particle has (2s+1) states in general:
  Spin-0: 1 state  = R(6)
  Spin-1: 3 states = σ/τ ... but photon is massless

Massless gauge bosons lose one polarization:
  (2s+1) - 1 = 2s = 2·1 = φ for photon

The reduction from σ/τ = 3 to φ = 2 is gauge invariance.
```

### Extension to Other Particles

```
Graviton (spin-2, massless):  2 = φ helicity states (+2, -2)
Gluon (spin-1, massless):    2 = φ per gluon × 8 gluons = 16 = σ+τ
Gravitino (spin-3/2):        2 = φ helicity states
```
