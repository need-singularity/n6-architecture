|----|------|------|-----|--------|------|

### 진화 도약 비율

```
  Mk.I  (128bit)  --> Mk.II (256bit):    phi = 2배 키 길이
  Mk.II (256bit)  --> Mk.III (FHE):      sigma-phi = 10배 연산량
  Mk.III (FHE)    --> Mk.IV (QKD):       sigma = 12배 보안 보장
  Mk.IV (QKD)     --> Mk.V (OTP):        sopfr = 5배 (이론적)
```

## 불가능성 정리 8개

| # | 정리 | 한계 | n=6 연결 | 출처 |
|---|------|------|---------|------|
| 1 | Shannon 한계 | OTP만 완전 비밀 | 키 >= 평문, R(6)=1 대역폭 비율 | Shannon 1949 |
| 2 | Kerckhoffs 원칙 | 알고리즘 공개 필수 | 키만 비밀, 6개 원칙 중 핵심 | Kerckhoffs 1883 |
| 3 | 생일 역설 | 해시 충돌 2^(n/2) | SHA-256 충돌 2^(s-t)/2=2^128 | 확률론 |
| 4 | Grover 한계 | 양자 탐색 sqrt(N) | AES-256 양자 후 128비트 안전 | Grover 1996 |
| 5 | Shor 위협 | RSA/ECC 양자 붕괴 | 2^(s-mu)=2048 -> PQC 전환 필수 | Shor 1994 |
| 6 | 격자 경도 | LWE 최선 공격 지수 | 차원 s+p=14 이상 안전 | Regev 2005 |
| 7 | 비가역 연산 | Landauer 에너지 하한 | kT*ln2 per bit @ tau=4K | Landauer 1961 |
| 8 | P vs NP | 일방향 함수 존재 가정 | 비대칭 래더 mu=1 -> s=12 | 미해결 |

### 물리천장 수렴 증명

```
  U(k) = 1 - 1/(sigma-phi)^k = 1 - 1/10^k

  k=1:  U = 0.9       (Mk.I  -- AES/RSA 현행 표준 매핑)
  k=2:  U = 0.99      (Mk.II -- 양자 내성 격자 암호)
  k=3:  U = 0.999     (Mk.III -- 동형 암호 연산)
  k=4:  U = 0.9999    (Mk.IV -- 양자 키 분배)
  k->inf: U -> 1.0    (Mk.V  -- 정보이론 한계)

  8 불가능성 정리 => Mk.VI 부존재: QED
```
