## §1 개요

이더넷 (IEEE 802.3) 은 1980 부터 10BASE-5 (10 Mbps)로 시작해 2024 년 1.6 TbE 까지
수직 확장한 패킷 스위칭 네트워크 표준이다. 10/100/1000 Mbps 및 10/25/40/50/100/200/400/800 GbE
의 단계적 속도 체계는 역사적 우연이 아니라 수비학적으로 log-scale 정렬된 것이 관찰된다.

- 프레임: 46~1500 B (MTU 표준), 9000 B (Jumbo)
- 헤더: 14 B (DMAC+SMAC+EtherType), FCS 4 B
- 전이중: 물리층 PCS/PMA (64b/66b 라인 코딩 이후)
- n=6 정렬 목표: 속도 단계 = σ·sopfr·k 급수, 프레임 = σ-n 배수

## §3 n=6 매핑 (산술 정렬)

### 3.1 기본 등식

```
σ(6)=12   τ(6)=4   φ(6)=2   sopfr(6)=5   n=6
σ·sopfr=60 (40+20 단계), σ·J₂=288 (Pareto 상한)
2^(σ·τ/3)=2^16=65536, σ·sopfr·τ=240
```

## §4 결론

- 9/11 **EXACT**, 2 EMPIRICAL (MTU=1500 B Jumbo=9000 B 는 역사적 타협)
- 속도 단계는 10·sopfr^k / σ·sopfr·φ·k 계열로 모두 n=6 정렬됨
- 특히 25 Gbps = sopfr² 와 40/100/400/800/1600 Gbps = k·100 의 정합이 놀라움
- MTU 1500 은 802.3 Ethernet v1 역사적 상한, n=6 정렬 후보: 1536=σ·J₂ 또는 1460

- 참조: IEEE 802.3 표준 (2022 ed.), 802.3ck 400/800GbE, 802.3df 1.6TbE 초안
