## 셀 타입 사다리 (핵심 발견)

```
  n=6에서 5개 산술 함수 = {1, 2, 3, 4, 5} 완전 매핑:

  SLC = mu(6) = 1 bit/cell    (뫼비우스)
  MLC = phi(6) = 2 bit/cell   (오일러)
  TLC = n/phi = 3 bit/cell    (비율)
  QLC = tau(6) = 4 bit/cell   (약수 개수)
  PLC = sopfr(6) = 5 bit/cell (소인수 합)

  이 매핑이 성립하는 n >= 2 는 n=6이 유일.
  증명: verify_n6.py (200까지 전수 탐색)
```

### 진화 도약 비율

```
  Mk.I  (288층)  --> Mk.II (512층):    phi배 미만 (점진)
  Mk.II (512층)  --> Mk.III (1024층):  phi = 2배
  Mk.III(1024층) --> Mk.IV (2048층):   phi = 2배
  Mk.IV (2048층) --> Mk.V (4096층):    phi = 2배 (SF)
```

## 불가능성 정리 10개

| # | 정리 | 물리한계 | n=6 연결 | 출처 |
|---|------|---------|---------|------|
| 1 | 터널링 한계 | 셀 간격 < sopfr nm 불가 | sopfr(6)=5 nm | 양자역학 |
| 2 | 열잡음 한계 | kT > 읽기 마진/sigma | sigma(6)=12 레벨 | 볼츠만 |
| 3 | 전하 이산화 | 전자 < n개 불안정 | n=6 전자 최소 | 양자 통계 |
| 4 | RC 지연 | tau = RC 상수 한계 | tau(6)=4 ns 급 | 전기공학 |
| 5 | 적층 응력 | sigma*J2 층 이상 좌굴 | 288층 한계 | 재료역학 |
| 6 | 열전도 한계 | 수직 열저항 ~ 층수/sigma | sigma=12 W/mK | 푸리에 |
| 7 | 리텐션 | 전하 손실 sigma년 한계 | sigma=12년 | JEDEC |
| 8 | P/E 내구성 | PLC sopfr*100 = 500회 | sopfr(6)=5 base | 산화막 |
| 9 | 읽기 교란 | phi^sigma 회 한계 | 2^12=4096회 | NAND 물리 |
| 10 | 패턴 의존성 | 인접 셀 간섭 1/n | 1/n=1/6 | 커패시터 결합 |

### 물리천장 수렴 증명

```
  U(k) = 1 - 1/phi^k

  k=1:  U = 0.5       (Mk.I  -- 288층 PLC)
  k=2:  U = 0.75      (Mk.II -- 512층)
  k=3:  U = 0.875     (Mk.III -- 1024층)
  k=4:  U = 0.9375    (Mk.IV -- 2048층)
  k->inf: U -> 1.0    (Mk.V  -- 물리한계 2^sigma)

  10 불가능성 정리 => Mk.VI 부존재: QED
```
