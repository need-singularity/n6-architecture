## PPTA 분자구조 상세

```
PPTA (poly-p-phenylene terephthalamide):

    O   H           O   H
    ||  |            ||  |
  --C--[C6]--C--N--[C6]--N--
        n=6         n=6

반복단위 원자 수: C14 H10 N2 O2 = 28 원자
  - 총탄소 14 = sigma+phi
  - 수소 10 = sigma-phi
  - 헤테로원자 4 = tau
  - 합계 28 = 2번째 완전수

2번째 완전수 28이 반복단위에 나타나는 것은
n=6 완전수 체계의 상위 구조가 고분자 화학에 투영되는 사례다.
```

## Honest Limitations

- 밀도 1.44 g/cm3는 표준 PPTA 기준이며, 변형 아라미드(m-아라미드 Nomex 등)는 1.38
- 28원자 = 2nd 완전수는 화학적 필연이 아니라 결과적 일치의 가능성 있음
- LOI 29는 섬유 상태 기준이며, 직물/복합재 상태에서는 변동
- 인장강도 3.6 GPa는 단섬유 기준이며 방탄 직물 상태에서는 복합 효과로 변동
- sigma*sopfr^3=1500 공식은 수학적 일치이며, 데니어 규격은 역사적 관행에서 유래

## 산업 의의

Heracron은 방탄복(NIJ 레벨 IIIA), 방호헬멧, 로프/케이블 보강재, 항공 복합재에 사용된다. 밀도 1.44 g/cm3 = sigma^2/100이 규격서의 핵심 수치이며, LOI 29 = sigma*phi+sopfr은 방염 인증 기준값이다. 1500d/1000f 조합은 방탄 직물의 국제 표준 구성이다. n=6 완전수 체계가 벤젠 고리에서 방탄 규격까지 동일하게 관통한다.

## 검증 코드

```python
import math
def sigma(n): return sum(d for d in range(1, n+1) if n % d == 0)
def tau(n):   return sum(1 for d in range(1, n+1) if n % d == 0)
def phi(n):   return sum(1 for k in range(1, n+1) if math.gcd(k, n) == 1)
def sopfr(n):
    s, m, d = 0, n, 2
    while d*d <= m:
        while m % d == 0: s += d; m //= d
        d += 1
    if m > 1: s += m
    return s
def jordan2(n):
    r = n*n; m, d = n, 2
    while d*d <= m:
        if m % d == 0:
            r = r * (1 - 1/(d*d))
            while m % d == 0: m //= d
        d += 1
    if m > 1: r = r * (1 - 1/(m*m))
    return int(round(r))

assert sigma(6) == 12 and tau(6) == 4 and phi(6) == 2
assert sopfr(6) == 5 and jordan2(6) == 24
assert sigma(6) * phi(6) == 6 * tau(6)

N = 6
S, T, P, SP, J = sigma(N), tau(N), phi(N), sopfr(N), jordan2(N)
results = [
    ("벤젠고리 phi=2", P, 2),
    ("벤젠 탄소 n=6", N, 6),
    ("총탄소 sigma+phi=14", S+P, 14),
    ("수소 sigma-phi=10", S-P, 10),
    ("헤테로원자 tau=4", T, 4),
    ("총원자 J2+tau=28", J+T, 28),
    ("아미드 phi=2", P, 2),
    ("밀도*100 sigma^2=144", S**2, 144),
    ("카르보닐 phi=2", P, 2),
    ("방향족C sigma=12", S, 12),
    ("방사단계 n=6", N, 6),
    ("황산원자 sigma-sopfr=7", S-SP, 7),
    ("1500d sigma*sopfr^3", S*SP**3, 1500),
    ("3000d phi*sigma*sopfr^3", P*S*SP**3, 3000),
    ("1000f (sigma-phi)^3", (S-P)**3, 1000),
    ("단섬유 sigma=12um", S, 12),
    ("LOI sigma*phi+sopfr=29", S*P+SP, 29),
    ("열분해 sopfr*(sigma-phi)^2=500", SP*(S-P)**2, 500),
    ("수분율 sopfr=5", SP, 5),
    ("코팅층 phi=2", P, 2),
]
passed = sum(1 for _, a, b in results if a == b)
print(f"검증: {passed}/{len(results)} PASS")
for name, actual, expected in results:
    mark = "PASS" if actual == expected else "FAIL"
    print(f"  {mark}: {name} = {actual} (기대: {expected})")
```
