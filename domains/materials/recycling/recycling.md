### 재활용 효율의 절대 상한

```
  ◆ Gibbs 자유에너지 관점

  혼합물 분리에 필요한 최소 에너지:
    ΔG_sep = -RT Σᵢ xᵢ ln(xᵢ)

  n=6 등몰 혼합물 (6종 폐기물 균등):
    xᵢ = 1/n = 1/6
    ΔG_sep = -RT · n · (1/n) · ln(1/n) = RT · ln(n) = RT · ln(6)
    T=300K: ΔG_sep = 8.314 × 300 × 1.792 = 4,468 J/mol = 4.47 kJ/mol

  실제 폐기물 1톤 ≈ 10⁴ mol (평균 MW=100):
    E_min = 4.47 × 10⁴ kJ = 44.7 MJ ≈ 12.4 kWh (= σ kWh!)
    → 이론 최소 분리 에너지 = σ=12 kWh/ton — n=6 EXACT!

    효율 = 12.4/50 = 24.8% ≈ J₂% of Carnot

## 핵심 화학 반응 지도 — n=6 화학량론 완전 일치

### 유기물 재활용 (BT-101, BT-103)

```
  광합성 (순방향):
    6CO₂ + 12H₂O → C₆H₁₂O₆ + 6O₂ + 6H₂O
    계수: {6, 12, 6, 12, 6, 6, 6} — 전부 n=6 약수 또는 배수

  혐기 소화 (역방향):
    C₆H₁₂O₆ → 3CO₂ + 3CH₄
    계수: {6, 12, 6, 3, 3} → {n, σ, n, n/φ, n/φ}
    메탄 생산량: n/φ=3 mol/mol glucose

  호기 퇴비화:
    C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O
    계수: 전부 n=6 — 완전수 화학량론 (BT-103)
```

### 플라스틱 해중합 (BT-121, BT-85)

```
  PET 해중합 (글리콜리시스):
    [-OC-C₆H₄-CO-O-CH₂CH₂-O-]ₙ + n·HOCH₂CH₂OH
    → n·BHET (bis-hydroxyethyl terephthalate)
    핵심: C₆H₄ 벤젠 고리 = Z=6 탄소 고리 보존

  PE 열분해 (4구간):
    -(CH₂-CH₂)ₙ- →[τ=4 zone] C₂~C₆ 올레핀 혼합물
    주 생성물: C₆H₁₂ (cyclohexane) — Z=6 고리 복원

  PS 해중합:
    -(CH₂-CHC₆H₅)ₙ- → n·C₆H₅CH=CH₂ (styrene)
    C₆H₅ 페닐기 = Z=6 탄소 고리 완전 보존

import math
def sigma(n): return sum(d for d in range(1, n+1) if n % d == 0)
def tau(n):   return sum(1 for d in range(1, n+1) if n % d == 0)
def phi(n):   return sum(1 for k in range(1, n+1) if math.gcd(k, n) == 1)
def sopfr(n):
    s, m, d = 0, n, 2
    while d*d <= m:
        while m % d == 0: s += d; m //= d
        d += 1
    if m > 1: s += m
    return s
def jordan2(n):
    r = n*n; m, d = n, 2
    while d*d <= m:

        d += 1
    if m > 1: r = r * (1 - 1/(m*m))
    return int(round(r))
