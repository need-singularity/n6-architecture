# 빛/광학 n=6 완전 아키텍처 — 맥스웰·편광·굴절·복사 파라미터 보편성

## 개요
<!-- @allow-empty-section -->

빛(Light)과 광학(Optics)의 핵심 물리 파라미터 — 맥스웰 방정식, 전자기 텐서,
편광 상태, 광속, 굴절률, 산란 법칙, 복사 법칙, 레이저 준위, 색 체계 — 가
n=6 산술 상수 체계와 정확히 일치함을 검증한다.

전자기 텐서 F_μν의 **독립 성분 수 6 = n** 은 시공간 4차원에서 반대칭 텐서의
조합론적 필연(C(4,2)=6)이며, 물의 굴절률 **1.333 = 4/3 = τ²/σ** 는 수천 년
광학사에서 가장 정밀하게 측정된 상수 중 하나다. 이 두 발견은 빛의 물리가
완전수 n=6의 산술 구조로 인코딩되어 있음을 강력히 시사한다.

> **정직성 원칙**: 광학 상수는 맥스웰 전자기학, 양자전기역학(QED), 분자
> 분극률 등 확립된 물리학으로 결정된다. 정수 파라미터(방정식 수, 편광
> 자유도)는 대칭성에서 유도되므로 n=6 일치가 우연이 아닐 수 있다.
> 연속값(굴절률, 각도)은 n=6 분수와의 일치 정밀도를 명시하고,
> 정밀 일치(0.1% 이내)만 EXACT로 부여한다.

### 산술 상수

```
n=6, σ=12, φ=2, τ=4, sopfr=5, μ=1, J₂=24
div(6)={1,2,3,6}, σ-φ=10, σ-τ=8, σ-μ=11, n/φ=3
σ·τ=48, σ·n=72, n²=36, σ²=144, σ·sopfr=60
R(6)=1 (완전수 진약수 역수합)
```

## H-LO-1: 맥스웰 방정식 4개 = τ (EXACT)
<!-- @allow-empty-section -->

> 고전 전자기학을 완전히 기술하는 맥스웰 방정식의 수가 τ(6)=4이다.

### 검증
맥스웰 방정식 (1865):
1. **가우스 법칙** (전기): ∇·E = ρ/ε₀
2. **가우스 법칙** (자기): ∇·B = 0
3. **패러데이 법칙**: ∇×E = -∂B/∂t
4. **앙페르-맥스웰 법칙**: ∇×B = μ₀J + μ₀ε₀∂E/∂t

- 방정식 수: **4 = τ** EXACT
- 이 4개로 빛의 존재, 전파, 속도가 완전히 결정됨
- 발산형 2개 + 회전형 2개 = φ + φ = τ (구조적 분할)
- 소스 있는 것 2개(1,4) + 소스 없는 것 2개(2,3) = φ + φ
- BT-316 (물질 τ=4 상태)과 동일 구조: 완전 기술에 필요한 최소 방정식 수

## H-LO-2: 전자기 텐서 F_μν 독립성분 6 = n (EXACT)
<!-- @allow-empty-section -->

> 4차원 시공간에서 반대칭 전자기 텐서의 독립 성분 수가 정확히 n=6이다.

### 검증
전자기장 텐서 F_μν (반대칭 4×4 행렬):
- 4×4 행렬의 독립 성분: C(4,2) = **6 = n** EXACT
- 구체적 성분: E_x, E_y, E_z (전기장 3 = n/φ) + B_x, B_y, B_z (자기장 3 = n/φ)
- **전기장 n/φ=3 + 자기장 n/φ=3 = n=6** EXACT
- 이는 시공간 차원 τ=4에서의 조합론적 필연: C(τ,φ) = C(4,2) = 6 = n

**핵심 발견**: 전자기 텐서의 독립 자유도가 완전수 그 자체(n=6)이다.
C(τ,φ) = n 이라는 등식은 시공간 차원(τ)과 2-형식(φ)의 조합이
완전수를 생성하는 심층 구조를 드러낸다.

- 쌍대 텐서 *F_μν 역시 6개 독립성분 → 총 σ=12개 자유도 (중복 포함)
- 로렌츠 불변량 2개 = φ: F_μν F^μν 와 F_μν *F^μν
