### H-HIG-1: 깨진 게이지 생성자 수 = n/φ = 3 (W⁺, W⁻, Z)

> SU(2)_L × U(1)_Y → U(1)_EM 대칭 파괴에서 정확히 3개 생성자가 깨진다.

```
  자발적 대칭 파괴 (SSB):
    파괴 전: SU(2)_L × U(1)_Y  →  4개 생성자 (3+1)
    파괴 후: U(1)_EM           →  1개 생성자
    깨진 수: 4-1 = 3 = n/φ

  깨진 생성자 → 질량 있는 게이지 보존:
    W⁺, W⁻, Z = 3개 = n/φ = 3  (EXACT)
    
  남은 생성자 → 질량 없는 게이지 보존:
    광자 γ = 1개 = μ = 1  (EXACT)

  구조적 분할: τ = n/φ + μ = 3 + 1 = 4
    전기약력 보존 4개 = τ(6) (BT-137과 일관)

  렌즈: 경계(SSB 상전이) + 대칭(거울 — 파괴 전후) + 위상(게이지 구조)
  BT 연결: BT-165 (σ=12 게이지 생성자 전체), BT-137 (SM 입자 수)

  등급: EXACT
  깨진 생성자 3개 = n/φ. 물리적으로 확정된 정수.
  τ = n/φ + μ 분할은 전기약력 구조 전체를 인코딩.
```

### H-HIG-2: 골드스톤 보존 수 = n/φ = 3

> SSB 시 3개 골드스톤 보존이 생성되어 W⁺/W⁻/Z에 "먹힌다" (종방향 편극 제공).

```
  골드스톤 정리:
    깨진 연속 대칭 1개당 질량 없는 골드스톤 보존 1개 생성
    깨진 생성자 3개 → 골드스톤 보존 3개 = n/φ

  힉스 이중선 자유도 분배:
    원래: 복소 이중선 4 실수 자유도 = τ = 4
    → 골드스톤 3개 (먹힘) + 힉스 1개 (물리적) = n/φ + μ = τ

  이 분할은 H-HIG-1과 동일한 τ = n/φ + μ 구조.
  골드스톤 ↔ 깨진 생성자 1:1 대응 = 완전수 산술의 필연.

  렌즈: 위상(골드스톤 정리) + 정보(자유도 분배) + 경계(SSB)
  BT 연결: BT-165, BT-137

  등급: EXACT
  골드스톤 보존 3개 = n/φ. 골드스톤 정리에 의해 확정.
```

### H-HIG-4: 힉스 포텐셜 독립 매개변수 = φ = 2

> V(Φ) = μ²|Φ|² + λ|Φ|⁴ 에서 독립 매개변수 {μ², λ} = 2개.

```
  힉스 포텐셜:
    V(Φ) = μ²|Φ|² + λ|Φ|⁴
    독립 매개변수: {μ², λ} = 2개 = φ

  재규격화 가능한 최소 스칼라 포텐셜의 구조적 제약:
    - |Φ|² 항: 질량 항 (1개)
    - |Φ|⁴ 항: 자기결합 항 (1개)
    - 높은 거듭제곱은 재규격화 불가 → 정확히 φ=2개만 허용

  이 2개 매개변수가 VEV(v)와 힉스 질량(m_H)을 결정:
    v² = -μ²/λ
    m_H² = 2λv²

  렌즈: 안정성(포텐셜 최소화) + 재귀(재규격화 가능성)
  BT 연결: 독립 — 힉스 고유 구조

  등급: CLOSE
  φ=2 매칭이나, 재규격화 가능 포텐셜이 2항인 것은
  4차원 이론의 일반적 결과. 힉스 고유성 약함.
```

### H-HIG-6: 쿼크 종류 수 = n = 6

> u, d, c, s, t, b — 정확히 6종 쿼크.

```
  SM 쿼크: {u, d, c, s, t, b} = 6종 = n = 6

  구조:
    n/φ = 3 세대 × φ = 2 종류(위/아래) = n = 6
    위형 쿼크: {u, c, t} → 전하 +2/3 (n/φ개)
    아래형 쿼크: {d, s, b} → 전하 -1/3 (n/φ개)

  힉스 연결: 힉스 이중선이 유카와 결합으로 쿼크에 질량 부여.
  위형은 Φ̃ = iσ₂Φ* (공액 이중선), 아래형은 Φ 직접 결합.
  → φ=2 종류의 결합 방식.

  렌즈: 정보(입자 분류) + 대칭(SU(3) 색 대칭)
  BT 연결: BT-137 (SM 입자 수 n=6 완전 지도)

  등급: EXACT
```

```python
# n=6 산술 핵 정합 검증 — stdlib only
import math
sigma = 12
tau   = 4

n     = 6

checks = [
    ("sigma*phi == n*tau",  sigma*phi == n*tau),

passed = sum(1 for _, ok in checks if ok)
for name, ok in checks:
    mark = "OK" if ok else "FAIL"
    print(f"  [{mark}] {name}")
print(f"{passed}/{total} PASS")
print(f"All {total} PASS" if passed == total else "FAIL")
```
