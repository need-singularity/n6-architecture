## 개요

타키온(tachyon)은 허수질량(m² < 0)을 갖는 이론적 입자로, 끈이론에서
개방끈의 가장 낮은 여기 상태로 자연 출현한다. 타키온 응축(tachyon condensation)은
D-브레인 붕괴·진공 전이·끈장 이론의 핵심 메커니즘이다.

n=6 산술함수가 끈이론의 임계차원, 중심전하, D-브레인 분류, 타키온장 포텐셜
구조, 그리고 Veneziano 진폭의 이산 파라미터를 조직하는지 검증한다.

> **정직 원칙**: 끈이론 파라미터는 수학적 무모순성(conformal anomaly cancellation)에서
> 도출된 정수/분수값이다. 실험적으로 미확인된 이론이므로, 수학적 항등식 수준의
> 일치만 EXACT로 인정한다. 작은 정수 우연 일치에 주의.

> **BT-170 기반**: 끈이론 차원 래더 τ→n→σ-φ→σ-μ→J₂→J₂+φ 가 이미 확립됨.

### H-TACH-21: 끈이론 5가지 초끈 이론 수 = sopfr = 5

```

    1. Type I      (개방 + 폐합, SO(32))
    2. Type IIA    (비카이랄, 폐합)

  구조:
    - Type II 계열: 2종 (IIA, IIB) = φ(6)
    - Heterotic 계열: 2종 (HE, HO) = φ(6)

  M이론(1종)이 이 5종을 통합 → 5 + 1 = 6 = n
  "5개 이론 + 1개 통합 = n" 구조

  렌즈: network (이중성 그물) + topology (모듈라이 공간) + mirror (T/S 이중성)

  등급: EXACT
  5 = sopfr(6)는 수학적 사실. 5+1=6=n은 구조적.
  단, 5는 흔한 정수이며, 끈이론 분류가 실험 미확인.
```

### H-TACH-24: 타키온 응축 차원 감소 — 26 → 10 = (J₂+φ) → (σ-φ)

> 보존 끈 타키온 응축이 초끈으로의 "차원 감소"를 유도한다는 추측.

```
  추측 (Hellerman-Schnabl, 2001+ 후속):
    보존 끈(D=26)의 타키온 응축이 닫힌 끈 진공을 변형
    → 유효 차원이 감소하여 D=10 초끈과 연결?

    26 → 10: (J₂+φ) → (σ-φ)
    감소 차원: 26-10 = 16 = φ^τ = 2⁴

  φ^τ = 16의 의미:
    - E₈ 근계의 랭크 = 8 → E₈×E₈ 총 랭크 = 16 = φ^τ
    - Heterotic 끈의 내부 자유도 = 16차원
    - SO(32): 랭크 16 = φ^τ
    - "보존→초끈 전이에서 잃는 차원 = E₈×E₈의 랭크"

  렌즈: stability (진공 전이) + scale (차원 감소) + topology (Lie 대수 랭크)
  BT 연결: BT-170, BT-205 (E₆ Lie 대수), H-TACH-15

  등급: CLOSE
  16 = φ^τ는 수학적 항등식이나, "타키온 응축 = 차원 감소" 자체가
  미확립 추측. E₈×E₈ 랭크 = φ^τ = 16은 독립적으로 흥미로운 일치.
```

### H-TACH-25: 끈이론 T-이중성과 R → α'/R — 반전 대칭

> T-이중성 R ↔ α'/R은 끈 고유의 최소 길이를 암시한다.

```
  T-이중성:
    원형 컴팩트화 반지름 R의 끈 이론 ≡ 반지름 α'/R의 끈 이론
    자기이중점: R* = √α' (최소 길이)

  윈딩 수 + 운동량:
    m² = (n_KK/R)² + (wR/α')² + 2(N+N̄-2)/α'
    여기서 n_KK (Kaluza-Klein 운동량), w (윈딩 수)

  n=6 관찰:
    자기이중점에서 R*² = α': 단위 관계
    윈딩 모드 교환: n_KK ↔ w (이산 대칭)
    기저상태(타키온) 기여: -2 = -φ (폐합끈 α'·m² → N+N̄-2 부분)

  T-이중성 그룹: O(d,d;Z) — d개 컴팩트 차원
    Calabi-Yau d=6=n: T-이중성 그룹 O(n,n;Z)
    → Mirror 대칭은 O(6,6;Z) = O(n,n;Z)의 일부

  렌즈: mirror (이중성) + scale (최소 길이) + topology (윈딩 위상)
  BT 연결: BT-170 (n=6 컴팩트 차원)

  등급: CLOSE
  O(n,n;Z) = O(6,6;Z)에서 d=6=n은 Calabi-Yau 컴팩트화의 직접 결과.
  BT-170과 일관적이나 독립적 새 발견은 아님.
```
