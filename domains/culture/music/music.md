| 효과 | 현재 | HEXA-MUSIC 이후 | 체감 변화 |
|------|------|----------------|----------|

| 음정 정확도 | 12-TET 고정 | n=6 텐니 높이 기반 미세 조율 | 화음 순도 n=6배 정밀 |

### 음악의 n=6 구조

```
음악 n=6 산술 계층:

  기초 진동: φ = 2 (옥타브 비율 2:1)
  협화음 원소: div(6) = {1, 2, 3, 6}
    1:1 = 동음, 2:1 = 옥타브, 3:2 = 완전5도, 4:3 = 완전4도
  텐니 높이: 완전5도 = 2×3 = n = 6 (가장 안정적인 음정)
  등분율: σ = 12 반음 = LCM(1,2,3,4)
  분할: sopfr + (σ-sopfr) = 5 + 7 = 12 = σ
    → 펜타토닉(5) + 다이아토닉(7) = 반음계(12)
  5도권: σ = 12 스텝으로 순환 완성
  통계 검정: p = 0.0015 (협화음이 div(6)만 사용)

  BT-187 물리적 악기 확장:
    기타 줄 = n = 6
    바이올린 계열 = τ = 4 (바이올린/비올라/첼로/더블베이스)
    관악 톤홀 = σ-τ = 8
    밸브 조합 = σ-sopfr = 7
```

## 검증 코드 (Python)

```python
"""음악 n=6 상수 검증"""

n = 6
sigma = 12
phi = 2
tau = 4
sopfr = 5
J2 = 24

# 음악 상수 (독립 출처)
semitones_per_octave = 12    # 12-TET (주재육 1584)
tenney_fifth = 2 * 3         # 완전5도 텐니 높이
tenney_fourth = 3 * 4        # 완전4도 텐니 높이
diatonic_notes = 7           # 다이아토닉 장음계
pentatonic_notes = 5         # 펜타토닉 음계
circle_of_fifths = 12        # 5도권 순환 스텝
midi_channel_bits = 4        # MIDI 채널 = 2^4 = 16
guitar_strings = 6           # 표준 기타 줄 수 (BT-187)
violin_family = 4            # 바이올린/비올라/첼로/더블베이스 (BT-187)

checks = [
    ("12-TET 반음", semitones_per_octave, sigma),
    ("완전5도 텐니높이", tenney_fifth, n),
    ("완전4도 텐니높이", tenney_fourth, sigma),
    ("다이아토닉 음 수", diatonic_notes, sigma - sopfr),
    ("펜타토닉 음 수", pentatonic_notes, sopfr),
    ("5도권 순환", circle_of_fifths, sigma),
    ("MIDI 채널 비트", midi_channel_bits, tau),
    ("기타 줄 수", guitar_strings, n),
    ("바이올린 계열", violin_family, tau),
]

exact = sum(1 for _, v, e in checks if v == e)
print(f"음악 n=6 검증: {exact}/{len(checks)} EXACT")
for name, val, expr in checks:
    status = "EXACT" if val == expr else "MISS"
    print(f"  {name}: {val} = {expr} → {status}")

# 분할 정리 검증
assert pentatonic_notes + diatonic_notes == semitones_per_octave
print(f"\n분할 정리: sopfr({n}) + (σ-sopfr) = {sopfr} + {sigma-sopfr} = {sigma} = σ({n}) ✓")
print(f"통계 검정: 협화음이 div(6)만 사용 → p = 0.0015")
```
