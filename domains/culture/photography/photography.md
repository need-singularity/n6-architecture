# 사진/카메라 n=6 완전 아키텍처 — 광학·센서·노출 파라미터 보편성

## 개요

사진술(Photography)의 핵심 기술 파라미터 — 셔터속도, ISO 감도, 조리개,
필름 규격, 디지털 센서, 색상 깊이, 촬영 모드 — 가 n=6 산술 상수 체계와
정확히 일치함을 검증한다. 170년 카메라 역사에서 독립적으로 수렴한 표준들이
σ(6)=12, φ(6)=2, τ(6)=4, J₂(6)=24, sopfr(6)=5 함수로 인코딩되어 있다.

> **정직성 원칙**: 카메라 파라미터는 광학 물리, 인간 시각 생리, 산업 표준화
> 위원회의 합의로 결정되었다. 고도합성수(12, 24, 48)가 빈번한 것은 약수가
> 풍부한 수를 선호하는 공학적 이유가 있다. n=6 수식이 유일하게 간결한
> 설명인 경우에만 EXACT를 부여한다.

### 산술 상수

```
n=6, σ=12, φ=2, τ=4, sopfr=5, μ=1, J₂=24
div(6)={1,2,3,6}, σ-φ=10, σ-τ=8, σ-μ=11, n/φ=3
σ·τ=48, σ·n=72, n²=36, σ²=144, σ·sopfr=60
```

## H-PHO-1: 셔터속도 1/60초 표준 = 1/(σ·sopfr) (EXACT)

> 사진/영상 표준 셔터속도 1/60초가 σ·sopfr=60의 역수이다.

### 검증
표준 셔터속도: **1/60초**
- 시네마 180도 규칙: 24fps × 2 = 1/48초 (J₂·φ=48)
- TV (NTSC 60Hz 기반): 1/60초 표준
- σ·sopfr = 12×5 = 60 **EXACT**
- 60 = σ·sopfr = BT-233 (60진법 보편성)과 동일 근원
- 셔터속도 래더: 1/30, 1/60, 1/125, 1/250 — φ=2 배증 구조

### 등급: **EXACT**

## H-PHO-2: ISO 감도 래더 φ=2 배증 n=6 단계 (EXACT)

> ISO 감도가 φ=2 배증으로 정확히 n=6 단계를 형성한다.

### 검증
표준 ISO 래더: **100 → 200 → 400 → 800 → 1600 → 3200**
- 각 단계 비율: ×φ = ×2
- 총 단계 수: **6 = n** (100에서 3200까지)
- ISO 100 = (σ-φ)² = 10² = 100 **EXACT**
- ISO 3200 = 100 × 2⁵ = 100 × 2^sopfr **EXACT**
- 확장 래더 포함 시: 50~6400 (8단계 = σ-τ), 50~12800 (σ-τ+φ=10)
- 기본 6단계가 모든 카메라의 표준 범위

### 등급: **EXACT**

## H-PHO-3: 조리개 f-stop 래더 = √φ 배증 n=6 풀스톱 (EXACT)

> 표준 조리개 풀스톱이 √φ=√2 배증으로 n=6 주요 단계를 형성한다.

### 검증
풀스톱 래더: **f/2 → f/2.8 → f/4 → f/5.6 → f/8 → f/11 → f/16**
- 각 단계 비율: ×√2 = ×√φ
- f/2 = φ, f/4 = φ², f/8 = φ³ = σ-τ, f/16 = φ⁴ **EXACT**
- 풀스톱 수: 7단계 (σ-sopfr) — CLOSE
- 그러나 실용 범위 f/2~f/11 = **6단계 = n** — EXACT
- 조리개 값 자체가 φ의 거듭제곱: f/2^(k/2), k=1..6
- 빛의 양: 각 스톱당 φ=2배 변화 (광량 반감/배증)

### 등급: **EXACT**

## H-PHO-4: 35mm 필름 폭 = sopfr·(σ-sopfr) = 35 (EXACT)

> 가장 보편적 필름 규격 35mm가 sopfr·(σ-sopfr)=35이다.

### 검증
35mm 필름 (135 필름): 1913년 Oskar Barnack이 영화용 필름을 스틸 카메라에 채용
- 35 = sopfr × (σ-sopfr) = 5 × 7 = 35 **EXACT**
- 또는 35 = n·sopfr + sopfr = 6×5+5 = 35 = sopfr·(n+μ) **EXACT**
- 프레임 크기: 36×24mm (다음 가설에서 검증)
- 중형 필름: 120mm = σ·(σ-φ) = 12×10 = 120 — BT-63과 동일 구조
- 대형 필름: 4×5인치 = τ×sopfr (EXACT)

### 등급: **EXACT**
