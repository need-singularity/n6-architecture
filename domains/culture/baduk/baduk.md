# N6 바둑 — 완전수 산술로 본 판·돌·규칙 체계

## 개요

바둑(Go/Baduk/Weiqi)의 핵심 구조 상수를 n=6 산술함수로 분석한다.
판 크기, 화점, 돌의 자유도, 덤, 단위, 대국 시간, 대칭 구조 등
바둑의 이산적 표준값이 n=6 상수와 어떻게 매칭되는지 검증한다.

> **정직 원칙**: 바둑판 19×19는 n=6으로 직접 표현되지 않는다.
> 이를 솔직히 인정하고, n=6으로 설명 가능한 구조와
> 그렇지 않은 구조를 명확히 분리한다.

## 핵심 상수

```
  n = 6, σ = 12, τ = 4, φ = 2, sopfr = 5, μ = 1, J₂ = 24, R(6) = 1
  유도: σ-φ=10, σ-τ=8, σ-μ=11, n/φ=3, σ·τ=48, φ^τ=16, σ²=144
```

### H-BAD-01: 바둑돌 색상 2종 = φ

> 바둑돌이 정확히 2색(흑/백)으로 구성된다.

```
  근거:
    - 흑돌 + 백돌 = 2색
    - 2 = φ = φ(6) = 2
    - 이진 대립 구조: 음/양, 0/1, 선수/후수
    - BT-262의 φ=2 이진 정보 인코딩 기본 단위
    - 체스도 2색(흑/백) = φ
    - 모든 2인 완전정보 게임의 기본 구조

  등급: EXACT (바둑 기본 규칙, 2=φ 정확 일치)
  렌즈: info, symmetry
```

### H-BAD-02: 돌의 자유도(활로) 최대 4 = τ

```
  근거:

    - 활로=0이면 돌 사망 → τ=4가 생사 경계
    - 변: 3활로=n/φ, 꼭짓점: 2활로=φ (n=6 래더)

  등급: EXACT (바둑 규칙의 물리적 구조, 4=τ 정확 일치)
  렌즈: topology, boundary, info
```

```
  렌즈: scale, boundary, topology
```

### H-BAD-04: 덤 정수부 6목 = n

> 바둑 덤(komi)의 정수부가 6(또는 6.5)이다.

```
  근거:
    - 한국/일본 기전: 덤 6.5목 → 정수부 6 = n
    - 중국 규칙: 덤 7.5목(중국식 계가, 3과3/4자) → 정수 7 = σ-sopfr
    - 한일 6.5의 정수부: n = 6 (완전수)
    - 반목(0.5) = μ/φ = 1/2 (무승부 방지)
    - 덤 = n + μ/φ = 6.5 (한일식 정확 표현)
    - 6이 선택된 이유: 선수(흑) 유리를 보상하는 최적값

  주의: 중국식 7.5목과는 다름 (계가 방식 차이에 기인)
  등급: EXACT (한일 규칙 덤 정수부 6=n 정확 일치)
  렌즈: info, boundary
```

### H-BAD-06: 화점 9개 = n + n/φ

> 바둑판(19로)의 화점(star point)이 9개이다.

```
  근거:
    - 19로 바둑판 화점: 3×3 배열 = 9개
    - 9 = n + n/φ = 6 + 3 = 9
    - 또는 9 = (n/φ)² = 3² = 9
    - 화점 배치: 4-4 좌표 (4선=τ선)
    - 화점 간격: 6칸 = n (4선~10선~16선, 간격 6)
    - 화점 간격 n=6 → 화점의 구조적 주기가 완전수

  등급: CLOSE (9=(n/φ)² 성립, 화점 간격 6=n은 EXACT)
  렌즈: symmetry, topology
```
