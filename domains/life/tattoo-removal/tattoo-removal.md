## 10. 3개 사업 도메인 (n/phi = 3)

### 도메인 1: 피코초+PFD R20 클리닉 (즉시 실행)
- **성숙도**: 100% | **투자**: 2~3.5억 | **손익분기**: 월 23세션
- **차별화**: R20+PFD 제대로 하는 클리닉 한국 내 희소
- **수가**: 세션당 80~150만원 (기존 30~50만 대비 프리미엄)
- **n=6 연결**: Fitzpatrick 6타입 전용 프리셋, 4패스(tau) 프로토콜

### 도메인 2: 클로드로네이트 마이크로니들 패치 (3~5년)
- **성숙도**: 40% | **인허가 기간**: 4~6년 | **비용**: 30~80억원
- **핵심**: 3부품(클로드로네이트+리포솜+MN) 조합 전임상 — 각 부품은 100% 성숙
- **단축 전략**: 기존 승인 성분 → 독성 데이터 재활용, 한국 MN 기업 위탁 (라파스/쿼드메디슨)
- **n=6 연결**: 2cm x 2cm 패치, 니들 121개(11x11), 서방 6~12개월 임상 일정

### 도메인 3: CCR2 길항제 국소 제형 (5~10년)
- **성숙도**: 30% | **혁신성**: 100% (세계 최초, 논문/특허 0건)
- **후보**: cenicriviroc (NASH 임상2상 완료, 안전성 풍부)
- **특허 전략**: 즉시 PCT 출원 가능 (방법론 특허, 전임상 불필요)
- **n=6 연결**: PLGA 서방 7일 = 골든 윈도우 + 6일 버퍼, 6경로 시스템 완성

# 타투 제거 100% 완전 골화 돌파

> 최종 갱신: 2026-04-10
> 상태: 돌파 설계 완료, 전임상 미착수

## 핵심 한 줄

**"부수는 건 됐다. 다시 삼키는 걸 막아라."**

문신 제거의 50년 병목은 레이저 성능이 아니라 대식세포 재포획 사이클이다.

## 문신 영구성의 진짜 생물학

```
문신 잉크 → 진피 대식세포가 포식 → 안정화 (문신 보임)
   ↓
대식세포 수명 약 1년 → 자연사
   ↓
잉크 세포외 방출 → 100% 자유 상태 (48시간)
   ↓
인접 새 대식세포가 CCR2 경유 진입 → 즉시 재포획
   ↓
문신 위치 동일하게 유지 → 수십 년 반복
```

출처: Baranska et al. 2018, J Exp Med 215(4):1115-1133

### 재포획 정량 타임라인 (마우스 모델)

| 시점 | 사건 |
|------|------|
| 0시간 | 대식세포 사멸 유도 (DT 투여) |
| 24시간 | CD64+ 세포 소멸 |
| 48시간 | 잉크 100% 세포외 상태 ← 골든 윈도우 열림 |
| 48~72시간 | Ly-6C(hi) 단핵구 CCR2 경유 진피 진입 시작 |
| 3일 | 단핵구 피크 ← 골든 윈도우 닫히기 시작 |
| 10일 | 대식세포 풀 완전 재구성 |
| 20일 | 멜라노파지 완전 재구성 |
| 90일 | 색소 함량 원래 수준 완전 정상화 |

### 경로 1: 피코초 레이저 + PFD 패치 R20 (임상 표준, 현재 가능)

- 파장: 1064nm(검정), 532nm(빨강), 755nm(초록/파랑), 694nm(초록/청록)
- PFD 패치: CO2 서리 제거 20분→2분 단축 → 동일 세션 4패스
- 효과: 검정 단색 95%, 다색 50~75%
- 세션: 6~15회, 간격 8~12주
- 흉터율: <1% (피코초, 숙련 시술자)
- 한계: 흰색(역산화), 형광(흡수 불가), 초록(프탈로시아닌 저항)

### 경로 2: 나노입자 강화 광열 (전임상, 5~10년 내)

- 금 나노로드(AuNR) 진피 주입 + NIR(750~900nm) 조사
- 국소 플라즈몬 공명(LSPR) → 잉크 인근 극한 국소 가열
- 기존 레이저 대비 파쇄 효율 2~5배 (in vitro)
- 장점: 낮은 플루언스 → Fitzpatrick V~VI 멜라닌 경쟁 해소
- 현 상태: 동물 실험 단계, 인간 데이터 없음

### 경로 3: 면역학적 대식세포 사이클 차단 (전임상, 핵심 돌파점)

**100% 도달의 진짜 열쇠.**

#### 3층 차단 아키텍처

```
  층 3: 배출 가속 (림프관 → 림프절 → 처리)
  층 2: 모집 차단 (CCR2 길항 → 단핵구 진입 불가)
  층 1: 보유자 제거 (클로드로네이트 → 대식세포 사살)
```

#### 전략 A: CCR2 길항제 — 가장 안전한 핵심

- CCR2 = 단핵구 표면 수용체, CCL2(MCP-1) = 진피 손상 시 모집 신호
- CCR2 길항 → 혈중 단핵구가 진피에 진입 불가 → 재포획 차단
- 기존 대식세포를 죽이지 않음 (CSF1R 대비 온건)
- 후보 약물: cenicriviroc (NASH 임상2상 완료, 안전성 풍부)
- 투여: 레이저 24시간 전 ~ 시술 후 7일 (골든 윈도우 커버)
- 부작용: 두통 11%, 오심 8% (경미)
- 현재 상태: 문신 제거 맥락 직접 연구 0건

#### 전략 B: CSF1R 국소 억제

- PLX5622 IC50 = 0.016 uM → 조직 대식세포 90% 이상 소멸
- 전신 투여 불가 (pexidartinib 간독성 6.1%)
- 국소 전달: PLGA 나노입자 + 마이크로니들
- 현재 상태: 국소 CSF1R + 문신 연구 없음

#### 전략 C: 클로드로네이트 리포솜 피내 주사

- 활발히 포식하는 대식세포만 선택적 사살
- 잉크 보유 대식세포 = 가장 활발히 포식한 세포 = 가장 잘 삼킴
- → 잉크 들고 있는 놈이 독약도 가장 많이 먹는 자연적 선택성
- 섬유아세포/호중구/내피세포 무영향
- 클로드로네이트: 골다공증 치료제, 특허 만료, 제네릭 사용 가능

### 경로 4: 바르는 약물 (연구 초기)

- 시판 "타투제거 크림": FDA 승인 없음, 효과 없음 (사기 수준)
- 진짜 가능한 경로: 마이크로니들 어레이 + 약물 (크림 단독 진피 미도달)
- 탑재 후보: 클로드로네이트, CSF1R 억제제, 히알루로니다제, 림프촉진제
- 이온토포레시스 겔: 경로 6과 결합 가능

### 경로 5: 먹는 약물 (이론적/보조)

- 경구 비스포스포네이트: 전신 대식세포 영향 → 면역 위험
- 경구 CSF1R 억제: 간독성 → 문신 목적 부적합
- 현실적 보조: 림프순환 촉진제(루틴/디오스민) + 수분 + 유산소
- 단독 문신 제거 불가, 레이저 후 배출 보조

### 경로 6: 전기구동 잉크 이동 (실험적)

- 이온토포레시스: 하전 금속 입자(Fe, Cr, Cu) 림프관 방향 유도
- 전기천공: 대식세포 세포막 일시 개방 → 잉크 강제 방출
- 저강도 초음파(LIUS): 진피 간극 확대 → 입자 이동 촉진

## 왜 클로드로네이트가 "자연적 정밀 무기"인가

```
클로드로네이트 리포솜(200~400nm) 진피 주입
   ↓
"활발히 포식하는 세포"만 삼킨다
   ↓
잉크 보유 대식세포 = 활발히 포식한 대식세포
= 클로드로네이트 리포솜을 가장 잘 삼키는 세포
   ↓
잉크 들고 있는 놈이 독약도 가장 많이 먹는다
   ↓
선택적 사살 → 잉크 세포외 방출
```

비표적 세포 안전성:

- 내피세포: 리포솜 안 삼킴 → 무사

# 도메인 3: CCR2 길항제 국소 제형

> 성숙도 30% | 5~10년 | 혁신성 100% (세계 최초, 논문 0건)

## 핵심 통찰

Baranska 2018이 확인한 사실: **새 대식세포는 CCR2 의존적으로 혈중 Ly-6C(hi) 단핵구에서 분화한다.**

→ CCR2를 막으면 보급선이 끊긴다.
→ 기존 대식세포가 죽어도 후속 부대가 오지 않는다.
→ 자유 잉크가 재포획 없이 림프관으로 배출.

## CCR2 vs CSF1R — 왜 CCR2가 맞는가

| 항목 | CCR2 길항 | CSF1R 억제 |
|------|-----------|-----------|
| 작용 대상 | 이동중 단핵구 | 모든 조직 대식세포 |
| 기존 대식세포 | 유지 | 90% 사살 |
| 면역 방어 | 대부분 유지 | 심각하게 저하 |
| 간독성 | 경미 (두통 8%) | 심각 (6.1%) |
| 복원 시간 | 중단 후 즉시 | 10~20일 |
| 비유 | 보급선 차단 | 핵폭격 |

CCR2 = 외과적 정밀 타격. CSF1R = 카펫 폭격.
문신 제거는 전쟁이 아니라 수술이다.
