### 생명 코드 n=6 기본 래더

```
DNA 염기 = tau = 4
코돈 길이 = n/phi = 3
총 코돈 수 = 2^n = 64
아미노산 = J2 - tau = 20
포도당 원자 = J2 = 24 (C6H12O6)
광합성 계수 = 전수 n=6 (6CO2 + 6H2O -> C6H12O6 + 6O2)

```
