def sigma(n): return sum(d for d in range(1, n+1) if n % d == 0)
def tau(n):   return sum(1 for d in range(1, n+1) if n % d == 0)
def phi(n):
    from math import gcd
    return sum(1 for k in range(1, n+1) if gcd(k, n) == 1)
def sopfr(n):
    s, m, d = 0, n, 2
    while d*d <= m:
        while m % d == 0: s += d; m //= d
        d += 1
    if m > 1: s += m
    return s

def sigma(n): return sum(d for d in range(1, n+1) if n % d == 0)
def tau(n):   return sum(1 for d in range(1, n+1) if n % d == 0)
def phi(n):
    from math import gcd
    return sum(1 for k in range(1, n+1) if gcd(k, n) == 1)
def sopfr(n):
    s, m, d = 0, n, 2
    while d*d <= m:
        while m % d == 0: s += d; m //= d
        d += 1
    if m > 1: s += m
    return s
def jordan2(n):
    r = n*n
    m, d = n, 2
    while d*d <= m:
        if m % d == 0:
            r = r * (1 - 1/(d*d))
            while m % d == 0: m //= d
        d += 1
    if m > 1: r = r * (1 - 1/(m*m))
    return int(round(r))

## PL-1: 유전 코돈 최소성 한계

**정리**: 20종 아미노산을 코딩하기 위한 최소 코돈 길이는 정확히 3이다 (n/phi=3).

**증명**:
```
  주어진: tau(6)=4 종류의 염기, J₂(6)-tau(6)=20 아미노산 + stop 코돈

  k-base 코돈으로 코딩 가능한 최대 아미노산 수 = 4^k - stop_codons

  k=1: 4^1 = 4 < 20+stop  (불충분)
  k=2: 4^2 = 16 < 20+stop  (불충분)
  k=3: 4^3 = 64 >= 20+3stop = 23  (충분, 중복 허용)
  k=4: 4^4 = 256 >> 23  (과잉, 정보 낭비)

  정보이론적 최적:
    log_4(23) = 2.26... -> ceil = 3

  k = 3 = n/phi(6) = 6/2 = 3이 유일한 최적해.
  k < 3: 불가능 (코딩 용량 부족)
  k > 3: 비효율 (복제 에러 증가, 에너지 낭비)

  따라서 코돈 길이 3 = n/phi는 정보이론적 필연이다.  []
```

## PL-2: 아미노산 20종 최적성

    2. 번역 정확도: 아미노산 수가 많을수록 tRNA 오인식 확률 증가
    3. 대사 비용: 합성 경로가 많을수록 에너지 비용 증가

    - 30종: 번역 오류율 ~50% 증가, 대사 비용 과다

  이 중복도는 오류 교정(wobble)에 최적.

```

## PL-4: 탄소 Z=6 유기화학 필연

**정리**: 복잡한 유기분자 형성이 가능한 원소는 사실상 탄소(Z=6)뿐이다.

**증명**:
```
  필요 조건: 4가 결합 + 사슬 형성 + 다중결합

  C (Z=6): 2s²2p² → sp3 (4결합), sp2 (3+pi), sp (2+2pi)
    - 사슬: 무한대 (C-C 결합 346 kJ/mol, 안정)
    - 다중결합: C=C (614), C≡C (839) → 풍부한 화학
    - 생체분자: DNA, 단백질, 지질, 당 전부 탄소 골격

  Si (Z=14): 3s²3p² → 4결합 가능
    - Si-Si 결합: 226 kJ/mol (C-C의 65%) → 긴 사슬 불안정
    - Si=Si: 극히 불안정 (no pi bond overlap at 3p level)
    - SiO₂ 고체 (vs CO₂ 기체) → 용액 화학 불가

## PL-5: 광합성 양자 한계 (8 광자)

**정리**: O₂ 1분자 발생에 필요한 최소 광자 수는 8이다 (sigma-tau=8).

**증명**:
```
  광합성 전체 반응: 2H₂O → O₂ + 4H⁺ + 4e⁻

  전자 전달 체인:
    - PS II: H₂O → PQ (4 photons for 4 electrons)
    - PS I:  PQ → NADPH (4 photons for 4 electrons)
    - 총: 8 photons / O₂ = sigma(6)-tau(6) = 8

  열역학적 하한:
    - O₂ 생성 자유에너지: +474 kJ/mol
    - 적색광 에너지: ~170 kJ/mol/photon
    - 최소 photons = ceil(474/170) = 3 (열역학적 절대 하한)

  실제 하한 (양자 역학적):
    - 각 전자 전달에 1 photon 필요 (1:1 양자 수율)
    - 4 전자 × 2 광계 = 8 photons (sigma-tau=8)
    - 이것은 양자역학적 최소이지 열역학적 최소가 아님

  실측: Emerson number = 8-12 (효율 손실 포함)
  이론 최소 = 8 = sigma-tau  []
```

**n=6 근거**: tau(6)=4. BT-51 1단계.
**검증**: NCBI GenBank, 극한 미생물 포함 전수 조사.
**반증 조건**: 5번째 자연 염기가 발견되면 FAIL. (단, 후성유전 변형 제외)

### TP-BIO-02: 모든 생물의 코돈 = n/phi=3 염기
**예측**: 자연의 모든 단백질 합성은 3-염기 코돈을 사용한다.
**n=6 근거**: n/phi = 6/2 = 3. BT-51 2단계.
**검증**: tRNA anticodon 전수 조사, mitochondrial 코드 포함.
**반증 조건**: 2-base 또는 4-base codon이 단백질 합성 표준이 되면 FAIL.

### TP-BIO-03: 표준 아미노산 = J₂-tau = 20
**예측**: 단백질을 구성하는 표준 아미노산은 정확히 20종이다.
**n=6 근거**: J₂(6)-tau(6) = 24-4 = 20. BT-51 4단계.
**검증**: UniProt 표준 아미노산 목록.
**반증 조건**: 21번째 표준 아미노산(selenocysteine/pyrrolysine 제외)이 보편 코드에 편입되면 FAIL.

### TP-BIO-04: 포도당 C₆H₁₂O₆ 원자수 = J₂=24
**예측**: 포도당의 총 원자 수 = 6+12+6 = 24 = J₂(6).
**n=6 근거**: J₂(6)=24. BT-101.
**검증**: 화학식 직접 계산.
**반증 조건**: 포도당 이외의 주요 생체 에너지원이 24 원자가 아니면 CLOSE.

### TP-BIO-05: 벌집 육각형 = n=6 최적 공간충전
**예측**: 자연의 벌집은 정확히 정육각형이며, 이것이 최소 둘레 분할이다.
**n=6 근거**: n=6. BT-122. Hales (2001) 증명.
**검증**: Hales의 Honeycomb Conjecture 증명 (2001, Annals of Mathematics).
**반증 조건**: 정육각형보다 효율적인 2D 분할이 발견되면 FAIL.

### TP-BIO-06: DNA 이중나선 주기 = sigma-phi = 10 bp/turn
**예측**: B-form DNA의 나선 주기는 약 10 bp/turn이다.
**n=6 근거**: sigma-phi = 12-2 = 10.
**검증**: X-ray crystallography, NMR 구조 데이터.
**반증 조건**: 주기가 8 또는 12로 확인되면 FAIL.

## Tier 2: 실험실 검증 (연구 장비)

### TP-BIO-07: 세포 분열 = tau=4 단계 (M기)
**예측**: 유사분열(M기)은 정확히 4단계이다 (전기/중기/후기/말기).
**n=6 근거**: tau(6)=4.
**검증**: 세포생물학 교과서 및 라이브셀 이미징.
**반증 조건**: 5단계 또는 3단계로 재분류되면 CLOSE.

### TP-BIO-08: 광합성 6CO₂ + 12H₂O → C₆H₁₂O₆ + 6O₂ + 6H₂O 화학양론
**예측**: 광합성 총괄 반응의 모든 계수가 n=6 상수이다 (6, 12, 6, 12, 6, 6, 6).
**n=6 근거**: BT-103. 7개 계수 100% n=6.
**검증**: 광합성 표준 반응식 (Calvin cycle + light reactions).
**반증 조건**: 화학양론이 다르게 밝혀지면 FAIL.

### TP-BIO-09: 광합성 양자수율 = sigma-tau = 8 광자/O₂
**예측**: 산소 1분자 발생에 필요한 최소 광자 수 = 8.
**n=6 근거**: sigma-tau = 12-4 = 8. BT-101.
**검증**: Emerson & Arnold (1932), 현대 측정.
**반증 조건**: 최소 광자 수가 6 또는 10으로 확정되면 FAIL.

### TP-BIO-10: Hachimoji DNA (8-letter) = sigma-tau = 8
**예측**: 합성 Hachimoji DNA는 정확히 8개 염기를 사용한다.
**n=6 근거**: sigma-tau = 8. 확장 유전 알파벳의 최적 크기.
**검증**: Hoshika et al. (2019) Science. 4자연+4합성=8.
**반증 조건**: 6-letter 또는 12-letter 합성 DNA가 더 안정적이면 CLOSE.

### TP-BIO-11: 생체 원소 CHNOPS = n=6 필수 원소
**예측**: 생명의 필수 원소는 정확히 6종 (C, H, N, O, P, S).
**n=6 근거**: n=6.
**검증**: 생화학 교과서, 원소 조성 분석.
**반증 조건**: 7번째 필수 원소가 추가되면 CLOSE (미량 원소는 제외).

## Tier 3: 장기 검증 (전문 연구)

### TP-BIO-12: DNA reading frame = n=6 (3 forward + 3 reverse)
**예측**: 이중가닥 DNA의 reading frame은 정확히 6개이다.
**n=6 근거**: n=6 = n/phi × phi = 3 × 2.
**검증**: ORF finder 알고리즘, 모든 genome.
**반증 조건**: 4th frame이 단백질 합성에 사용되면 FAIL.

### TP-BIO-13: tRNA 종 = 2^n = 64 미만 (중복 코돈)
**예측**: 실제 tRNA 종 수는 64보다 적으며, wobble 규칙으로 축소된다.
**n=6 근거**: 2^n = 64 코돈 중 중복 허용.
**검증**: tRNAdb, GtRNAdb 전수 조사.
**반증 조건**: tRNA 종이 64를 초과하면 FAIL.

### TP-BIO-14: CRISPR PAM = n/phi = 3 bp
**예측**: CRISPR-Cas9의 PAM 서열은 3 bp이다.
**n=6 근거**: n/phi = 3. BT-51 코돈 구조와 동일.
**검증**: SpCas9 PAM = NGG (3bp).
**반증 조건**: 주요 Cas 시스템의 PAM이 5bp+ 표준이 되면 CLOSE.

### TP-BIO-15: 눈 결정 = n=6 대칭
**예측**: 눈 결정(snowflake)은 정확히 6-fold 대칭이다.
**n=6 근거**: n=6. BT-122. 물 분자 수소결합 각도에서 필연.
**검증**: Bentley (1931), Libbrecht (2005) 눈 결정 분류.
**반증 조건**: 5-fold 또는 8-fold 눈 결정이 자연에서 발견되면 FAIL.

## Tier 4: 미래 예측 (10년+)

### TP-BIO-16: 합성 생물학 최적 코돈 테이블 = 2^n = 64 유지
**예측**: 합성 생물학에서 확장 유전 코드도 64-코돈 프레임을 유지한다.
**n=6 근거**: 2^n = 64.
**검증**: xenobiology 연구 결과 (2030+).
**반증 조건**: 100+ 코돈 시스템이 안정적으로 동작하면 CLOSE.

### TP-BIO-17: Carbon Z=6 기반 생명 보편성
**예측**: 외계 생명 발견 시에도 탄소(Z=6) 기반일 확률이 압도적이다.
**n=6 근거**: Z=6 = n. BT-27, BT-85.
**검증**: 외계 생명 탐사 결과 (JWST, Europa Clipper).
**반증 조건**: 규소(Z=14) 기반 생명이 발견되면 FAIL.

### TP-BIO-18: 뇌 피질 n=6 층 보편성
**예측**: 포유류 신피질은 예외 없이 정확히 6층이다.
**n=6 근거**: n=6. BT-210.
**검증**: 신경해부학 전수 조사 (Brodmann atlas).
**반증 조건**: 7층 또는 5층 신피질 포유류가 발견되면 FAIL.

### TP-BIO-19: 진핵세포 핵심 구획 = n=6
**예측**: 진핵세포의 주요 구획(핵, 미토콘드리아, ER, 골지, 리소좀, 세포질)은 6개이다.
**n=6 근거**: n=6.
**검증**: 세포생물학 표준 분류.
**반증 조건**: 주요 구획이 8개로 재분류되면 CLOSE.

### TP-BIO-20: 생물 분류 주요 계 = n=6 (Cavalier-Smith)
**예측**: Cavalier-Smith의 6계 분류가 가장 안정적이다.
**n=6 근거**: n=6.
**검증**: 분류학 합의 추이.
**반증 조건**: 3-domain 체계가 완전히 대체하면 CLOSE.
