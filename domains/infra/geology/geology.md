## 불가능성 정리

### 정리 GEO-IMP-1: 지진파 최소 분류

```
  주장: 1종 지진파로는 3D 지구 내부 구조 결정 불가
  증명:
    P파: 종파 (속도 VP → 비압축률+밀도)
    S파: 횡파 (속도 VS → 강성률+밀도)
    P파만: 1 방정식, 2 미지수 (비압축률, 밀도) → 불결정
    P+S: 2 방정식, 2 미지수 → 결정 가능
    → φ=2 체적파가 지구 내부 결정의 최소 완전 집합 ∎
```

### 정리 GEO-IMP-2: 판경계 3유형 완전성

```
  주장: 2유형 이하 판경계로는 판구조론 기술 불가능
  증명:
    2D 구면 위 강체 운동 = 회전 (오일러 정리)
    두 판 경계: 접근(수렴)/이탈(발산)/미끄러짐(보존)
    이 3유형 = 상대 운동의 완전 분류 (접선/법선 분해)
    2유형: 1 운동 미기술 → 판 운동 불완전
    n/φ = 3 = 판경계 완전 분류 ∎
```

## 검증 코드 (Python)

```python
import math

def sigma(n): return sum(d for d in range(1, n+1) if n % d == 0)
def tau(n):   return sum(1 for d in range(1, n+1) if n % d == 0)
def phi(n):   return sum(1 for k in range(1, n+1) if math.gcd(k, n) == 1)
def sopfr(n):
    s, m, d = 0, n, 2
    while d*d <= m:
        while m % d == 0: s += d; m //= d
        d += 1
    if m > 1: s += m
    return s

assert sigma(6) * phi(6) == 6 * tau(6)

tests = [
    ("불연속면", 6, 6),
    ("지구 내부 층", 5, sopfr(6)),
    ("체적파", 2, phi(6)),
    ("지진파 유형", 4, tau(6)),
    ("판경계 유형", 3, 6//phi(6)),
    ("지각 원소", 6, 6),
    ("규산염 분류", 6, 6),
    ("MMI 진도", 12, sigma(6)),
    ("대륙", 6, 6),
    ("암석 순환", 3, 6//phi(6)),
    ("단층 유형", 3, 6//phi(6)),
    ("내핵 Fe CN", 12, sigma(6)),
    ("지각 두께 (km)", 35, sigma(6)*6//phi(6)),  # σ·n/φ=36 근사
]

print("=" * 60)
print("HEXA-GEO n=6 지질학 EXACT 검증")
print("=" * 60)
passed = sum(1 for _, o, p in tests if o == p)
for name, obs, pred in tests:
    mark = "PASS" if obs == pred else "MISS"
    print(f"  [{mark}] {name}: {obs} vs {pred}")
print(f"\n결과: {passed}/{len(tests)} PASS")
```
