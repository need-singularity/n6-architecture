## 핵심 상수 매핑

```
n=6           : 잉크 채널 수 (CMYK + O + V), 멀티머티리얼 동시 토출 노즐 그룹
tau=4         : 잉크젯 헤드 단계 (가열→방울형성→토출→안정화), 적층 단계 (예열→분사→경화→냉각)
sopfr=5       : 후처리 공정 (세척→건조→경화→연마→코팅)
n/phi=3       : 잉크 상태 (액체/반고체/고체), 광경화 파장대 (UV-A/B/C)
phi=2         : 토출 모드 (DOD/Continuous), 노즐 행렬 축
sigma=12      : 전체 공정 (모델링→슬라이싱→재료→예열→토출→경화→층이동→센싱→보정→후처리→검사→포장)
sigma-phi=10  : 동시 모니터링 센서 채널 (온도/유량/점도/액적/평탄/위치/광량/진동/전압/이미지)
J_2=24        : 표준 경화 시간(h), 헤드 채널 폭 그룹
```
