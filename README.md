### Design Space Exploration (DSE) 방법론

```
  ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
  │  소재    │───▶│  공정    │───▶│  코어    │───▶│  칩     │───▶│ 시스템   │
  │ 후보 K₁ │    │ 후보 K₂ │    │ 후보 K₃ │    │ 후보 K₄ │    │ 후보 K₅ │
  └─────────┘    └─────────┘    └─────────┘    └─────────┘    └─────────┘
       │              │              │              │              │
       └──────────────┴──────────────┴──────────────┴──────────────┘
                          K₁×K₂×K₃×K₄×K₅ 전수 탐색
                                  ↓
                     ┌──────────────────────┐
                     │  Pareto Frontier     │
                     │  n6_EXACT · 성능 ·   │
                     │  전력 · 면적 · 비용  │
                     └──────────────────────┘
```

| 항목 | 내용 |
|------|------|
| **원칙** | 각 레벨(소재/공정/코어/칩/시스템)마다 후보군 정의 → 전수 조합 탐색 |
