## 0. Purpose and Scope

Among the Millennium problems, Poincaré (demonstrated), Yang-Mills (4-manifold structure), and Hodge
(algebraic-geometry / topology interface) all rest on topology. At the P1 stage we cover six foundations.

1. Homotopy: paths, loops, and the definition and functoriality of π_n
2. Singular homology H_n(X; ℤ) — chain complex and boundary homomorphism
3. Cohomology H^n(X; ℤ) — dual, cup product, ring structure
4. Fundamental group π_1 and the van Kampen theorem
5. CW complexes and cellular computation
6. Intersection form Q_M of 4-manifolds and its classification

These notes first define the three "basic languages" of algebraic topology (π_n, H_n, H^n), make the
bridges between them explicit (Hurewicz, universal coefficient, Künneth), and then specialise to
4-manifolds.

## 1. Homotopy

### 1.1 Continuous Deformation and Equivalence

For topological spaces X, Y and continuous maps f, g: X → Y, a homotopy is a continuous function
H: X × [0,1] → Y with H(x, 0) = f(x), H(x, 1) = g(x). Write f ~ g.

### 1.2 Higher Homotopy Groups π_n

Fix basepoint x₀ ∈ X. π_n(X, x₀) = [(S^n, s₀), (X, x₀)], the homotopy equivalence classes of basepoint-
preserving continuous maps from the sphere S^n into X. At n=1 this is the fundamental group; at n≥2 it is
abelian (Eckmann-Hilton).

### 1.3 Functoriality and Product Spaces

π_n(-, x₀) is a covariant functor, and π_n(X × Y, (x_0,y_0)) ≅ π_n(X,x₀) × π_n(Y,y₀). For a connected Lie
group G, π_1(G) is abelian (general topological π_1 can be non-abelian).

### 1.4 π_n of Spheres — Computational Difficulty

- π_n(S^n) = ℤ (Hurewicz, n≥1, simply connected space)
- π_3(S²) = ℤ (Hopf invariant, 1931 Hopf)
- Computing π_{n+k}(S^n) is very difficult depending on n, k (Serre finiteness theorem: finite for k>0)

For this reason, attempts to classify spaces by π_n alone typically hit uncomputability. Homology is the
auxiliary language that restores computability.

(Source: Hatcher §4.1, notably Serre's 1953 PhD thesis)

### 1.5 CW Approximation and Weak Equivalence

Every topological space has a weak-equivalence CW approximation (Whitehead: a continuous map inducing
π_n-isomorphism is a homotopy equivalence between CWs). Homotopy computations therefore reduce to cellular
structure.

## 2. Singular Homology

### 2.1 Standard Simplex and Singular Chains

Δ^n = {(t_0,...,t_n) ∈ ℝ^{n+1} : t_i ≥ 0, Σ t_i = 1} is the standard n-simplex. A singular n-chain is a
finite free abelian group C_n(X) generated by continuous maps σ: Δ^n → X.

### 2.2 Boundary Homomorphism

∂_n: C_n(X) → C_{n-1}(X), σ ↦ Σ_{i=0}^n (-1)^i σ ∘ d_i (d_i omits the i-th vertex). ∂_{n-1} ∘ ∂_n = 0.
Hence C_•(X) is a chain complex.

### 2.3 Definition of H_n(X; ℤ)

```
  Z_n = ker ∂_n   (cycles)
  B_n = im ∂_{n+1} (boundaries)
  H_n(X; ℤ) = Z_n / B_n
```

H_0(X; ℤ) = ℤ^{c(X)} (c the number of path components). H_1(X; ℤ) ≅ π_1(X)^{ab} (abelianisation,
Hurewicz n=1).

### 2.4 Mayer-Vietoris and Excision

For an open cover X = A ∪ B there is a long exact sequence

```
  ... → H_n(A ∩ B) → H_n(A) ⊕ H_n(B) → H_n(X) → H_{n-1}(A ∩ B) → ...
```

Excision: for closed Z ⊂ int(A), H_n(X, A) ≅ H_n(X\Z, A\Z). The two theorems allow recursive computation
of homology on CW complexes.

(Source: Bredon §IV.8, Hatcher §2.2)

### 2.5 Hurewicz Theorem

If X is (n-1)-connected (π_i(X) = 0, 0 < i < n), the natural map π_n(X) → H_n(X; ℤ) is an isomorphism for
n ≥ 2 and an isomorphism after abelianisation for n=1.

## 3. Cohomology

### 3.1 Dual Cochains

C^n(X; ℤ) = Hom(C_n(X), ℤ), coboundary δ^n = (∂_{n+1})*. This gives a cochain complex and
H^n(X; ℤ) = ker δ^n / im δ^{n-1}.

### 3.2 Universal Coefficient Theorem (UCT)

```
  0 → Ext(H_{n-1}(X; ℤ), ℤ) → H^n(X; ℤ) → Hom(H_n(X; ℤ), ℤ) → 0   (short exact)
```

This sequence splits naturally (unnaturally at the level of splittings). If the coefficients are changed
to a field k the Ext term vanishes and H^n(X; k) = Hom(H_n(X; k), k).

### 3.3 Cup Product

∪: H^p(X) × H^q(X) → H^{p+q}(X). Associative with unit (1 ∈ H^0(X)=ℤ), but in general
ab = (-1)^{pq} ba (graded-commutative). So H^*(X; k) is a graded ring.

Example: H^*(ℂP^n; ℤ) = ℤ[α]/α^{n+1}, deg α = 2. This truncated polynomial structure carries information
not visible from homology alone (ring vs. module).

### 3.4 Poincaré Duality

For a closed oriented n-manifold M,

```
  H^k(M; ℤ) ≅ H_{n-k}(M; ℤ)     (cap product with the fundamental class [M])
```

In the non-orientable case the same holds with ℤ/2 coefficients. This is a foundational tool for
4-manifold research.

### 3.5 de Rham Cohomology — Smooth Manifolds

For a smooth M, the de Rham complex Ω^0(M) → Ω^1(M) → ...

```
  H^n_{dR}(M; ℝ) = ker d / im d
```

de Rham theorem: H^n_{dR}(M; ℝ) ≅ H^n(M; ℝ) (real-coefficient singular cohomology). Chern-Weil
characteristic classes in gauge theory rest on this identification.

## 4. Fundamental Group and van Kampen

### 4.1 Redefinition of π_1

For a loop γ: [0,1] → X, [γ] is the path-homotopy class. Multiplication is concatenation; identity is the
constant loop. π_1(X, x₀) is in general non-abelian.

### 4.2 van Kampen Theorem

If X = U ∪ V with U, V open path-connected and U ∩ V path-connected, x₀ ∈ U ∩ V, then

```
  π_1(X, x₀) ≅ π_1(U, x₀) *_{π_1(U∩V, x₀)} π_1(V, x₀)
```

(amalgamated free product, a pushout of groups)

### 4.3 Representative Calculations

- π_1(S^1) = ℤ
- π_1(S^n) = 0  (n ≥ 2)
- π_1(T^2) = ℤ²
- π_1(Σ_g) = ⟨a_1,b_1,...,a_g,b_g | [a_1,b_1]...[a_g,b_g]⟩ (genus-g surface)
- π_1(ℝP^n) = ℤ/2  (n ≥ 2)

### 4.4 Covering-Space Correspondence

If X is path-connected, locally path-connected, and semilocally simply connected, there is a Galois
correspondence between subgroups of π_1(X, x₀) and connected coverings of X. The universal cover X̃
corresponds to π_1(X).

## 5. CW Complexes and Cellular Homology

### 5.1 CW Definition

X⁰ ⊂ X¹ ⊂ X² ⊂ ... strictly increasing, X^n = X^{n-1} ∪_{∂D^n_α} D^n_α, attaching n-cells via attaching
maps φ_α: ∂D^n_α = S^{n-1} → X^{n-1}. X = colim X^n.

### 5.2 Cellular Homology

C^{CW}_n(X) = ℤ^{|n-cells|} (free abelian group). Boundaries are given by the degrees of the attaching
maps.

```
  H_n^{CW}(X) ≅ H_n(X; ℤ)       (isomorphic to singular)
```

In practice simplicial complexes, Δ-complexes, and CW all give the same homology, so the smallest cellular
structure is the efficient one to compute.

### 5.3 Euler Characteristic

χ(X) = Σ (-1)^n (number of n-cells) = Σ (-1)^n rank H_n(X; ℤ) = Σ (-1)^n dim H^n(X; k). All three
expressions give the same value (Betti numbers and rank agree algebraically on a chain complex).

## 6. Special Structure of 4-Manifolds

### 6.1 Intersection Form Q_M

M a closed oriented 4-manifold. Poincaré duality gives a symmetric bilinear form on H^2(M; ℤ):

```
  Q_M: H^2(M; ℤ) × H^2(M; ℤ) → ℤ,    Q_M(α, β) = ⟨α ∪ β, [M]⟩
```

This form is the fundamental invariant of M. Classifying ℤ-valued symmetric bilinear forms of finite rank
(= b_2) is the linear-algebraic core of 4-manifold classification.

### 6.2 Classification Theorem for Intersection Forms

For a simply connected closed oriented 4-manifold M, Q_M is

- Even: ⟨v, v⟩ ∈ 2ℤ for all v, the spin condition
- Signature σ(Q) = b_2^+ - b_2^-
- Diagonalisability: rank, signature, and type (even/odd) determine the isomorphism class (independent of
  the non-degenerate structure; Serre's theorem).

### 6.3 Freedman's Theorem (1982)

For a simply connected closed oriented 4-manifold, the topological homeomorphism class is determined by
(Q_M, Kirby-Siebenmann invariant ks ∈ ℤ/2). Two manifolds M, N with Q_M ≅ Q_N are topologically
homeomorphic (spin case requires ks to match).

This theorem reduces the classification of simply connected topological 4-manifolds to an algebraic
problem.

### 6.4 Donaldson's Theorem (1983)

If a simply connected closed oriented 4-manifold M admits a smooth structure, then Q_M must be
diagonalisable whenever it is positive definite or negative definite. That is, exotic positive-definite
forms such as 8+8 (-E_8) cannot be realised smoothly.

Result: among the forms topologically (Freedman) admissible, only a small fraction admits a smooth
structure. Infinitely many inequivalent smooth structures can share the same topological type (even ℝ⁴
alone admits infinitely many exotic smooth structures) — a 4D-only phenomenon.

### 6.5 Seiberg-Witten Invariants

Since the 1994 introduction of the Seiberg-Witten equations, many more smooth structures on 4-manifolds
have been classified via Seiberg-Witten invariants, which are easier to compute than Donaldson invariants.
The basis is U(1)-gauge theory + the Dirac equation.

## 7. Special Structure of 3-Manifolds (Poincaré Connection)

### 7.1 Classification of Simply Connected 3-Manifolds

Via Thurston's geometrization candidate (1982) → Perelman's demonstration (2003), a closed 3-manifold
decomposes into finitely many "geometric pieces" (8 geometries). A simply connected closed 3-manifold is
homeomorphic to S³ (the original Poincaré candidate, 1904).

### 7.2 Ricci Flow Preparation

With (g_ij) a Riemannian metric,

```
  ∂g_{ij}/∂t = -2 R_{ij}          (Ricci flow)
```

Perelman's entropy F = ∫(R + |∇f|²)e^{-f} dV and its monotonicity control the "scale". The concrete
argument is treated in PROB-P1-7 (Poincaré deep dive).

## 12. Further Topic — Spectral Sequence Primer

Outside the P1 scope but unavoidable in P2, the Serre spectral sequence and Adams spectral sequence basics
are recorded here in appendix form.

### 12.1 Serre Spectral Sequence

For a fibration F → E → B,

```
  E_2^{p,q} = H^p(B; H^q(F; G)) ⟹ H^{p+q}(E; G)
```

This sequence computes the cohomology of E by a "progressive approximation" from those of B and F. On each
page E_r → E_{r+1}, take the d_r-differential boundary.

Representative example: S¹ → S³ → S² (Hopf fibration). Serre can recompute H^*(S³).

### 12.2 Leray Spectral Sequence

(Generalisation to a continuous map f: X → Y.) The same structure appears in sheaf cohomology.

### 12.3 Adams Spectral Sequence

Approximates stable homotopy groups π^s_*(X) via Ext_{A}(H*(X; 𝔽_p), 𝔽_p). The Steenrod algebra A is
central. Concrete computations are very involved.

- Partial computation of stable homotopy groups of spheres π^s_n

- Structural link to Seiberg-Witten

## 13. Further Topic — Kirby Diagrams and 4-Manifolds

### 13.1 Kirby Diagram

Represent a 4-manifold pictorially as an attachment of 1- and 2-handles. 2-handles appear as framed knots,
1-handles as "dotted unknots". Standard reference: Gompf-Stipsicz "4-Manifolds and Kirby Calculus"
AMS 1999.

### 13.2 Exotic ℝ⁴

ℝ⁴ admits infinitely many smooth structures inequivalent to the standard one — a consequence of Donaldson's
theorem. A 4D-exclusive anomaly.
