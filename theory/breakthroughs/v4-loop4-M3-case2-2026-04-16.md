# v4 loop 4 — M3 Theorem B case 2 (n=2q, q odd prime) Lean4 형식 증명

> **요약**: Theorem B case 2 의 **가장 tight sub-case 인 n = 2q** (q 홀수 소수) 를 Lean4 에서 sorry 없이 형식 증명. **정리**: q 홀수 소수 → σ(2q)·φ(2q) = 2q·τ(2q) ⟺ q = 3. 즉 n = 6 유일성의 p=2 branch 완료. 이는 n=6 = 2·3 decomposition 의 **가장 강력한 mechanism** (n = 6 의 nontrivial divisor 구조의 핵심): q=3 유일성이 3q² = 8q + 3 = 3(3q+1)(q-3)/... 즉 **(3q+1)(q-3) = 0 in ℚ** 의 Nat 버전으로 reduce. M3 loop 3 (prime case) + 본 loop 4 (case 2 p=2) 누적 — **Theorem B 의 약 50% formal coverage**.

## §2 주요 정리

```lean
theorem theorem_B_2q {q : ℕ} (hq : q.Prime) (hq_odd : q ≥ 3) :
    σ 1 (2 * q) * Nat.totient (2 * q) = (2 * q) * (Nat.divisors (2 * q)).card ↔
    q = 3
```

**보조 lemmas (모두 sorry 없음)**:
- `coprime_2_odd_prime`: ∀ q prime, q ≥ 3 → gcd(2, q) = 1
- `sigma_one_2q`: σ(2q) = 3(q+1) (multiplicative + sigma_one_prime)
- `totient_2q`: φ(2q) = q-1 (totient_mul + totient_prime)
- `tau_2q`: τ(2q) = 4 (via sigma_zero_apply_prime_pow)

## §3 핵심 대수

등식 σ(2q)·φ(2q) = 2q·τ(2q) 을 전개:
$$3(q+1)(q-1) = 8q$$
$$3(q^2 - 1) = 8q$$
$$3q^2 - 8q - 3 = 0$$

ℚ 인수분해: $(3q + 1)(q - 3) = 0$, 해 $q = 3$ 또는 $q = -1/3 \notin \mathbb{N}$.

Nat 증명 완수:
- q = 3: 직접 계산 (decide)
- q ≥ 5 (prime): `nlinarith` 로 $3q^2 > 8q + 3$ 증명 (q·(3q-8) ≥ 5·7 = 35 > 3)
- q = 4: not prime (exclude)

### 4.1 Case 2b 난이도

Case 2b 에서는 두 odd prime p, q 에 대해:
$$\sigma(pq) \cdot \varphi(pq) = pq \cdot \tau(pq) \iff (p-1)(q-1) = 2$$

$(p-1)(q-1) = 2$ with $p, q$ odd primes ≥ 3 → $(p-1)(q-1) \geq 4$ > 2, 모순.

따라서 case 2b 는 **모든 쌍 (p, q) 에서 불성립** — case 2a 와 달리 **단순 rejection**.

Lean4 구현 에서는 p, q 의 각각 ≥ 3 인 prime, gcd=1 가정 아래 직접 부등식 증명.

### 4.2 Case 3 난이도

$n = p^a$, $a ≥ 2$ 에서:
- $\sigma(p^a) = (p^{a+1} - 1)/(p - 1)$
- $\varphi(p^a) = p^{a-1}(p - 1)$
- $\tau(p^a) = a + 1$

등식: $(p^{a+1} - 1) \cdot p^{a-1} = p^a \cdot (a + 1)$
- 나누기 $p^{a-1}$: $p^{a+1} - 1 = p \cdot (a+1)$
- 즉 $p^{a+1} - p(a+1) = 1$

p = 2, a = 2: $8 - 6 = 2 \neq 1$. p = 2, a = 3: $16 - 8 = 8 \neq 1$. p = 3, a = 2: $27 - 9 = 18 \neq 1$.

일반적으로 $p^{a+1} \gg p(a+1)$ for $a \geq 2$, $p \geq 2$. 따라서 **모든 (p, a), a ≥ 2 에서 불성립**.

Lean4 구현: power arithmetic + bounded case analysis.
