# v4 loop 12 — M3 case 4c(ii): Theorem B n = 2^a·q·r Lean4 formal

## 목표

a ≥ 2, q ≥ 3, r ≥ 5 odd distinct primes 인 모든 n = 2^a·q·r 에 대하여
  σ(n)·φ(n) ≠ n·τ(n)

## 증명 전략 — STRICT bounds 곱

### 핵심 아이디어

개별 strict bound 두 개를 곱해서 결합:
- σφ(2^a) > 2^a·(a+1)  STRICT for a ≥ 2  (`sigma_phi_2pow_strict`)
- σφ(qr) > qr·4         STRICT for q,r odd distinct ≥ 3  (`sigma_phi_qr_strict`)

Multiplicative decomposition: σφ(2^a·q·r) = σφ(2^a)·σφ(qr)
nτ 동일: nτ(2^a·q·r) = 2^a·(a+1) · qr·4

양의 정수 곱: A > A', B > B', A', B' > 0 → AB > A'B'.
하므로 σφ(n) > nτ(n) STRICT → σφ ≠ nτ.

### `sigma_phi_2pow_strict` 증명

σφ(2^a) = (2^(a+1) - 1) · 2^(a-1) (geom_sum + totient_prime_pow)
RHS = 2^a·(a+1) = 2·2^(a-1)·(a+1)

정리 2^(a+1) = 4·2^(a-1) 사용 (X := 2^(a-1) set):
  (4X - 1)·X > 2X·(a+1)

`prime_pow_strict_gt` (case 3): 2^(a+1) > 2(a+1)+1, i.e., 4X > 2a+3.

4X² ≥ (2a+4)X (곱하여), diff = 4X² - X ≥ 2aX + 3X > 2aX + 2X = 2X(a+1). ✓

### `sigma_phi_qr_strict` 증명

σ(qr)·φ(qr) = (q+1)(r+1)(q-1)(r-1).

보조 bound:
- (q+1)(q-1) = q² - 1 ≥ 2q + 2  (for q ≥ 3, from (q-1)² ≥ 4)
- (r+1)(r-1) ≥ 2r + 8            (for r ≥ 5, from (r-1)² ≥ 16)

곱: (q+1)(q-1)·(r+1)(r-1) ≥ (2q+2)(2r+8) = 4qr + 16q + 4r + 16 > 4qr. ✓
