# N6 Solar Architecture --- Ultimate Goal Roadmap

**궁극적 목표: n=6 산술로 광자부터 그리드까지 관통하는 태양전지 아키텍처**
**비전: 전기요금 0원 시대 — 지붕 하나로 집+차 전력 자급**

---

## Core n=6 Anchors (BT-30, BT-63)

```
  BT-30: Shockley-Queisser Bridge
    - 최적 밴드갭:   1.34 eV  ≈ τ/(n/φ) = 4/3 = 1.333 eV
    - SQ 효율 한계:  33.7%   ≈ φ/n     = 1/3  = 33.33%
    - 열전압 (300K): 25.85 mV ≈ J₂+φ    = 26 mV  (EXACT)
    - 무한접합 한계: 68.7%   ≈ φ²/n    = 2/3  = 66.67%

  BT-63: Solar Panel Cell Ladder
    - 60셀  = σ·sopfr     (주거용 표준)
    - 72셀  = σ·n         (상업용 표준)
    - 120셀 = σ·(σ-φ)     (하프셀 주거용)
    - 144셀 = σ²          (하프셀 상업용)
```
