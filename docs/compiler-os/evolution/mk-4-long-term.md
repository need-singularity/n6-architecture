**Evolution Checkpoint**: Mk.IV
**Date**: 2026-04-04
**Status**: 장기 비전
**Feasibility**: 🔮 30~50년 (의도 기반 프로그래밍)

## 1. 목표

Mk.IV는 인간의 의도를 직접 이해하여 검증된 코드를 생성하는 인지 컴파일러/OS를 실현한다.
