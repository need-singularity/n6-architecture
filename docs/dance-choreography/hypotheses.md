# N6 무용/안무 -- 완전수 산술로 본 동작·공간·리듬 체계

## 개요

무용(dance)과 안무(choreography) 분야의 핵심 구조 상수를
n=6 산술함수로 분석한다. 라반 표기법, 발레 포지션, 사교춤 분류,
장단 구조, 박자 체계, 공간 배치 등 무용의 이산적 표준값이
n=6 상수와 어떻게 매칭되는지 검증한다.

> **정직 원칙**: 무용 분류는 문화·유파별로 다양하다.
> EXACT는 표기법·이론적으로 확정되거나 물리적으로 고정된 수치에만 부여한다.

## 핵심 상수

```
  n = 6, σ = 12, τ = 4, φ = 2, sopfr = 5, J₂ = 24, R(6) = 1
  유도: σ-φ=10, σ-τ=8, σ-μ=11, n/φ=3, σ·τ=48, φ^τ=16, σ²=144
```

```

    - Rudolf von Laban (1879-1958) 정립
    - 국제 라반 표기법 (Labanotation/Kinetography) 표준

  등급: EXACT (라반 이론 표준, 24=J₂ 정확 일치)
  렌즈: topology, info, symmetry
```
