## n=6 통계 프레임워크 설계

### 귀무가설 검정

BT-371의 65/65 EXACT가 우연인지 검정하는 엄밀한 통계 방법:

| 검정 방법 | 접근 | 예상 결과 |
|----------|------|----------|
| 순열 검정 | n=6 상수를 랜덤 상수로 교체 → EXACT 비율 비교 | p < 10⁻(σ-φ) |
| 부트스트랩 | 물리 상수 세트를 리샘플링 → EXACT 비율 분포 | 65/65 ≫ 95% CI |

| 교차 검증 | 도메인별 EXACT 독립성 검증 | n/φ=3+ 도메인 독립 |

### Falsifiability 강화

현재 z=0.74에서 더 엄밀한 검증으로:
- 동일 물리 상수에 다른 수론 함수(n=28 완전수, n=12 풍부수 등) 적용
- 다른 수에서 EXACT 비율이 유의미하게 낮으면 n=6 특수성 확인
- 높으면 "완전수" 자체가 아닌 다른 수론적 성질이 원인
