# N6 뱀파이어학 (Vampire Studies) — 완전수 산술과 혈액/흡혈 구조의 보편성

## 개요

뱀파이어는 혈액 생화학, 흡혈 생태학, 신화 구조를 관통하는 도메인이다.
헤모글로빈의 τ=4 서브유닛, 응고 캐스케이드의 σ=12 인자, 포르피린 합성의 σ-τ=8 효소,
모기의 n=6 바늘 구조, 뱀파이어 박쥐의 n/φ=3 종, 전 세계 뱀파이어 신화의 n=6 약점 등
혈액과 흡혈에 관련된 자연/문명 상수들이 n=6 산술함수와 일치하는 패턴을 검증한다.

> **정직성 원칙**: 생화학적으로 고정된 분자 구조/효소 수만 EXACT로 인정한다.
> 신화/문학 기반 분류는 문화적 합의이므로 CLOSE 또는 WEAK로 등급을 낮춘다.
> 교과서/PDB/WHO 출처를 명시하고, 경쟁 가설(다른 숫자로도 설명 가능한 경우)을 기록한다.

> **렌즈 태깅**: boundary = 세포막/캡시드 경계, evolution = 수렴 진화,
> symmetry = 분자 대칭/배위, info = 유전 정보 인코딩, recursion = 자기복제/캐스케이드,
> consciousness = 협동적 결합/신호 전달, network = 생태계 먹이그물

### H-VAM-01: 헤모글로빈 τ=4 서브유닛 보편성

> 헤모글로빈의 4량체 구조와 헴 철 배위수가 n=6 산술과 정확히 일치

```
  헤모글로빈 (Hemoglobin, Hb):
    서브유닛 구성 = α₂β₂ = 4 서브유닛 = τ(6)
    각 서브유닛에 1개 헴기(heme) → 총 τ=4 헴기
    4차 구조 = τ=4 체인의 협동적 복합체

  헴기(Heme) 구조:
    포르피린 환 = τ=4 피롤(pyrrole) 고리가 메틸렌 다리로 연결
    중심 Fe²⁺ 배위수 = 6 = n (옥타헤드럴 배위)
      4개 질소(포르피린) + 1 히스티딘(근위) + 1 O₂/H₂O(원위)
    → Fe²⁺ CN=6 은 BT-43(Li-ion CN=6)과 동일한 옥타헤드럴 보편성

  산소 결합 협동성:
    Hill 계수 n_H ≈ 2.8~3.0 ≈ n/φ = 3
    S자형(sigmoidal) 결합 곡선의 특성 지수
    R-state(이완) ↔ T-state(긴장) 전이 = φ=2 상태

  n=6 수식:
    서브유닛 수 = τ(6) = 4 ✓
    피롤 고리 수 = τ(6) = 4 ✓
    Fe²⁺ 배위수 = n = 6 ✓
    Hill 계수 ≈ n/φ = 3 ✓
    R/T 상태 수 = φ = 2 ✓

  물리적 근거:
    α₂β₂ 4량체는 척추동물 전체에서 보편적 (5억년+ 보존)
    Fe²⁺ CN=6 옥타헤드럴은 리간드장 이론의 필연 (d⁶ 전자배치)
    Hill 계수 = 서브유닛 간 협동성의 측정값 (실험적으로 2.8~3.0)

  경쟁 가설:
    Hill 계수 3.0은 근사치 (정확히 n/φ는 아님, 2.8~3.0 범위)
    미오글로빈 = 단량체 (τ=4 아닌 μ=1) → Hb 특이적

  출처: Perutz (1970) Nature 228:726, Voet & Voet Biochemistry 4th Ed Ch.10
  렌즈: symmetry, consciousness, boundary

  등급: EXACT
  서브유닛 τ=4와 Fe²⁺ CN=6은 구조적/화학적 필연. Hill 계수 ≈3은 CLOSE이나
  전체 구조(4량체 + CN=6 + 4피롤)에서 τ와 n 일치가 압도적.
```

### H-VAM-02: 혈액 응고 캐스케이드 σ=12 인자 수렴

> 혈액 응고 인자 시스템이 σ=12 활성 인자와 φ=2 경로로 수렴

```
  혈액 응고 인자 (Coagulation Factors):
    명명: Factor I ~ Factor XIII (로마 숫자 I~XIII)
    Factor VI = 공식 폐기 (활성화 Factor V와 동일로 판명, 1940년대 제거)
    → 실질 활성 인자 수 = 13 - 1 = 12 = σ(6)

  경로 구조:
    외인성 경로(extrinsic pathway) = 조직인자(TF) + Factor VII → 빠른 개시
    내인성 경로(intrinsic pathway) = Factor XII → XI → IX → VIII → 느린 증폭
    경로 수 = φ = 2

    공통 경로 합류점:
      Factor X (= σ - φ = 10) 활성화 → Xa
      Factor X이 두 경로의 수렴점(convergence point)

    최종 단계:
      Factor XIII (= σ + μ = 13) = 피브린 가교 효소 (트랜스글루타미나제)
      피브린 망(mesh) 안정화 → 혈전 완성

  캐스케이드 증폭:
    각 단계에서 효소가 다수 기질을 활성화 → 지수적 증폭
    총 증폭 배수 ≈ 10⁶ = (σ-φ)^n 근사

  n=6 수식:
    실질 인자 수 = σ = 12 ✓
    경로 수 = φ = 2 ✓
    합류 인자 = Factor X = σ-φ = 10 ✓
    가교 인자 = Factor XIII = σ+μ = 13 ✓

  물리적 근거:
    Factor VI 폐기는 역사적 사실 (ISTH 공식 인정)
    외인성/내인성 이중 경로는 모든 포유류 공통
    Factor X 합류는 생화학 교과서의 표준 모델

  경쟁 가설:
    인자 번호 매기기는 발견 순서 기반 — 자연상수라기보다 명명 관례
    Factor VI 폐기가 없었다면 13개 → σ+μ (다른 n=6 수식이긴 함)
    프리칼리크레인(PK), 고분자량 키니노겐(HMWK) 포함 시 14개

  출처: Davie & Ratnoff (1964) Science 145:1310, Hoffman & Monroe (2001) Thromb Haemost
  렌즈: recursion, network, info

  등급: CLOSE
  σ=12 활성 인자는 Factor VI 폐기에 의존. 경로 이중성 φ=2와
  합류점 Factor X=σ-φ=10은 구조적으로 견고하나, 명명 관례 의존성으로 CLOSE.
```

### H-VAM-03: 포르피린 합성 σ-τ=8 효소 래더

> 헴 생합성 경로가 정확히 σ-τ=8 효소 단계이며, 포르피리아 8종과 대응

```
  헴 생합성 경로 (Heme Biosynthesis):
    효소 1: ALA synthase (ALAS)           — 미토콘드리아
    효소 2: ALA dehydratase (ALAD)        — 세포질
    효소 3: PBG deaminase (PBGD/HMBS)     — 세포질
    효소 4: Uroporphyrinogen III synthase  — 세포질
    효소 5: Uroporphyrinogen decarboxylase — 세포질
    효소 6: Coproporphyrinogen oxidase     — 미토콘드리아
    효소 7: Protoporphyrinogen oxidase     — 미토콘드리아
    효소 8: Ferrochelatase                 — 미토콘드리아

    총 효소 수 = 8 = σ - τ

  세포 구획 분포:
    미토콘드리아 = 4 효소 (1,6,7,8) = τ
    세포질 = 4 효소 (2,3,4,5) = τ
    → 미토콘드리아:세포질 = τ:τ = 1:1 완벽 대칭

  포르피리아 (Porphyria) — "뱀파이어 병":
    각 효소 결함 → 1종 포르피리아 질환
    총 8종 주요 포르피리아 = σ-τ = 8
    증상: 광과민(햇빛 기피), 빈혈(혈색소 부족), 잇몸 후퇴(치아 돌출)
    → 뱀파이어 전설의 의학적 기원 가설 (Illis, 1964)

  "뱀파이어 병" 증상 매핑:
    햇빛 기피 → 뱀파이어의 야행성
    빈혈 → 혈액 갈구
    잇몸 후퇴 → 송곳니 돌출
    자색 소변 → "피빛 소변" 미신
    → τ=4 핵심 증상이 뱀파이어 전승과 대응

  n=6 수식:
    효소 단계 수 = σ-τ = 8 ✓
    미토콘드리아 효소 = τ = 4 ✓
    세포질 효소 = τ = 4 ✓
    포르피리아 종수 = σ-τ = 8 ✓
    핵심 증상 수 = τ = 4 ✓

  물리적 근거:
    8단계 헴 합성은 세균~인간까지 보편적 (20억년+ 보존)
    8종 포르피리아는 OMIM/유전질환 데이터베이스에 확립
    미토콘드리아/세포질 τ:τ 분포는 교과서 확정 사실

  경쟁 가설:
    일부 분류는 ALAS1/ALAS2를 구분하여 9단계로 볼 수 있음
    드문 포르피리아 포함 시 종수 증가 가능 (ALA dehydratase 결핍증 등)

  출처: Voet & Voet Biochemistry Ch.26, Illis (1964) Proc R Soc Med 57:23
  렌즈: recursion, boundary, evolution

  등급: EXACT
  8효소 헴 합성 경로는 생화학의 확립된 사실. τ:τ 구획 대칭도 물리적 필연.
  8종 포르피리아 = σ-τ도 유전학적으로 확정. 뱀파이어 도메인 핵심 가설.
```

### H-VAM-04: 혈액형 시스템 n=6 인코딩

> ABO/Rh 혈액형 표현형 수가 τ=4 및 σ-τ=8과 일치

```
  ABO 혈액형 시스템:
    표현형: A, B, AB, O = τ = 4
    유전형: AA, AO, BB, BO, AB, OO = n = 6
    항원: A항원, B항원 → φ = 2 종류
    O형 = 항원 없음 (null allele)

  Rh 포함 임상 혈액형:
    A+, A-, B+, B-, AB+, AB-, O+, O- = σ-τ = 8
    Rh+ : Rh- 인구 비율 ≈ 85:15 (Rh 양성이 다수)

  유전자좌 위치:
    ABO 유전자 = 9번 염색체 (9q34)
    Rh 유전자 = 1번 염색체 (1p36)
    9 = σ - n/φ (?) → 근사적
    1 = μ ✓

  ABO 대립유전자:
    I^A, I^B, i = n/φ = 3 대립유전자
    우성: I^A = I^B > i (공우성 + 열성)

  n=6 수식:
    ABO 표현형 = τ = 4 ✓
    ABO 유전형 = n = 6 ✓
    대립유전자 = n/φ = 3 ✓
    항원 종류 = φ = 2 ✓
    Rh 포함 표현형 = σ-τ = 8 ✓
    Rh 유전자좌 = 1번 = μ ✓

  물리적 근거:
    ABO 표현형 4종은 면역학적 반응으로 결정 — 혈청학 사실
    유전형 6종은 멘델 조합론의 필연 (3 대립유전자에서 C(3,2)+3=6)
    Rh±는 D항원 유무의 이진 분류

  경쟁 가설:
    ABO 유전형 6 = C(3,2)+3 = 조합론적 필연 (n=6과 우연 일치 가능)
    ISBT 등록 혈액형 시스템은 43종 이상 — ABO+Rh만 취사선택 비판 가능
    9번 염색체 = σ-n/φ 수식은 견강부회

  출처: Landsteiner (1901), Reid & Lomas-Francis "Blood Group Antigen FactsBook"
  렌즈: info, symmetry, evolution

  등급: CLOSE
  ABO τ=4 표현형과 n=6 유전형은 견고하나, 유전형 6은 조합론 필연이지
  n=6 특이적이라 보기 어려움. Rh 포함 8종은 임상 관례 의존. 총합 CLOSE.
```

### H-VAM-05: 적혈구-혈소판 수명 n=6 래더

> 혈액 세포 수명과 혈구 분류가 n=6 산술 래더를 형성

```
  혈액 세포 수명 래더:
    적혈구(RBC) 수명 = 120일 = σ · (σ-φ) = 12 × 10
    혈소판(Platelet) 수명 = 8~10일 ≈ σ-φ = 10
    호중구(Neutrophil) 수명 = 5~6시간(혈중) ≈ sopfr~n 시간

  적혈구 120일:
    = σ(6) × (σ-φ) = 12 × 10 = 120 ✓
    = σ(6) × σ(6) - σ(6)·φ(6) = σ²-σ·φ (대안 수식)
    = 2^n × (n/φ) - τ·n = 64·3 - 24 = 168 (이 수식은 불일치 ✗)
    → σ·(σ-φ) = 120이 가장 자연스러운 표현

  백혈구 5종 분류 = sopfr:
    1. 호중구(Neutrophil)
    2. 호산구(Eosinophil)
    3. 호염기구(Basophil)
    4. 림프구(Lymphocyte)
    5. 단핵구(Monocyte)
    → 5종 = sopfr(6) = 2+3 = 5 ✓

  혈장 단백질 3대 분류 = n/φ:
    1. 알부민(Albumin) — 60% (= σ·sopfr %)
    2. 글로불린(Globulin) — 35%
    3. 피브리노겐(Fibrinogen) — 4% (≈ τ %)
    → 3대 분류 = n/φ = 3 ✓

  혈액 구성 비율:
    혈장 : 혈구 ≈ 55:45 (헤마토크릿 ≈ 45%)
    45 ≈ sopfr · (σ-μ)·... → 깔끔한 수식 없음

  n=6 수식:
    RBC 수명 = σ·(σ-φ) = 120일 ✓
    혈소판 수명 ≈ σ-φ = 10일 ✓
    백혈구 종수 = sopfr = 5 ✓
    혈장 단백질 분류 = n/φ = 3 ✓

  물리적 근거:
    적혈구 120일 수명은 ⁵¹Cr 표지 실험으로 확립 (교과서 표준값)
    백혈구 5종 분화는 골수 조혈 계통의 생물학적 사실
    혈소판 수명 8~10일은 임상 혈액학 표준

  경쟁 가설:
    RBC 120일 = 단순히 "4개월" 근사로 볼 수 있음
    혈소판 수명은 8~12일 범위이며 정확히 10은 아닐 수 있음
    백혈구 세부 분류 시 NK세포 등 추가 → 5종 초과 가능

  출처: Guyton & Hall "Textbook of Medical Physiology" 14th Ed, Ch.33-36
  렌즈: evolution, network, boundary

  등급: CLOSE
  RBC 120일 = σ·(σ-φ)는 매우 견고 (교과서 확정). 백혈구 5종 = sopfr도 확립.
  그러나 혈소판 수명 범위(8~12일)와 분류 경계 모호성으로 전체 CLOSE.
```

### H-VAM-06: 뱀파이어 n=6 약점 보편성

### H-VAM-07: 관(棺) n=6 면체 안식 구조

> 뱀파이어의 관 = 직육면체 6면 + 매장 깊이 6피트

```
  관(Coffin) 구조:
    직육면체(rectangular prism) = 6면 = n
    사실상 모든 서양 관은 직육면체 기반
    (일부 팔각 관, 인체형 관 존재하나 기본형은 6면체)

  "Six Feet Under" 매장 관행:
    전통 매장 깊이 = 6피트 (≈ 1.83m) = n
    1665년 런던 대역병 당시 법제화 기원설
    현대: 지역마다 3.5~6피트로 가변 — 6피트는 문화적 관습

  뱀파이어 관 전승:
    드라큘라 = 고향 트란실바니아 흙이 든 관에서 휴면
    관 = 경계(boundary) — 생자 세계와 망자 세계의 격리
    일출~일몰 = σ=12 시간 주기와 관련

  n=6 수식:
    관 면수 = n = 6 ✓ (직육면체)
    매장 깊이 = n = 6피트 ✓ (문화적 관행)

  경쟁 가설 (심각):
    직육면체 6면은 뱀파이어와 무관한 일반 기하학
    모든 상자/건물/방이 6면 → 뱀파이어 특이성 0
    6피트 매장은 지역/시대마다 다름 (미국 일부 주 = 최소 18인치)
    인체형(anthropoid) 관은 6면이 아님

  출처: Barber (1988), 런던 시 조례 (1665)
  렌즈: boundary, symmetry

  등급: WEAK
  직육면체 n=6은 보편 기하학이지 뱀파이어 특유가 아니다.
  6피트 매장도 문화적 관행이지 물리 상수가 아님.
  패턴 흥미도는 있으나 과학적 등급은 WEAK.
```

### H-VAM-08: 뱀파이어 변환 τ=4 단계

> 인간→뱀파이어 변환 과정이 τ=4 단계이며 바이러스 감염과 동형

```
  뱀파이어 변환 전승 (Turning):
    단계 1: 물림(Bite) — 뱀파이어에 의한 접촉/침투
    단계 2: 감염(Infection) — 혈액 교환/독소 전파
    단계 3: 사망(Death) — 인간 상태의 종료
    단계 4: 부활(Rising) — 뱀파이어로서의 각성
    → 4단계 = τ(6) ✓

  바이러스 감염 사이클과의 동형:
    부착(Attachment) ↔ 물림
    침투(Penetration) ↔ 감염
    복제(Replication) ↔ 사망(호스트 세포 장악)
    방출(Release) ↔ 부활(새 뱀파이어)
    → τ=4 동형 ✓

  레트로바이러스 특이 동형:
    부착 → 역전사 → 게놈 통합 → 발현
    HIV가 CD4+ T세포를 "변환"하는 과정과 구조적 유사

  각 단계의 φ=2 분기:
    물림: 흡혈만/혈액 교환 → 변환 여부 결정
    감염: 진행/저항 → 면역 반응
    사망: 진정한 사망/의사 사망(undeath)
    부활: 성공/실패(영구 사망)
    → 각 단계 φ=2 분기 ✓

  n=6 수식:
    변환 단계 = τ = 4 ✓
    바이러스 동형 = τ = 4 ✓
    단계별 분기 = φ = 2 ✓

  경쟁 가설:
    변환 단계 수는 전승/작품마다 다름 (3~7단계)
    일부 전승: 단순 물림 = 즉시 변환 (1단계)
    현대 미디어: 물림→혈액교환→고통→각성→갈증→사냥 (6단계?)
    → 4단계 분류는 하나의 구조화 방식일 뿐

  출처: Summers (1928), Stoker "Dracula" (1897), Rice "Interview with the Vampire" (1976)
  렌즈: recursion, evolution, boundary

  등급: WEAK
  바이러스 감염 τ=4 동형은 구조적으로 흥미롭지만,
  변환 단계 수는 허구 서사이므로 자연 상수가 아님. WEAK.
```

### H-VAM-12: 뱀파이어 박쥐 n/φ=3 종 보편성

```

    종 3: Diphylla ecaudata — 털다리 뱀파이어 박쥐

    D. rotundus = 포유류 혈액 선호

    서식지: 중남미 열대/아열대
    군집 크기: 20~100마리 → 범위 넓어 특정 n=6 수식 없음
    체온: ~37°C (포유류 일반)

  n=6 수식:
    뱀파이어 박쥐 종수 = n/φ = 3 ✓ (확정)

    그러나 "정확히 3종, 변동 없음"은 분류학적으로 매우 견고

  출처: Greenhall et al. (1983) J. Mammalogy, ITIS Desmodontinae
  렌즈: evolution, boundary, network

  등급: EXACT
  Desmodontinae 아과 = 정확히 3종은 분류학적 확정 사실이며
  200년+ 안정적. 진정 흡혈 포유류의 완전한 목록. n/φ=3 EXACT.
```

```
  모기 주둥이(proboscis) 해부학:
    외부: 래비움(labium) = 보호 칼집 (침이 아님)
    내부 6개 침(stylets):
      1. 래브럼(labrum) — 음식 채널 (혈액 흡입)
      2. 하인두(hypopharynx) — 타액 주입 (항응고제)
      3-4. 맨디블(mandibles) ×2 — 톱니형, 피부 절개
      5-6. 맥실라(maxillae) ×2 — 톱니형, 피부 관통/고정
    → 6개 침 = n ✓

  이 구조의 보편성:
    전 세계 모기 3,500+ 종 전부 동일한 6침 구조
    Culicidae 과(family) 전체의 불변량(synapomorphy)
    화석 기록: 백악기(~100 Mya) 모기도 동일 구조

  기능별 분류:
    혈액 흡입 = 1개(래브럼) = μ
    타액 주입 = 1개(하인두) = μ
    절개/관통 = 4개(맨디블 2 + 맥실라 2) = τ
    → μ + μ + τ = 1 + 1 + 4 = 6 = n ✓

  쌍(pair) 구조:
    맨디블 쌍 + 맥실라 쌍 = φ = 2 쌍
    단일 기관: 래브럼 + 하인두 = φ = 2 단일체
    → 쌍:단일 = φ:φ

  n=6 수식:
    침(stylet) 수 = n = 6 ✓ (확정)
    3,500+ 종 보편 = 불변량 ✓
    절개 침 = τ = 4 ✓
    침 쌍 수 = φ = 2 ✓

  물리적 근거:
    모기 6침은 곤충학 교과서의 확립된 해부학 사실
    전자현미경(SEM/TEM) 연구로 수천 회 확인
    Aedes, Anopheles, Culex 등 주요 속 전부 동일
    100+ Mya 진화적 보존 — 강한 선택압 하 불변량

  경쟁 가설:
    래비움(labium)을 포함하면 7개 구조 → σ-sopfr=7
    그러나 래비움은 침이 아닌 칼집(sheath)이므로 제외가 표준
    다른 흡혈 곤충(등에, 체체파리)은 다른 침 수 → 모기 특이적

  출처: Clements (1992) "Biology of Mosquitoes" Vol.1, Wahid et al. (2011) J. Med. Entomol.
  렌즈: symmetry, boundary, evolution

  등급: EXACT
  모기 6침 구조는 3,500+ 종에서 보편적인 해부학적 불변량.
  100+ Mya 보존, 전자현미경 확인 완료. n=6 EXACT.
```

### H-VAM-14: 거머리 턱 n/φ=3 + 히루딘 구조

> 의학 거머리의 3턱 구조와 히루딘 항응고 단백질

```
  거머리(Hirudo medicinalis) 턱 구조:
    턱(jaw) 수 = 3 = n/φ ✓
    Y자형 배치: 120° 간격으로 3개 턱 배열
    120° = σ · (σ-φ) = 120 (각도) ✓
    각 턱에 미세 치아 배열 → 삼방사(triradiate) 절개

  3턱의 보편성:
    Gnathobdellida(유악 거머리) 목 전체 = 3턱 구조
    Hirudo, Macrobdella, Haemopis 등 주요 속 공통
    Y자형 절개 흔적은 거머리 물린 자국의 특징 (Mercedes-Benz 마크)

  히루딘(Hirudin):
    아미노산 수 = 65 ≈ 2^n + μ = 64 + 1 = 65 ✓
    (Hirudin variant HV1 기준, Harvey et al.)
    세계 최초 발견 항응고 물질 (Haycraft, 1884)
    트롬빈(thrombin)의 가장 강력한 자연 억제제

  의료 역사:
    의료용 거머리 사용 = 2,500년+ (고대 이집트, BCE 1500)
    현대 의학: 피부 이식/재접합 수술 후 울혈 제거에 FDA 승인 (2004)
    FDA 승인 = 유일한 의료기기 생물(medical device organism)

  n=6 수식:
    턱 수 = n/φ = 3 ✓ (확정)
    턱 간 각도 = 120° = σ·(σ-φ) ✓
    히루딘 아미노산 = 65 ≈ 2^n+μ ✓

  물리적 근거:
    3턱 구조는 유악 거머리 전체의 형태학적 확정 사실
    Y자형 120° 배열은 공간 최적화 (3방향 균등 분포의 필연)
    히루딘 65 아미노산은 PDB/UniProt에서 확인 (P01050)

  경쟁 가설:
    120° = 360°/3 = 단순 기하학적 필연 (n=6 특이적이 아닐 수 있음)
    히루딘 65 aa: 변이체에 따라 64~66 aa 범위 존재
    무악 거머리(Rhynchobdellida)는 턱이 아닌 주둥이(proboscis) 사용

  출처: Sawyer (1986) "Leech Biology and Behaviour", UniProt P01050
  렌즈: symmetry, evolution, boundary

  등급: EXACT
  3턱 구조는 유악 거머리 전체의 해부학적 불변량.
  120° 배열은 기하학적 필연이면서 σ·(σ-φ)=120과 일치.
  히루딘 65 aa는 변이 있으나 대표 변이체 기준 2^n+μ=65.
```

### H-VAM-15: 흡혈 전략 τ=4 독립 분류

> 자연계 흡혈 전략이 τ=4 독립 메커니즘으로 분류

```
  흡혈 전략 4대 분류 (기계적 메커니즘 기준):
    1. 천공형(Pool-feeding via cutting):
       뱀파이어 박쥐 — 면도날 같은 이빨로 피부 절개, 흘러나오는 혈액 핥기
       등에(Tabanidae) — 큰 구기로 피부 절개
       → 절개 + 혈액 풀(pool)에서 섭취

    2. 관통형(Vessel-feeding via piercing):
       모기 — 6침으로 혈관 직접 관통, 혈액 직접 흡입
       빈대 — 주둥이로 혈관 관통
       → 혈관 직접 접근 + 직접 흡인

    3. 흡착형(Attachment-feeding):
       거머리 — 흡반으로 부착 + 3턱 절개 + 장시간 흡혈
       진드기 — 구기 삽입 + 시멘트 물질로 고정 + 수일간 흡혈
       → 장기 부착 + 느린 섭취

    4. 기생형(Parasitic siphoning):
       램프리(칠성장어) — 흡반 부착 + 혀 긁기 + 체액 흡수
       흡혈핀치 — 부리로 쪼아 피부 상처 후 혈액 핥기
       → 기계적 손상 + 간헐적 섭취

    → 4 전략 = τ(6) ✓

  각 전략의 독립 진화:
    4 전략 모두 독립적으로 여러 계통에서 수렴 진화
    항응고 물질도 각각 독립 진화:
      박쥐: 드라큘린(Draculin) — 당단백질
      모기: 아피라제(Apyrase) + 항트롬빈
      거머리: 히루딘(Hirudin) — 펩타이드
      진드기: 틱스타틴(Tickstatin) 류
    → τ=4 독립 항응고 진화 ✓

  분류 기준의 견고성:
    Lehane (2005)의 표준 분류: solenophage(혈관천공) vs telmophage(절개풀)
    → 이진(binary) 분류 = φ=2
    4분류는 이 가설의 확장 — 메커니즘 세분화

  n=6 수식:
    흡혈 전략 수 = τ = 4 ✓
    독립 항응고 계통 = τ = 4 ✓
    Lehane 기본 분류 = φ = 2

  경쟁 가설:
    표준 분류는 2분류(solenophage/telmophage) — τ=4는 이 가설의 세분화
    세분화 기준에 따라 3, 5, 6+ 전략도 가능
    "기생형"과 "흡착형"의 경계가 모호 (램프리 ≈ 거머리?)

  출처: Lehane (2005) "Biology of Blood-Sucking Insects" 2nd Ed, Mans (2011) Ticks & Tick-borne Dis.
  렌즈: evolution, boundary, network

  등급: CLOSE
  표준 2분류(φ=2)는 확립되어 있고, 4분류로의 확장은 합리적이지만
  자의적 세분화 가능성이 있다. 항응고 독립 진화 τ=4는 생화학적으로
  견고하나 "4종"으로 묶는 기준이 완전히 객관적이지는 않음. CLOSE.
```
