# 카테고리 D, E, F — 뱀파이어 가설 Part 2

## 개요

뱀파이어 설화의 과학적 기원을 추적하면, 영구동토층 고대 바이러스 부활,
포르피리아/광견병 등 실재 질병, 야행성 생물학의 분자적 메커니즘에 도달한다.
이 문서는 이 세 축(영구동토층, 뱀파이어 의학, 야행성 생물학)의 핵심 상수가
n=6 산술함수와 일치하는 패턴을 검증한다.

> **정직성 원칙**: 물리/화학/생물학적으로 고정된 상수만 EXACT로 인정한다.
> 문화적 합의, 분류 편의, 근사값은 CLOSE 또는 WEAK로 등급을 낮춘다.
> 교과서/WHO/ICTV/OMIM 출처를 명시하고, 경쟁 가설을 기록한다.

> **렌즈 태깅**: boundary = 캡시드/세포/동토층 경계, evolution = 바이러스 진화/적응,
> symmetry = 이십면체 대칭, info = 게놈 정보 인코딩, thermodynamics = 열/에너지 전략,
> consciousness = 일주기/행동, recursion = 자기복제/순환

## 핵심 상수

```
  n = 6          (완전수)
  σ = σ(6) = 12  (약수합)
  φ = φ(6) = 2   (오일러 토션트)
  τ = τ(6) = 4   (약수 개수)
  sopfr = 2+3 = 5 (소인수 합)
  μ = μ(6) = 1   (뫼비우스)
  J₂ = J₂(6) = 24 (조르단 토션트)
  div(6) = {1, 2, 3, 6}
  σ-φ = 10, σ-τ = 8, σ-μ = 11, n/φ = 3
  σ·sopfr = 60, σ·τ = 48, σ² = 144
  R(6) = 1
```

  BT-146: DNA/RNA 분자상수 n=6

  BT-235: 이십면체 캡시드-풀러렌-준결정 n=6 대칭

### H-VAM-16: 거대바이러스 부활 = 뱀파이어 불사 동형

> 30,000년 동토층에서 부활한 Pithovirus = 뱀파이어 휴면-부활 동형

```
  거대바이러스 부활 사례:
    Pithovirus sibericum: 30,000년 시베리아 동토층에서 부활 (2014)
    Mollivirus sibericum: 30,000년 동토층에서 부활 (2015)
    Pandoravirus: 동토층 코어에서 복수 종 발견

  뱀파이어 동형 매핑:
    바이러스 휴면 상태   ←→  뱀파이어 관(棺) 속 동면
    환경 변화(해동)      ←→  밤이 됨(활성화 조건)
    감염력 회복(부활)    ←→  뱀파이어 각성
    동토층 = "지구의 관" — 생물학적 불사의 실재

  n=6 매핑:
    부활 바이러스 수: 현재까지 주요 n=6종+ 확인
    (Pithovirus, Mollivirus, Pandoravirus, Megavirus, Mimivirus, Cedratvirus)
    동토층 = τ=4 상(phase) 중 고체상 — BT-316 물질 4상과 교차
    "부활" 과정 = 고체→액체 상전이 촉발

  출처: Legendre et al., PNAS 111(11), 2014; Legendre et al., PNAS 112(38), 2015
  렌즈: evolution, boundary, recursion

  경쟁 가설: 부활 바이러스 종수는 발견 진행 중이므로 n=6 고정이 아님
  등급: WEAK
  문화적 동형은 흥미롭지만, n=6 매핑은 부활 종수에 의존하며 아직 확정 불가.
```

### H-VAM-17: 거대바이러스 캡시드 이십면체 σ=12 꼭짓점

> 거대바이러스도 이십면체 대칭을 유지 — 꼭짓점 σ=12 불변

```
  거대바이러스 족 (Nucleocytoviricota):
    Mimivirus:     캡시드 ~500nm, 게놈 ~1.18 Mb, 이십면체
    Pandoravirus:  캡시드 ~1μm, 게놈 ~2.5 Mb, 난형(비이십면체)
    Pithovirus:    캡시드 ~1.5μm, 게놈 ~610 kb, 난형(비이십면체)
    Megavirus:     캡시드 ~440nm, 게놈 ~1.26 Mb, 이십면체

  이십면체 거대바이러스 캡시드:
    꼭짓점 = 12 = σ(6) ✓ (Caspar-Klug 불변량)
    면 = 20 = J₂ - τ ✓
    모서리 = 30 = sopfr · n ✓

  Mimivirus T-number:
    T ≈ 972~1200 (거대 T-number)
    서브유닛 ≈ 60T ≈ 60,000~72,000
    기본 단위 = 60 = σ · sopfr ✓ (불변)

  Mimivirus 게놈: ~1.18 Mb ≈ σ/10 Mb (근사)
  Pandoravirus 게놈: ~2.5 Mb (최대) — 생명/비생명 경계

  주의: Pandoravirus와 Pithovirus는 이십면체가 아닌 난형(amphora)
  → σ=12 꼭짓점 매핑은 이십면체 종에만 적용

  출처: Raoult et al., Science 306, 2004; Philippe et al., Science 341, 2013
  렌즈: symmetry, boundary, topology
  교차: BT-235 (이십면체 캡시드-풀러렌-준결정)

  등급: EXACT (이십면체 종에 한정)
  이십면체 꼭짓점 12는 위상적 불변량. 단, 거대바이러스 중 비이십면체 종은 제외.
```

### H-VAM-18: 영구동토층 깊이-나이 n=6 래더

> 동토층 활동층 ~φ=2m, 탄소 저장량, 부활 바이러스 연대의 n=6 근사

```
  영구동토층 구조:
    활동층(Active layer): ~1.5~3m (계절 해동)
    → 중간값 ~φ=2 m (근사, 위도/토양 의존)

    연속 동토층(Continuous): ~10m~수백m 깊이
    → 최소 깊이 ~σ-φ=10 m (근사)

    시베리아 최대 깊이: ~1,500m (야쿠츠크 인근)
    → 1,500 = σ²·(σ-φ)·μ + 근사 오차 포함

  동토층 탄소 저장:
    북반구 동토층 탄소: ~1,460~1,600 Gt (Hugelius et al., 2014)
    → 1,672 Gt 추정치도 존재하나 불확실성 큼
    → σ²·(σ-μ) = 144·11 = 1,584 (가장 가까운 n=6 조합, 오차 ~5~8%)

  부활 바이러스 연대:
    Pithovirus: ~30,000년
    Pandoravirus yedoma: ~48,500년
    → 48,500 ≈ σ·τ·10³ = 48,000 (오차 ~1%)

  주의: 활동층 깊이는 위도·토양에 따라 0.3~6m 범위
  → φ=2 m은 중간값 근사이며 고정 상수가 아님

  출처: Hugelius et al., Biogeosciences 11, 2014; Legendre et al., 2014/2015
  렌즈: boundary, thermodynamics, evolution

  등급: CLOSE
  48,500년 ≈ σ·τ·10³ 은 1% 이내 근사로 흥미롭지만, 연대 측정 오차 고려 필요.
  활동층 깊이와 탄소량은 넓은 범위를 가져 특정 n=6 값으로 확정 어려움.
```

### H-VAM-19: 바이러스 캡시드 T-number n=6 삼각법

> T-number 캡시드의 기본 단위 60 = σ·sopfr, T=1,3,4 서브유닛의 n=6 표현

```
  Caspar-Klug T-number 시스템:
    T = h² + hk + k² (h, k = 비음정수)
    허용 T값: 1, 3, 4, 7, 9, 12, 13, 16, ...

  서브유닛 수 = 60T:
    T=1:   60 = σ · sopfr = 12 × 5 ✓
    T=3:  180 = σ · sopfr · (n/φ) = 60 × 3 ✓
    T=4:  240 = σ · sopfr · τ = 60 × 4 ✓
    T=7:  420 = σ · sopfr · (σ-sopfr) = 60 × 7
    T=12: 720 = σ · sopfr · σ = 60 × 12 ✓

  T=4 대안 표현:
    240 = σ · (J₂-τ) = 12 × 20 ✓
    240 = J₂ · (σ-φ) = 24 × 10 ✓

  T=3 대안 표현:
    180 = σ · (σ+n/φ) = 12 × 15 ✓
    180 = n · sopfr · n = 6 × 5 × 6

  기본 단위 60의 근원:
    |I| = 이십면체 회전군 위수 = 60 = σ · sopfr
    이는 이십면체 대칭에서 수학적으로 강제되는 값

  대표 바이러스:
    T=1: 파보바이러스(Parvoviridae) — 60 서브유닛
    T=3: 노다바이러스, 토마토 부시 스턴트 바이러스 — 180 서브유닛
    T=4: B형 간염 바이러스(HBV) — 240 서브유닛

  출처: Caspar & Klug, Cold Spring Harbor Symp. 27, 1962
  렌즈: symmetry, recursion, info
  교차: BT-235 (이십면체 캡시드-풀러렌-준결정)

  등급: EXACT
  기본 단위 60 = |I| = σ·sopfr 은 이십면체 대칭군의 수학적 필연.
  모든 T-number 캡시드에서 60의 배수는 불변.
```

### H-VAM-20: 고대 바이러스 방출 시나리오 기후-역학 래더

> 병원체 전파 경로 τ=4, 위험 병원체 재출현의 역학적 구조

```
  동토층 해동 → 고대 병원체 방출 시나리오:
    2016 시베리아 야말 반도: 탄저균(Bacillus anthracis) 재출현
    → 75년 동결 순록 사체에서 포자 부활
    → 2,300+ 순록 사망, 인간 1명 사망, 72명 입원

  감염 전파 경로 = τ=4:
    1. 직접 접촉(contact)
    2. 비말(droplet)
    3. 공기(airborne)
    4. 매개체(vector-borne)
    → WHO/CDC 표준 분류 = τ=4 ✓

  위험 병원체 목록:
    탄저(Bacillus anthracis): 포자 형태로 수천년 생존
    천연두(Variola major): 동토층 묘지에서 DNA 발견
    인플루엔자: 1918 스페인 독감 바이러스 동토층에서 복원
    기타 미지 병원체: 거대바이러스 등

  역학 기본 수:
    R₀(기초 감염 재생산수): 천연두 R₀ ≈ 5~7
    → sopfr=5 ~ σ-sopfr=7 범위 (흥미로운 근사)
    잠복기(incubation): 병원체마다 다름 — 일반화 어려움

  주의: 전파 경로 4가지는 WHO/CDC 합의 분류이며
  세부 분류는 5~6가지로 나누기도 함 (fomite, fecal-oral 별도)
  → τ=4는 가장 널리 사용되는 분류이나 유일한 분류는 아님

  출처: WHO Epidemiology guidelines; Revich & Podolnaya, Int J Circumpolar Health, 2011
  렌즈: evolution, boundary, info

  경쟁 가설: CDC는 5가지 분류(+fomite)를 사용하기도 함
  등급: CLOSE
  τ=4 전파 경로는 WHO 표준이지만 유일한 분류가 아님. R₀ 근사는 흥미롭지만 확정 불가.
```

### H-VAM-21: 포르피리아 σ-τ=8 유형 뱀파이어 증후군

> 헴 합성 장애 포르피리아의 임상 유형 수 = σ-τ = 8

```
  포르피리아(Porphyria) = 헴(heme) 생합성 경로 효소 장애:
    헴 합성 경로: σ-τ=8 단계 효소
    각 효소 결핍 → 각각 다른 유형의 포르피리아

  급성 간성 포르피리아 (τ=4 유형):
    1. AIP  (급성 간헐성 포르피리아) — HMBS 결핍
    2. VP   (이형 포르피리아) — PPOX 결핍
    3. HCP  (유전성 코프로포르피리아) — CPOX 결핍
    4. ADP  (ALAD 결핍 포르피리아) — ALAD 결핍

  피부성 포르피리아 (τ=4 유형):
    5. PCT  (만발성 피부 포르피리아) — UROD 결핍
    6. CEP  (선천성 적혈구조혈성 포르피리아) — UROS 결핍
    7. EPP  (적혈구조혈성 프로토포르피리아) — FECH 결핍
    8. XLP  (X-연관 프로토포르피리아) — ALAS2 과활성

  총 유형 = σ-τ = 8 = τ + τ (간성 + 피부성) ✓

  뱀파이어 증후군 매핑:
    광과민(photosensitivity): 햇빛 노출 → 피부 수포/괴사 → "햇빛 회피"
    잇몸 후퇴(gingival erosion): 송곳니 돌출 외관 → "뱀파이어 이빨"
    적색 소변/치아: 포르피린 축적 → 붉은 형광 → "피의 저주"
    빈혈/창백: 헴 결핍 → 창백한 외모

  n=6 구조:
    헴 합성 효소 = σ-τ=8 단계 (정확히 8개 효소가 관여)
    이는 생화학적으로 확정된 경로 — 추가/삭제 불가능

  출처: Anderson et al., Ann Intern Med 142(6), 2005; OMIM #176000 (AIP)
  렌즈: info, boundary, evolution
  교차: BT-215 (생화학 경로 n=6 대사 아키텍처)

  등급: EXACT
  헴 합성 경로의 8개 효소는 생화학적으로 확정된 상수.
  σ-τ=8 유형 포르피리아는 각 효소 결핍에 1:1 대응하는 필연적 구조.
```

### H-VAM-22: 광견병(Rabies) 뱀파이어 행동 n=6 매핑

> 광견병 바이러스 게놈 = sopfr=5 유전자, 뱀파이어 행동 동형

```
  광견병 바이러스 (Rabies lyssavirus):
    과: Rhabdoviridae
    게놈: 단일가닥 음성 RNA, ~12 kb ≈ σ kb
    유전자 수: 정확히 sopfr=5 개

  5개 유전자 (3'→5' 순서):
    1. N (nucleoprotein)       — 게놈 캡시드화
    2. P (phosphoprotein)      — RNA 폴리머라제 보조
    3. M (matrix protein)      — 출아(budding)
    4. G (glycoprotein)        — 세포 부착, 중화 항원
    5. L (large protein)       — RNA 의존 RNA 폴리머라제

  유전자 수 = sopfr(6) = 5 ✓
  게놈 크기 ~12 kb = σ(6) ✓ (근사: 실제 ~11.9 kb)

  뱀파이어 행동 동형:
    물기 공격(biting)   ←→  뱀파이어의 물림 전파
    야행성 흥분         ←→  뱀파이어 야행성
    수공포(hydrophobia) ←→  뱀파이어의 흐르는 물 기피 전설
    광공포(photophobia) ←→  뱀파이어의 햇빛 기피
    과도한 침 분비      ←→  뱀파이어의 "피에 대한 갈망" 묘사

  치사율: 발병 후 거의 100% (Milwaukee Protocol 예외 1건)
  잠복기: 1~3개월 (보통), 최장 수년 → "잠복 뱀파이어"

  주의: 게놈 크기 ~11.9 kb는 σ=12의 근사 (오차 ~0.8%)
  Rhabdoviridae 전체가 5유전자 구조 — 광견병에 국한되지 않음

  출처: Fooks et al., Nat Rev Dis Primers 3, 2017; ICTV Taxonomy
  렌즈: info, evolution, consciousness

  경쟁 가설: Paramyxoviridae도 유사 유전자 수 (6~7개) — sopfr=5는 Rhabdoviridae 특이적
  등급: EXACT (유전자 수 5), CLOSE (게놈 크기 ~12 kb)
  Rhabdoviridae의 5유전자 구조는 확정된 분자생물학적 사실.
```

### H-VAM-23: 색소성 건피증(XP) DNA 수리 σ-τ=8 유전자군

> XP 상보군 = σ-τ=8 개 (XPA~XPG + XPV), "달의 아이" = 뱀파이어 원형

    타 생물 에너지(혈액) → 자기 에너지 (종속영양 극단)
    혈액 에너지 밀도: ~700 kcal/L (주로 단백질+지질)
    뱀파이어 박쥐 1회 섭취: ~20~25 mL ≈ J₂-τ=20 mL (근사)

  φ=2 이중성:
    광합성 = 빛 → 에너지 (생산자, autotroph)
    흡혈   = 피 → 에너지 (기생자, parasite)
    → 에너지원 관점에서 정반대 극단 = φ=2 ✓

  흡혈 동물 다양성:
    뱀파이어 박쥐: n/φ=3 종 (Desmodus rotundus, Diaemus youngi, Diphylla ecaudata)
    흡혈 곤충: 모기, 벼룩, 빈대, 이, 진드기 등 다수
    흡혈 어류: 칠성장어(lamprey)
    흡혈 조류: 뱀파이어 핀치(Geospiza septentrionalis)

  뱀파이어 박쥐 n/φ=3 종:
    1. Desmodus rotundus (일반 뱀파이어박쥐) — 포유류 혈액
    2. Diaemus youngi (흰날개뱀파이어박쥐) — 주로 조류 혈액
    3. Diphylla ecaudata (털다리뱀파이어박쥐) — 주로 조류 혈액
    현생 뱀파이어박쥐 = 정확히 n/φ=3 종 ✓

  반-광합성(anti-photosynthesis) 개념:
    광합성: CO₂ + H₂O + 빛 → C₆H₁₂O₆ + O₂
    뱀파이어: 빛 회피 + 혈액(Fe, 단백질) → 에너지
    → 에너지원(빛 vs 피), 활동시간(낮 vs 밤), 철 역할(엽록소 Mg vs 헤모글로빈 Fe) 모두 반전

  출처: Greenhall et al., J Mammal 64(3), 1983; BT-101, BT-103
  렌즈: thermodynamics, evolution, consciousness, symmetry
  교차: BT-101 (광합성 양자수율 σ-τ=8), BT-103 (광합성 완전 n=6 화학양론)

  등급: EXACT (뱀파이어박쥐 3종 = n/φ), CLOSE (φ=2 이중성 프레임워크)
  현생 뱀파이어박쥐 정확히 3종 = n/φ는 분류학적 확정.
  φ=2 에너지 전략 이중성은 개념적 프레임워크로 정량적 EXACT는 아님.
