### H-MS-01: Carbon Z=6=n — EXACT (유지)

```
  검증:
    ┌───────────────────────────────────────────────────────────────┐
    │ 항목            │ 주장        │ 실제        │ 판정          │
    ├─────────────────┼─────────────┼─────────────┼───────────────┤
    │ 원자번호        │ Z=6=n       │ Z=6         │ EXACT ✓      │
    │ 가전자          │ 4=τ         │ 4 (2s²2p²)  │ EXACT ✓      │
    │ 혼성 종류       │ 3=n/φ       │ sp,sp²,sp³  │ EXACT ✓      │
    │ 주요 동소체     │ 4=τ         │ 4 (관례)    │ CLOSE        │
    └───────────────────────────────────────────────────────────────┘

  출처: IUPAC, Z=6 확정. 전자배치는 양자역학에서 유도.
  혼성 sp/sp²/sp³ = 3종은 화학에서 확립.

  세부 조정:
    Z=6, 가전자=4, 혼성=3은 물리적으로 확정 → EXACT 유지.
    동소체=4는 CLOSE (분류 관례 의존).
    종합: 3/4 항목 EXACT → 전체 EXACT 유지.

  최종 Grade: EXACT
```

### H-MS-02: Carbon 동소체 = τ = 4 — CLOSE (유지)

```
  검증:
    주요 동소체: 다이아몬드, 그래파이트, 풀러렌, CNT = 4종 (교과서)
    추가 형태: 그래핀, 비정질탄소, 카바인, 론스데일라이트 등
    차원별 분류: 0D/1D/2D/3D = 4종 (물리적 근거 더 강함)

  판정:
    "4종"은 어떤 기준으로 세느냐에 따라 4~8+.
    0D/1D/2D/3D 차원 분류 = τ = 4는 더 견고하지만,
    이 논증은 "차원이 4종" → 탄소 고유 성질이 아닌 3D+1 공간의 성질.

  최종 Grade: CLOSE (유지)
```

### H-MS-04: 벤젠 C₆H₆ = n — EXACT (유지)

```
  검증:
    분자식 C₆H₆: 탄소 6 = n, 수소 6 = n, 총 원자 12 = σ

  공명 에너지: 150 kJ/mol ≈ 36 kcal/mol. 36 = σ·(n/φ) 매칭은
  kcal 단위 의존이므로 추가 매칭으로 포함하지 않음.

  최종 Grade: EXACT
```

### H-MS-05: 풀러렌 C₆₀ = σ·sopfr = 60 — EXACT (유지)

```
  검증:
    C₆₀: Kroto, Curl, Smalley (1985), 노벨상 1996.
    탄소 원자 수: 60 (확정)
    오각형: 12 (오일러 공식에서 필연: V-E+F=2, 3-정규 그래프)
    육각형: 20 (60-12×5)/6+12... 실제 계산:
      V=60, E=90, F=32, 오각형=12, 육각형=20 ✓

  n=6 매칭:
    60 = σ·sopfr = 12·5                    ✓ EXACT
    12 오각형 = σ                           ✓ EXACT
    20 육각형 = τ·sopfr = 4·5              ✓ EXACT
    32 면 = 2^sopfr = 2^5                  ✓ EXACT

  BUT: 60 = 2²·3·5 = 다수의 n=6 표현 가능 (overfitting 우려).
  오각형=12=σ는 위상수학적 필연 (모든 폐곡면 풀러렌에 12개 오각형).

  최종 Grade: EXACT (4개 매칭 모두 정확, 물리적 근거 확고)
```

### H-MS-06: 다이아몬드 단위셀 = σ-τ = 8 — EXACT (유지)

```
  검증:

  최종 Grade: EXACT
```

### H-MS-07: FCC/HCP CN=σ=12 — EXACT (유지)

```
  검증:
    FCC 배위수: 12 (같은 층 4 + 위 4 + 아래 4)
    HCP 배위수: 12 (같은 층 6 + 위 3 + 아래 3)

  주의: FCC 같은 층 이웃 분해가 가설과 다름.
    가설: 같은 층 6, 위 3, 아래 3
    실제 FCC: 같은 층 4, 위 4, 아래 4 (FCC)
    실제 HCP: 같은 층 6, 위 3, 아래 3 (HCP)

  가설의 분해는 HCP에 맞고 FCC에는 부정확.
  그러나 총 배위수 12 = σ는 두 구조 모두 정확.

  충전률: π√2/6 ≈ 0.7405... = 74.05%
    여기서 분모에 6=n 등장! (정확히 π√2/n)

  BCC CN=8=σ-τ, SC CN=6=n, diamond CN=4=τ 래더도 확인.

  최종 Grade: EXACT
```

### H-MS-16: STM 해상도 0.1nm = 1/(σ-φ) — CLOSE (유지)

```
  검증:
    STM 사양 (Binnig & Rohrer, 1986 노벨상):
    수평: 0.1 nm (원자 해상도)
    수직: 0.01 nm (피코미터 해상도)

  0.1nm = 원자 크기 스케일. Angstrom 단위 정의와 일치.
  터널링 감쇠 상수 κ ≈ 10 nm⁻¹ = (σ-φ) nm⁻¹ 검증:
    κ = √(2mΦ)/ℏ, Φ ≈ 4eV
    = √(2 × 9.11×10⁻³¹ × 4 × 1.6×10⁻¹⁹) / (1.055×10⁻³⁴)
    ≈ 1.02 × 10¹⁰ m⁻¹ = 10.2 nm⁻¹ ≈ σ-φ nm⁻¹ ✓

  최종 Grade: CLOSE (대표적 사양, 물리적 근거 견고)
```

### H-MS-28: 자기복제 2배 = φ — CLOSE (하향 조정)

```
  검증:

    φ=2 매칭은 정확하지만, 2는 가장 작은 소수이자 가장 기본적인 정수.
    φ(n)=2는 n=3,4,6,8,10,12,... 에서 성립 → n=6에 특이적이지 않음.

  원본 EXACT → CLOSE로 하향:
    "자기복제 = 2배"는 보편적 사실이지 n=6 특이성이 아님.
    φ(6)=2라는 표현은 가능하지만 φ(4)=2, φ(3)=2이기도 함.

  최종 Grade: CLOSE (φ=2 매칭은 정확하나 n=6 특이성 부족)
```

### H-MS-29: Avogadro 선행 6 = n — WEAK (하향 조정)

```
  검증:
    N_A = 6.02214076 × 10²³ mol⁻¹
    선행 자릿수 6 = n ✓

  원본 CLOSE → WEAK로 하향:
    N_A의 수치값은 ¹²C 기준 mol 정의에서 유래.
    12g of ¹²C에서 12=σ → 이미 탄소 기반 정의.
    선행 자릿수 6은 "12g of A=12 원자 → 0.5 × 10²⁴ ≈ 6 × 10²³"에서 유래.
    순환적 관계: 탄소 기반 → 6이 나옴 → 탄소 Z=6 연결.

  지수 23: J₂-μ = 24-1 = 23 매칭은 흥미롭지만 SI 단위 의존.

  최종 Grade: WEAK (순환적 정의 + 단위계 의존)
```

### H-MS-30: 주기율표 구조 — WEAK (하향 조정)

```
  검증:
    ┌───────────────────────────────────────────────────────────────┐
    │ 항목           │ 주장             │ 실제        │ 판정        │
    ├────────────────┼──────────────────┼─────────────┼─────────────┤
    │ 족 수          │ 18 = n·(n/φ)     │ 18          │ EXACT ✓    │

    └───────────────────────────────────────────────────────────────┘

  원본 CLOSE → WEAK로 하향:
    블록 크기 {2,6,10,14} = 2(2l+1)은 양자역학의 결과.
    각운동량 양자수 l = 0,1,2,3에서 직접 유도.
    n=6 함수로 "표현 가능"하다는 것은 사후적 매핑이며 인과성 없음.
    18 = n·(n/φ)는 정확하지만 18 = 2(1+3+5) = 합 of odd numbers.
    118 = σ(σ-φ)-φ = 120-2: σ(σ-φ)=120은 5!=120이기도 함 → overfitting.

  최종 Grade: WEAK (수치 매칭은 정확하나 물리적 인과성 없음, 사후적 매핑)
```
