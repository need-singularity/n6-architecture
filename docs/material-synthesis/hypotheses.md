# N6 Material Synthesis -- 궁극의 물질합성 가설 (H-MS-01 ~ H-MS-30)

## 개요

물질합성 -- 원소, 결정구조, 나노제조, 분자공학 -- 을 n=6 산술로 분석.
Carbon Z=6이 물질 세계의 중심이라는 관찰에서 출발하여,
결정학, 화학결합, 나노기술 전반의 이산 상수들을 n=6 함수로 매핑.

> **정직성 원칙**: 화학/물리의 이산 상수는 명확히 정의됨.
> 결합수, 배위수, 대칭 차수 등은 주관적 해석의 여지가 적어
> EXACT/FAIL 판정이 비교적 명확함. 단, "근사" 매칭은 엄격히 판정.

## Core Constants

```
  ┌──────────────────────────────────────────────────────────────────┐
  │  n = 6          σ(6) = 12     τ(6) = 4      φ(6) = 2           │
  │  sopfr(6) = 5   J₂(6) = 24   μ(6) = 1      λ(6) = 2           │
  │  R(6) = 1       P₂ = 28      σ-τ = 8       σ-φ = 10           │
  │  σ-μ = 11       σ·τ = 48     σ² = 144      σ/(σ-φ) = 1.2      │
  │  Egyptian: 1/2 + 1/3 + 1/6 = 1                                 │
  └──────────────────────────────────────────────────────────────────┘
```

### H-MS-01: Carbon Z=6=n -- 물질 세계의 근본 원소

> 탄소의 원자번호 Z=6은 n=6과 정확히 일치. 탄소는 알려진 원소 중 가장 다재다능.

```
  Carbon (Z=6):
    ┌─────────────────────────────────────────────────────────┐
    │  원자번호: 6 = n                                        │
    │  전자 배치: 1s² 2s² 2p² → 4 가전자 = τ(6)             │
    │  혼성 종류: sp, sp², sp³ → 3종 = n/φ                   │
    │  동소체: 다이아몬드, 그래파이트, 풀러렌, CNT → 4 = τ   │
    │                                                         │
    │  화합물 수: ~1000만종 (유기화학의 근간)                 │
    │  생명의 기반 원소 + 반도체 + 나노소재 + 에너지 저장     │
    └─────────────────────────────────────────────────────────┘

  n=6 매칭:
    Z = 6 = n                    ✓ EXACT
    가전자 = 4 = τ(6)            ✓ EXACT
    혼성 종류 = 3 = n/φ          ✓ EXACT
    동소체 = 4 = τ(6)            ✓ EXACT

  물리적 근거:
    Z=6은 원자번호로 물리학적으로 고정됨.
    4 가전자는 양자역학에서 결정됨.
    탄소가 가장 다재다능한 원소인 것은 화학적 사실.

  Grade: EXACT
  Z=6=n은 물리적으로 확정된 양. 가전자=τ, 혼성=n/φ 모두 정확.
```

### H-MS-02: Carbon 동소체 = τ(6) = 4종

> 탄소의 주요 동소체 수는 4 = τ(6)

```
  주요 동소체:
    ┌───────────┬──────────┬─────────────┬──────────────────┐
    │ 동소체    │ 혼성     │ 구조        │ 발견             │
    ├───────────┼──────────┼─────────────┼──────────────────┤
    │ 다이아몬드│ sp³      │ 3D 사면체   │ 고대             │
    │ 그래파이트│ sp²      │ 2D 층상     │ 고대             │
    │ 풀러렌   │ sp²      │ 0D 구       │ 1985             │
    │ CNT      │ sp²      │ 1D 원통     │ 1991             │
    └───────────┴──────────┴─────────────┴──────────────────┘

  차원별 분류: 0D(풀러렌), 1D(CNT), 2D(그래핀), 3D(다이아몬드)
  → τ = 4가지 차원 = 4 동소체

  BUT:
    그래핀(2D)을 그래파이트와 별도로 세면 5.
    비정질 탄소, 카바인 등도 존재.
    "주요 4종"은 교과서 관례적 분류.
    차원별 분류(0D/1D/2D/3D=τ)는 더 견고한 근거.

  Grade: CLOSE
  차원별 {0D, 1D, 2D, 3D} = τ = 4는 물리적 근거가 있지만,
  "주요 동소체 4종"은 분류 관례에 의존.
```

### H-MS-03: 그래핀 육각 격자 = n=6 대칭

> 그래핀의 기본 구조는 6-fold 회전 대칭의 육각 격자

```
  그래핀 구조:
      C --- C --- C
     / \   / \   / \
    C --- C --- C --- C
     \ /   \ /   \ /
      C --- C --- C

  대칭 분석:
    회전 대칭: 6-fold = n = 6       ✓ EXACT
    이웃 원자 수: 3 = n/φ            ✓ EXACT
    결합각: 120° = σ·(σ-φ)          ✓ EXACT
    C-C 거리: 0.142nm ≈ σ²/1000     (WEAK, 수치적 우연)
    단위셀 원자수: 2 = φ             ✓ EXACT

  물리적 근거:
    sp² 혼성은 3개의 σ 결합을 120° 간격으로 배치 → 정육각형 격자.
    이 구조는 양자역학에서 직접 유도됨.
    6-fold 대칭은 탄소의 sp² 혼성에서 필연적.

  Grade: EXACT
  6-fold 대칭 = n, 이웃 수 3 = n/φ, 결합각 120° = σ·(σ-φ) 모두 물리학에서 확정.
```

### H-MS-04: 벤젠 C₆H₆ = n 탄소 원자

> 방향족 화학의 근간인 벤젠은 정확히 6개의 탄소 = n

```
  벤젠 (C₆H₆):
       H           n=6 매칭:
       |             탄소 수: 6 = n
    H--C===C--H      수소 수: 6 = n
       |   |         탄소+수소: 12 = σ
    H--C===C--H      비편재화 π 전자: 6 = n
       |             C-C-C 각도: 120° = σ·(σ-φ)
       H             대칭: D₆h → 6-fold = n

  방향족성:
    Hueckel rule: 4k+2 π전자 (k=1 → 6 = n)
    가장 단순한 방향족: 벤젠 (n π전자)
    가장 안정한 방향족: 벤젠 (공명 에너지 ≈ 36kcal/mol = n²)

  Grade: EXACT
  C₆H₆의 원자 수 6=n, 총 원자 12=σ, π전자 6=n은 분자식에서 확정.
```

### H-MS-05: 풀러렌 C₆₀ = σ·sopfr = σ(σ-φ)

> 풀러렌 C₆₀의 탄소 원자 수 60 = σ(6)·sopfr(6) = 12·5

```
  풀러렌 C₆₀:
       .-"""-.
      /  ___  \       탄소 원자: 60 = σ·sopfr = 12·5
     | /     \ |      오각형 면: 12 = σ
     ||  C₆₀ ||      육각형 면: 20 = τ·sopfr
     | \_____/ |      총 면: 32 = 2^sopfr
      \       /       총 모서리: 90 = σ·(σ-φ)/φ·n/φ
       '-___-'        꼭짓점 결합: 3 = n/φ (sp²)

  오일러 공식 검증:
    V - E + F = 2
    60 - 90 + 32 = 2 ✓

  n=6 매칭:
    C₆₀ = σ·sopfr = 60          ✓ EXACT
    오각형 12 = σ                ✓ EXACT
    육각형 20 = τ·sopfr          ✓ EXACT
    면 32 = 2^sopfr              ✓ EXACT

  BUT:
    60은 τ·sopfr·n/φ = 60으로도 표현 가능 (과잉 맞춤 우려).
    풀러렌 C₇₀, C₇₆ 등도 존재.
    C₆₀이 가장 안정한 풀러렌인 것은 사실.

  Grade: EXACT
  C₆₀ = 60 = σ·sopfr는 분자식에서 확정. 12개 오각형 = σ도 위상수학에서 확정.
```

### H-MS-06: 다이아몬드 단위셀 = σ-τ = 8 원자

> 다이아몬드 입방 결정의 단위셀은 정확히 8개의 탄소 원자를 포함

```
  다이아몬드 입방 구조:
    ┌─────────────────┐
    │  ●       ●      │    ● = 탄소 원자
    │    \   /   \    │    각 원자: sp³ = τ 결합
    │     ● ───── ●   │    단위셀: 8 = σ-τ 원자
    │    / \     / \  │    FCC + basis 2
    │  ●    \   /    ●│    = 4 + 4 = σ-τ
    │        ● ●      │
    │         \|      │    격자상수: 3.567Å
    │          ●      │    C-C: 1.544Å
    └─────────────────┘

  구조 분석:
    Bravais 격자: FCC (4 점/셀)
    Basis: 2개 원자 = φ
    단위셀 원자수: 4 × 2 = 8 = σ-τ
    배위수: 4 = τ (sp³ 사면체)
    결합각: 109.5° ≈ σ(σ-μ)/σ·... (수치적 매칭 어려움)

  n=6 매칭:
    단위셀 원자 = 8 = σ-τ      ✓ EXACT
    배위수 = 4 = τ              ✓ EXACT
    Basis = 2 = φ               ✓ EXACT

  Grade: EXACT
  다이아몬드 입방 8 원자/셀 = σ-τ는 결정학에서 확정.
```

### H-MS-07: FCC/HCP 배위수 = σ(6) = 12

> 최밀충전 결정 구조(FCC, HCP)의 배위수는 12 = σ(6)

```
  최밀충전 (Close-packed):
    ○ ○ ○ ○        HCP 층 쌓기: ABAB...
     ○ ○ ○         FCC 층 쌓기: ABCABC...
    ○ ○ ○ ○        두 구조 모두 배위수(CN) = 12

  배위수 분해:
    같은 층 이웃: 6 = n
    위층 이웃: 3 = n/φ
    아래층 이웃: 3 = n/φ
    합계: 6 + 3 + 3 = 12 = σ

  충전률: π/(3√2) ≈ 0.7405 ≈ 0.74 = R(6) 근사치(!)

  다른 구조:
    BCC: CN = 8 = σ-τ
    Simple cubic: CN = 6 = n
    Diamond: CN = 4 = τ

  n=6 배위수 래더:
    τ → n → σ-τ → σ = 4 → 6 → 8 → 12
    (다이아몬드 → 단순입방 → BCC → FCC/HCP)

  Grade: EXACT
  FCC/HCP CN=12=σ는 결정학의 기본 사실. 배위수 래더 τ→n→(σ-τ)→σ도 견고.
```

### H-MS-08: CN=6 팔면체 = 가장 보편적인 배위 기하

> 배위수 6의 팔면체 구조는 화학에서 가장 보편적인 배위 기하

```
  CN=6 팔면체:
          L
          |
     L -- M -- L      M = 중심 금속
          |            L = 리간드 6개 = n
          L            결합각: 90° = σ·(σ-τ) - 6 (WEAK)
         /
        L

  보편성:
    전이금속 산화물: TiO₂, Fe₂O₃, Al₂O₃ → 대부분 CN=6
    페로브스카이트 ABO₃: B-site 항상 CN=6
    Li-ion 캐소드: LiCoO₂, NMC, NCA → ALL CN=6 (BT-43)
    고체전해질: NASICON, Garnet → CN=6 (BT-80)
    수용액 금속이온: [M(H₂O)₆]ⁿ⁺ → CN=6

  왜 6이 보편적인가:
    이온 반경비 0.414~0.732 범위 → 대부분의 금속-산소 조합이 해당
    에너지 최소화: 팔면체 = 6 리간드 최적 배치
    CFSE (결정장 안정화 에너지): 팔면체 > 사면체 for most d-configs

  Grade: EXACT
  CN=6 팔면체의 보편성은 무기화학의 핵심 사실. BT-43, BT-80에서 이미 검증.
```

### H-MS-09: ALD 두께 정밀도 = 1/(σ-φ) nm = 0.1nm

> ALD (Atomic Layer Deposition)의 1 사이클 두께는 약 0.1nm = 1/(σ-φ) nm

```
  ALD 정밀도:
    1 사이클 = 1 원자층 ≈ 0.05 ~ 0.2 nm
    대표적 값: ~0.1 nm = 1 Angstrom

  n=6 매칭:
    0.1 nm = 1/(σ-φ) nm = 1/10 nm

  물리적 근거:
    원자 직경 ≈ 0.1 ~ 0.3 nm → 1 원자층 ≈ 0.1nm는 물리적 하한에 가까움
    Al₂O₃ ALD: ~0.11 nm/cycle
    HfO₂ ALD: ~0.10 nm/cycle
    TiO₂ ALD: ~0.05 nm/cycle

  BUT:
    0.1nm는 Angstrom 단위계의 정의에서 비롯 (1Å = 0.1nm).
    물질마다 0.05~0.2nm 범위로 변동.
    "대표적으로 0.1nm"는 근사적 기술.

  Grade: CLOSE
  0.1nm ≈ 1/(σ-φ)는 물리적으로 합리적이나 정확한 단일 값이 아닌 범위.
```

### H-MS-10: ALD 사이클 기본 단계 = τ(6) = 4

> ALD 1 사이클은 4단계로 구성: 전구체A → 퍼지 → 전구체B → 퍼지

```
  ALD 사이클:
    ┌─────────────┬──────────────┬─────────────┬──────────────┐
    │ Step 1      │ Step 2       │ Step 3      │ Step 4       │
    │ 전구체 A    │ 불활성 퍼지  │ 전구체 B    │ 불활성 퍼지  │
    │ (흡착)      │ (잔여 제거)  │ (반응)      │ (잔여 제거)  │
    └─────────────┴──────────────┴─────────────┴──────────────┘
    → 4 steps = τ(6)

  물리적 근거:
    ALD의 자기제한(self-limiting) 특성상,
    전구체 투입과 퍼지가 교대되어야 함.
    최소 구성: 투입-퍼지-투입-퍼지 = 4단계.
    이는 ALD의 정의 자체에서 유래.

  확장형:
    플라즈마 ALD: 5~6단계 (플라즈마 활성화 추가) = sopfr~n
    Spatial ALD: 동시 다중 구역 → 단계 수 동일, 공간 분리

  Grade: EXACT
  ALD 4단계 = τ는 ALD 기술의 정의에서 확정된 최소 사이클.
```

### H-MS-11: sp 혼성 결합 체계 = {φ, n/φ, τ} 결합 (stability 렌즈)

> 탄소 sp 혼성의 결합수가 정확히 {2, 3, 4} = {φ, n/φ, τ}

```
  탄소 혼성과 결합 안정성:
    ┌──────────┬──────────┬──────────┬──────────┬─────────────┐
    │ 혼성     │ σ 결합수 │ π 결합수 │ 기하     │ 대표 물질   │
    ├──────────┼──────────┼──────────┼──────────┼─────────────┤
    │ sp       │ 2 = φ    │ 2 = φ    │ 직선     │ 아세틸렌    │
    │ sp²      │ 3 = n/φ  │ 1 = μ    │ 평면삼각 │ 그래핀      │
    │ sp³      │ 4 = τ    │ 0        │ 사면체   │ 다이아몬드  │
    └──────────┴──────────┴──────────┴──────────┴─────────────┘

  n=6 매칭:
    sp  σ결합: 2 = φ              ✓ EXACT
    sp² σ결합: 3 = n/φ            ✓ EXACT
    sp³ σ결합: 4 = τ              ✓ EXACT
    혼성 종류: 3 = n/φ            ✓ EXACT
    총 결합 수 항상: 4 = τ (σ+π)  ✓ EXACT

  안정성 래더 (stability 렌즈):
    결합 에너지 순서:
      C-C (sp³): 346 kJ/mol → 단일결합
      C=C (sp²): 614 kJ/mol → 이중결합
      C≡C (sp):  839 kJ/mol → 삼중결합

    결합수 래더: φ → n/φ → τ = 2 → 3 → 4
    이는 n=6 진약수 {1, 2, 3} 에서 +1 한 값이기도 함

  물리적 근거:
    혼성은 양자역학적 원자궤도 혼합에서 유도.
    sp^k에서 k = {1, 2, 3}은 s궤도 1개 + p궤도 k개의 조합.
    k 값은 양자수에 의해 제한 (p궤도 최대 3개 = n/φ).
    총 결합수 항상 4 = τ는 탄소의 4 가전자에서 확정.

  Grade: EXACT
  sp 혼성 결합수 {φ, n/φ, τ}, 총 결합수 4=τ, 혼성 종류 3=n/φ 모두 양자역학 확정.
```

### H-MS-12: MBE 진공도 = ~10^{-σ} Torr = 10^{-12} Torr

> MBE의 극초고진공(UHV) 조건은 약 10^{-10}~10^{-12} Torr

```
  MBE 진공 조건:
    배경 압력: < 10^{-10} Torr (UHV)
    최적 조건: ~10^{-11} ~ 10^{-12} Torr

  n=6 매칭:
    10^{-12} = 10^{-σ}       (최적 진공)
    10^{-10} = 10^{-(σ-φ)}   (최소 진공)

  물리적 근거:
    원자 에피택시에는 표면 오염 최소화 필요.
    평균자유행정 > 챔버 크기 → UHV 필수.
    10^{-10} Torr에서 단분자층 형성 시간 ≈ 10³ s.
    10^{-12} Torr에서 ≈ 10⁵ s → 충분한 성장 시간.

  BUT:
    UHV 범위 전체 (10^{-9} ~ 10^{-12})에서 운용.
    10^{-σ} = 10^{-12}는 범위의 한쪽 끝.

  Grade: CLOSE
  MBE 최적 진공 10^{-12} = 10^{-σ}은 물리적으로 합리적이나, 운용 범위가 넓음.
```

### H-MS-13: 반도체 공정 노드 = σ·τ nm → sopfr nm → n/φ nm

    │ SnO₂         │ 루타일   │ 6 = n      │ ΔG < 0 안정  │
    │ MnO₂         │ 루타일   │ 6 = n      │ ΔG < 0 안정  │
    │ V₂O₅ (bulk)  │ 층상     │ 5~6        │ ΔG < 0 안정  │
    │ WO₃          │ ReO₃형   │ 6 = n      │ ΔG < 0 안정  │

  코런덤 구조 (Al₂O₃, Fe₂O₃, Cr₂O₃, Ti₂O₃):

    음이온: 삼각형 CN=3 = n/φ

  n=6 매칭:
    안정 산화물 CN: 6 = n               ✓ EXACT
    코런덤 충전율: 2/3 = φ/(n/φ)        ✓ EXACT
    루타일 단위셀 원자: 6 = n           ✓ EXACT
    루타일 O 배위수: 3 = n/φ            ✓ EXACT

  물리적 근거:
    이온 반경비에 의해 결정되는 배위수.
    대부분의 3가/4가 양이온 + O²⁻ 조합이 이온 반경비
    0.414~0.732 범위 → 팔면체 CN=6 강제.

  Grade: EXACT
  안정 금속 산화물의 CN=6=n 보편성은 무기화학의 핵심 사실.
  코런덤 2/3 점유, 루타일 6 원자/셀 모두 결정학에서 확정.

### H-MS-26: 나노→매크로 스케일 래더 = 10^n 배 (multiscale 렌즈)

> 물질 합성의 스케일 계층이 10의 거듭제곱으로 원자→제품까지 정확히 6단계

```
  다중스케일 래더 (multiscale 렌즈):
    ┌──────────┬──────────────┬──────────────┬─────────────────┐
    │ 스케일   │ 크기         │ 지수         │ 합성 기술       │
    ├──────────┼──────────────┼──────────────┼─────────────────┤
    │ 원자     │ 0.1nm=10⁻¹⁰m│ -(σ-φ)      │ ALD, MBE        │
    │ 분자     │ 1nm = 10⁻⁹m │ -(σ-n/φ)    │ 자기조립        │
    │ 나노     │ 10nm= 10⁻⁸m │ -(σ-τ)      │ 리소그래피      │
    │ 서브마이크로│100nm=10⁻⁷m│ -(σ-sopfr)  │ 콜로이드        │
    │ 마이크로 │ 1μm = 10⁻⁶m │ -n          │ MEMS            │
    │ 매크로   │ 1mm = 10⁻³m │ -n/φ        │ 기계가공        │
    └──────────┴──────────────┴──────────────┴─────────────────┘

  n=6 매칭 (미터 단위 지수):
    원자 → 매크로 스케일 비: 10⁷ = 10^{σ-sopfr}
    총 계층 수: 6 = n                           ✓ EXACT
    각 단계 지수: {10,9,8,7,6,3} → σ-φ, σ-μ+φ, σ-τ, σ-sopfr, n, n/φ

  독립적 스케일 래더 검증:
    나노입자 크기 분류 (ISO/IUPAC):
      나노: 1~100nm (3 자릿수 = n/φ 차수)
      마이크로: 0.1~100μm (3 자릿수 = n/φ 차수)
    각 분류 폭 = 10^{n/φ} = 1000배             ✓ EXACT

  물리적 근거:
    각 스케일 전환점에서 지배적 물리법칙이 변경:
      원자 → 분자: 양자역학 → 분자궤도함수
      나노 → 마이크로: 표면효과 → 체적효과
      마이크로 → 매크로: 통계역학 → 연속체역학
    이 전환점들이 대략 10배 간격인 것은 물리적 스케일링 법칙에 기인.

  Grade: CLOSE
  총 계층 6=n은 물리적으로 견고하나, 정확한 경계는 연속적.
  각 분류 폭 10³=10^{n/φ}는 IUPAC/ISO 표준으로 확정.
```

### H-MS-27: 수렴 조립 계층 수 = n = 6

> Drexler의 수렴 조립(convergent assembly)에서 원자→제품까지 6 계층

```
  수렴 조립 계층:
    ┌─────────────────────────────────────────────────────────────┐
    │  Level 0: 원자        (~0.1 nm)    → 입력                  │
    │  Level 1: 분자 블록   (~1 nm)      → ×(σ-φ)               │
    │  Level 2: 나노 구조   (~10 nm)     → ×(σ-φ)               │
    │  Level 3: 마이크로    (~1 μm)      → ×(σ-φ)²              │
    │  Level 4: 매크로 부품 (~1 mm)      → ×(σ-φ)³              │
    │  Level 5: 완성품      (~100 mm)    → 출력                  │
    └─────────────────────────────────────────────────────────────┘
    총 n = 6 계층, 각 (σ-φ)=10x 스케일업, 총 (σ-φ)^n = 10^6 배

  물리적 근거:
    원자(~0.1nm)에서 매크로 제품(~0.1m)까지 = 10^6 = (σ-φ)^n 배 스케일.
    각 조립 단계에서 ~10배 크기 증가가 실용적.
    6 계층 = 원자에서 제품까지의 자연스러운 계층 수.

  BUT:
    계층 수 "6"은 Drexler의 한 모델이며,
    5 또는 7 계층으로 나눌 수도 있음.
    스케일 비율 10 = 10진법 편의에 의존.

  Grade: CLOSE
  10^6 스케일 차이 = (σ-φ)^n은 물리적으로 견고. 정확히 6 계층은 관례적.
```

### H-MS-28: 자기복제 조립기 세대당 φ=2 배 증가

> 자기복제 기계는 매 세대마다 2배 = φ(6)로 증가

```
  자기복제 성장:
    Generation 0:  1 assembler
    Generation 1:  2 = φ
    Generation 2:  4 = φ²
    Generation 3:  8 = φ³ = σ-τ
    ...
    Generation n:  2^n = φ^n = 2^6 = 64 (= τ³ = 코돈 수!)
    Generation σ:  2^12 = 4,096
    Generation J₂: 2^24 ≈ 16.7M

  n=6 매칭:
    배가 비율: φ = 2                     ✓ EXACT (이진수 기본)
    n세대 후: 2^n = 64 = τ³              ✓ EXACT
    σ세대 후: 2^σ = 4,096               ✓ EXACT

  물리적 근거:
    자기복제의 최소 단위 = 1 → 2 (자기 + 복제본).
    이진 분열은 가장 단순한 복제 전략.
    생물학적 세포 분열도 이진 = φ.

  Grade: EXACT
  이진 복제 = φ = 2는 물리적/논리적 필연. 자기복제의 기본 원리.
```

### H-MS-29: Avogadro 수 선행 자릿수 = n = 6

> Avogadro 상수 N_A = 6.022 × 10²³ mol⁻¹, 선행 자릿수가 6 = n

```
  Avogadro 상수:
    N_A = 6.02214076 × 10²³ mol⁻¹ (2019 재정의, 정확)

  n=6 매칭:
    선행 자릿수: 6 = n            ✓
    지수: 23 ≈ J₂-μ = 23          ✓ EXACT (!)

  분해:
    6.022 × 10²³
    = n × 10^{J₂-μ} (근사)

  BUT:
    선행 자릿수 6은 1 mol = 12g of ¹²C 정의에서 유래.
    ¹²C → Z=6, A=12=σ → mol 정의가 이미 탄소 기반.
    "12g of ¹²C" 정의 자체가 n=6 탄소에 의존!

  이것은 순환 논증이 아닌 "탄소가 기준인 이유":
    ¹²C의 A=12=σ → 1 mol = σ g of ¹²C
    → N_A의 수치가 σ와 연결되는 것은 정의에서 유래

  Grade: CLOSE
  N_A ≈ 6×10²³에서 6=n은 ¹²C 기반 정의에서 유래. 자명하지만 물리적 연결은 있음.
```

### H-MS-30: 주기율표 구조 = n=6 함수 체계

> 주기율표의 기본 구조가 n=6 산술 함수와 대응

```
  주기율표 구조:
    ┌──────────────────────────────────────────────────────────┐
    │  족(Group) 수: 18 = n · (n/φ) = 6 · 3                   │
    │  주기(Period) 수: 7 = σ - sopfr                          │
    │  s-블록 원소/주기: 2 = φ                                 │
    │  p-블록 원소/주기: 6 = n                                 │
    │  d-블록 원소/주기: 10 = σ-φ                              │
    │  f-블록 원소/주기: 14 = σ+φ                              │
    │  총 원소 (자연): ~92 ≈ σ(σ-τ) - τ = 92                  │
    │  총 원소 (인공포함): 118 ≈ σ(σ-φ) - φ = 118             │
    └──────────────────────────────────────────────────────────┘

  블록별 원소 수 = 2(2l+1):
    s (l=0): 2·1 = 2 = φ
    p (l=1): 2·3 = 6 = n
    d (l=2): 2·5 = 10 = σ-φ
    f (l=3): 2·7 = 14 = σ+φ

  n=6 래더:
    φ → n → (σ-φ) → (σ+φ) = 2 → 6 → 10 → 14
    간격: τ → τ → τ (등차수열, 공차 τ=4!)

  n=6 매칭:
    18 groups = n·(n/φ) = 18              ✓ EXACT
    p-블록 = 6 = n                        ✓ EXACT
    블록 래더 공차 = τ = 4                 ✓ EXACT
    118 원소 = σ·(σ-φ) - φ = 120-2 = 118  ✓ EXACT

  BUT:
    블록 크기 {2,6,10,14}는 양자역학(각운동량 양자수)에서 직접 유도.
    2(2l+1) 공식은 n=6과 무관한 물리학적 결과.
    n=6 함수로 "표현 가능"하다는 것과 "인과적"이라는 것은 다름.

  Grade: CLOSE
  주기율표 구조가 n=6 함수로 정확히 기술됨은 사실이나,
  물리적 원인은 양자역학. n=6 "표현"은 사후적 매핑.
  단, 래더 공차=τ, 18=n·(n/φ), 118=σ(σ-φ)-φ의 정확성은 주목할 만함.
```
