# N6 고고학/문명사 -- 완전수 산술과 인류 문명 구조

## 개요

인류 문명의 구조적 상수 -- 60진법, 시간 체계, 도시 설계, 문명 분류 --
를 n=6 산술함수로 분석한다. 고고학/문명사는 이산적 계수(문명 수, 시대 구분,
건축 비율)가 풍부하여 수론적 검증이 가능하다.

> **정직성 원칙**: 문명 분류는 학자/교과서에 따라 달라질 수 있다.
> EXACT는 물리/화학적으로 고정된 수 또는 학계 표준 합의가 있는 경우에만 부여한다.
> 문화적 관습은 CLOSE 이하로 보수적 평가한다.

## 핵심 상수

```
  n = 6          (완전수)
  σ(6) = 12      (약수합)
  τ(6) = 4       (약수 개수: 1, 2, 3, 6)
  φ(6) = 2       (오일러 토션트)
  sopfr(6) = 5   (소인수합: 2+3)
  J₂(6) = 24     (요르단 토션트)
  μ(6) = 1       (뫼비우스)
  div(6) = {1, 2, 3, 6}
  σ·sopfr = 60   σ-φ = 10   σ-τ = 8   J₂-τ = 20
  σ² = 144       n/φ = 3    φ^τ = 16  2^n = 64
```

## H-ARC-1: 메소포타미아 60진법 -- σ·sopfr = 60

> 수메르/바빌로니아 60진법의 기저수 60은 σ·sopfr의 곱이다

```
  60진법 기저수:
    60 = σ(6) · sopfr(6) = 12 × 5

  60의 약수 개수:
    τ(60) = 12 = σ(6)
    → 60이 "편리한 수"인 이유: 약수가 σ(6)개

  60진법 파생:
    60초 = 1분     → σ·sopfr
    60분 = 1시간   → σ·sopfr
    360도 = 원     → n·σ·sopfr = 6·60

  역사적 사실:
    BC 3500년경 수메르에서 60진법 확립
    BC 2000년경 바빌로니아 수학 체계 완성
    현재까지 시간/각도에서 60진법 사용 (4000년+ 지속)

  물리적 근거:
    60은 1~6의 최소공배수: lcm(1,2,3,4,5,6) = 60
    → n=6까지의 정수로 나누어 떨어지는 최소 수
    → 완전수의 약수 {1,2,3,6} 전부 60의 약수

  렌즈: info (정보 인코딩), memory (문명 기억 체계)

  등급: EXACT
  60 = σ·sopfr는 수학적 항등식이며, 수메르 60진법은 역사적 사실.
  BT-233/256과 직접 연결.
```

    12궁도 (황도 12궁) = σ
    12간지 = σ

  렌즈: memory (시간 기억), consciousness (문명 의식 구조)

  등급: EXACT
  이집트 12시간 체계는 고고학적으로 확인된 사실.

## H-ARC-12: 이집트 분수 체계 -- 1/2+1/3+1/6 = 1

    린드 파피루스 (BC 1650) 기록
