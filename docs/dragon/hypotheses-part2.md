### H-DRG-16: 연소 삼각형 n/φ=3 + 메탄 화학양론 n=6

> 화염의 기본 조건과 메탄 연소의 원자 수가 n=6 산술

```
  연소 삼각형 (Fire Triangle):
    연료(Fuel) + 산소(O₂) + 열(Heat) = n/φ = 3 요소
    이 중 하나라도 제거 → 소화 = n/φ 조건의 필수성
    확장: 연소 사면체(Tetrahedron) = τ = 4 (+ 연쇄반응 추가)

  메탄 연소 화학양론:
    CH₄ + 2O₂ → CO₂ + 2H₂O
    반응물 원자: C(1) + H(4) + O(4) = 9 = (n/φ)²
    생성물 원자: C(1) + O(2) + H(4) + O(2) = 9 = (n/φ)²
    양론 계수 합: 1+2+1+2 = n = 6 ✓

  용의 불 가설 (생물학적 메탄):
    반추동물: 장내 세균(methanogen) → CH₄ 생산
    인광(PH₃): 자연 발화 = 도깨비불(will-o'-the-wisp)
    가설: 내장 CH₄ + 인광 자연 발화 = 생물학적 화염 방사

  화염 온도 래더:
    촛불: ~1,000°C ≈ 10³ = (σ-φ)³
    프로판: ~1,980°C ≈ 2×10³
    아세틸렌: ~3,100°C
    태양 표면: ~5,778°C ≈ n×10³ (근사)

  출처: Glassman, "Combustion" (5th ed.); Zumdahl, "Chemistry" (10th ed.)
  렌즈: boundary, symmetry

  등급: CLOSE
  메탄 양론 계수합 = n = 6은 화학적으로 고정. 연소 삼각형 n/φ = 3은 보편적.
  그러나 화염 온도 래더는 근사이므로 전체적으로 CLOSE.
```

### H-DRG-30: 용의 알(Dragon Egg) = 구(sphere) → 환체(torus) 위상 전이
