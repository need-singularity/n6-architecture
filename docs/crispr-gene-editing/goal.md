## 구현 로드맵

### Mk.I -- CRISPR 최적화 엔진 (1~2년)
- n=6 가이드 RNA 설계 알고리즘 (시드 12nt=sigma 최적화)
- 오프타깃 예측 모델 (PAM+시드+미스매치 6차원 스코어링)
- 실현가능성: 현재 기술로 구현 가능

### Mk.II -- 다중 편집 플랫폼 (3~5년)
- sigma=12 유전자 동시 편집 시스템
- tau=4 수복 경로 선택적 유도 (NHEJ/HDR/MMEJ/프라임)
- LNP+나노봇 하이브리드 전달 (BT-404 연동)
- 실현가능성: 현재 기초연구 진행 중

### Mk.III -- 합성 생물학 통합 (5~10년)
- n=6 모듈 유전자 회로 조립
- Cas-phi 초소형 편집기 + 자율 진단-치료 루프
- 개인 맞춤 유전자 치료 플랫폼 (7000+ 희귀질환 커버)
- 실현가능성: 핵심 요소 기술 확보 후 통합
