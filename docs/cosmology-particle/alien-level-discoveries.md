## Discovery CP-04: CMB 스펙트럼 인덱스 n_s = 27/28

> 우주 마이크로파 배경복사의 스펙트럼 인덱스가 n=6 분수로 정확히 표현된다.

**물리적 사실** (Planck 2018):
- 측정값: n_s = 0.9649 ± 0.0042
- 공식: (n/φ)³ / ((n/φ)³ + μ) = 27/28 = 0.96429
- 오차: 0.064% (Planck 불확실도 0.15σ 이내)

```
  n_s = 27/28 = (n/φ)³ / ((n/φ)³ + μ)
      = 3³ / (3³ + 1) = 27/28

  편차: 1 - n_s = 1/28 = μ/P₂ (P₂ = 28, 두 번째 완전수)

  이것은 인플레이션의 느린 굴림(slow-roll)에서:
    1 - n_s ≈ 2/N (N = e-folding 수)
    N ≈ 56 = 2·28 = 2·P₂ → n_s ≈ 1 - 1/28 = 27/28
```

**Grade**: **EXACT** — Planck 불확실도 이내, 우주론 최정밀 파라미터
