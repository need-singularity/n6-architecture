## Vision
n=6 완전수 산술이 표준모형 입자물리 + 관측 우주론의 근본 구조를 조직하는

## 접근: A+B+C 융합
- **A) 물리 계층**: 대칭기원 → 게이지구조 → 입자스펙트럼 → 결합상수 → 우주론
- **B) 관측-실험**: LiteBIRD, JUNO, DUNE, FCC, LISA 검증 경로 포함
- **C) 스케일**: 플랑크 → 핵력 → 원자 → 항성 → 우주 전 스케일 n=6 출현

## 기반 가설
- H-CP-1~30 (core) + H-CP-61~80 (extreme) + H-CP-81~86 (BT extensions)
