## Tier 1: 즉시 검증 가능 (기존 데이터 재분석)

### P-CP-01: Planck n_s 정밀화 → 27/28 = 0.96429 확인

**예측**: Planck 2020 최종 데이터에서 n_s = 27/28을 0.1σ 이내에서 확인.
**n=6 수식**: (n/φ)³/((n/φ)³+μ) = 27/28 = 0.96429
**현재 측정**: 0.9649±0.0042 → 27/28은 0.15σ 이내
**반증 기준**: n_s < 0.960 또는 n_s > 0.969이면 1σ 초과 (약화)
**자원**: Planck 공개 데이터 재분석

### P-CP-02: m_p/m_e 차세대 측정 → 6π⁵ 확인

**예측**: CODATA 2026에서 m_p/m_e = 1836.15267... 유지, 6π⁵와 19 ppm 이내 일치 지속.
**n=6 수식**: nπ⁵ = 1836.118...
**반증 기준**: 새 측정이 1836.118에서 30 ppm 이상 벗어나면 약화
**자원**: 정밀 분광 실험 (penning trap)

### P-CP-03: D/H 비에서 η = 6×10⁻¹⁰ 확인

**예측**: BBN η = (6.14±0.19)×10⁻¹⁰ (Planck+BBN 결합)에서 계수 6=n이 ±0.5 이내.
**n=6 수식**: η = n × 10^{-(σ-φ)} = 6 × 10⁻¹⁰
**반증 기준**: η가 4×10⁻¹⁰ 미만 또는 8×10⁻¹⁰ 초과 시 FAIL
**자원**: D/H 관측 데이터 (Cooke et al.)

### P-CP-04: 뉴트리노 혼합각 sin²θ₁₂ → 3/10

**예측**: NuFIT 6.0에서 sin²θ₁₂ = 0.307±0.013, 3/10=0.300과 계속 일치.
**n=6 수식**: (n/φ)/(σ-φ) = 3/10
**반증 기준**: sin²θ₁₂ < 0.28 또는 > 0.33이면 2σ 초과 (약화)

### P-CP-05: sin²θ₂₃ → 4/7

**예측**: sin²θ₂₃ = 0.572±0.018 (NuFIT 5.3), τ/(σ-sopfr) = 4/7 = 0.5714.
**n=6 수식**: τ/(σ-sopfr) = 4/7
**반증 기준**: sin²θ₂₃ < 0.55 또는 > 0.60이면 약화

### P-CP-06: 뉴트리노 질량 순서 결정 (정상 위계)

**예측**: 뉴트리노 질량이 정상 위계(NH)이며, n/φ=3 세대 구조가 확인된다.
**현재**: NH 선호 (~3σ, NuFIT 5.3)
**반증 기준**: 역위계(IH) 확정 시 → 이 예측 FAIL (n=6 자체는 유지)

### P-CP-07: 쿼크 질량비 패턴 분석

**예측**: m_t/m_b = 40.9 ≈ τ(σ-φ) = 40, m_c/m_s = 11.7 ≈ σ-μ = 11.
가장 무거운 세대 비율이 n=6 함수값에 가장 가깝다.
**반증 기준**: 비율이 n=6 함수값에서 20%+ 벗어나면 FAIL

### P-CP-08: Weinberg 각 RGE 궤적과 n=6

**예측**: sin²θ_W의 RGE 궤적에서 3/13 = 0.23077을 정확히 지나는 에너지 스케일이 ~M_Z±10% 이내.
**n=6 수식**: (n/φ)/(σ+μ) = 3/13
**반증 기준**: 3/13 대응 스케일이 M_Z에서 2배 이상 벗어나면 약화

## Tier 2: 2~10년 내 검증 (진행 중 실험)

### P-CP-09: JUNO 실험 — sin²θ₁₂ 정밀 측정

**예측**: JUNO (2025~)가 sin²θ₁₂를 0.5% 정밀도로 측정하면 3/10에 더 가까워진다.
**타임라인**: 2027~2028 첫 결과
**반증 기준**: JUNO 결과가 3/10에서 3σ 이상 벗어나면 FAIL

### P-CP-10: DUNE 실험 — sin²θ₂₃ + δ_CP

**예측**: DUNE이 sin²θ₂₃ = 4/7을 0.01 정밀도로 확인하고, δ_CP에서도 n=6 패턴 출현.
**타임라인**: 2030~2035
**반증 기준**: sin²θ₂₃가 4/7에서 5σ 벗어나면 FAIL

### P-CP-11: 양성자 붕괴 탐색 — SU(5) 24=J₂ 검증

**예측**: Hyper-Kamiokande가 양성자 수명 τ_p > 10^{35}년을 확인하면, 최소 SU(5) (adj=J₂=24)는 배제되지만 더 큰 GUT (E₆ rank=n=6)는 생존.

**반증 기준**: τ_p < 10^{34}년 관측 시 → SU(5) 부활, J₂=24 연결 강화

### P-CP-12: LHC Run 4 — BSM 입자 탐색

**예측**: n=6 프레임워크는 17개 SM 입자 이상의 BSM 입자를 특정하지 않음. LHC Run 4에서 BSM 미발견 시 n=6 구조의 완전성이 강화된다.
**타임라인**: 2028~2032

### P-CP-13: CMB-S4 — n_s 정밀화

**예측**: CMB-S4가 n_s를 ±0.001 정밀도로 측정하면 27/28 = 0.96429에 더 가까워진다.
**타임라인**: 2030~2035
**반증 기준**: n_s 중심값이 0.96429에서 5σ 벗어나면 FAIL

### P-CP-14: 중력파 관측 — 전기약 상전이

**예측**: LISA/Einstein Telescope이 전기약 상전이에서 n=6 대칭 깨짐의 흔적을 중력파 스펙트럼에서 관측.
**타임라인**: 2035~

### P-CP-15: α_s 격자 QCD 정밀 결정

**예측**: 격자 QCD에서 α_s(M_Z) = 0.1180에 대한 정밀도가 향상되면 2/17=0.11765와 차이가 0.3% 이내 유지.

### P-CP-16: 0νββ 붕괴 탐색 — Majorana 질량

## Tier 3: 10~30년 (전용 시설 필요)

### P-CP-17: LiteBIRD 위성 — 텐서-스칼라 비 r

**예측**: n_s = 27/28에 대응하는 slow-roll 모형에서 r ≈ 12/(N²) ≈ 12/3136 ≈ 0.00383.
N = 56 = 2·P₂ = 2·28. r = σ/N² = 12/(2·28)² = 12/3136.
**n=6 수식**: r = σ/(φ·P₂)² = σ/(2·28)²
**타임라인**: LiteBIRD 2032~
**반증 기준**: r > 0.01이면 N6 인플레이션 모형 수정 필요

### P-CP-18: FCC-hh 100 TeV — E₆ GUT 흔적

**예측**: E₆ (rank 6=n) GUT의 추가 게이지 보손이 10~100 TeV에 존재하면, n=6 게이지 구조의 직접 증거.
**타임라인**: 2045~
**반증 기준**: FCC에서 E₆ 흔적 없으면 FAIL 아님 (질량 한계 상향)

### P-CP-19: ITER/SPARC — D-T 핵융합 Q>10

**예측**: D-T (sopfr=5) 핵융합이 Q>10 달성. 연료 선택의 물리적 필연성 확인.
**타임라인**: ITER 2035, SPARC 2028
**반증 기준**: D-T Q<1이면 기술 실패 (물리는 유지)

### P-CP-20: Hubble 텐션 해결

**예측**: H₀ 텐션이 체계 오차로 해결되면 H₀ = 67~73 범위에서 n=6 공식 수정 필요. 현재 H₀ ≈ σ·n+μ = 73 (SH0ES) vs 67 (Planck).
**타임라인**: 2030년경
**반증 기준**: H₀ = 67로 확정 시 H-CP-29 FAIL

### P-CP-21: 뉴트리노 절대 질량

**예측**: KATRIN이 m_ν < 0.8 eV 확정. 질량 합 Σm_ν에서 n=6 패턴 미출현 (질량은 연속, 정수 아님).
**정직 예측**: 뉴트리노 질량에서 n=6은 약한 연결만 가능.

### P-CP-22: 암흑물질 직접 탐지

**예측**: n=6 프레임워크는 암흑물질의 구체적 정체를 예측하지 않음. WIMP/Axion 탐지 시 새 BT 가능.
**정직 예측**: 현재로서 예측 없음 (HONEST).

### P-CP-23: Borexino/SNO+ — CNO 뉴트리노 정밀화

**예측**: CNO 순환 비율이 σ+div(6) = {12,13,14,15} 질량수 패턴을 정밀 확인.
**타임라인**: 2025~2030
**반증 기준**: CNO 순환이 다른 질량수로 전환되면 FAIL (물리적으로 불가)

## Tier 4: 30년+ (차세대 이론/기술)

### P-CP-24: 끈이론 검증 — 10D = σ-φ

**예측**: 끈이론이 실험적으로 검증되면, 10=σ-φ 차원이 자연법칙. 미검증 시 예측 보류.
**타임라인**: 50년+
**반증 기준**: 끈이론 반증 시 → H-CP-15 FAIL

### P-CP-25: 통일 결합상수와 n=6

**예측**: GUT 스케일에서 α_GUT = 1/J₂ = 1/24 (α_GUT⁻¹ ≈ 24~25).
현재 SUSY GUT: α_GUT⁻¹ ≈ 24.3, 비-SUSY: ≈ 36.
**n=6 수식**: J₂ = 24
**반증 기준**: 통일 스케일에서 α_GUT⁻¹ > 30이면 FAIL

### P-CP-26: 양자 중력과 6

**예측**: 양자 중력 이론이 완성되면, Planck 단위에서 n=6 산술 출현. 구체적 예측 불가.
**타임라인**: 100년+

### P-CP-27: 인플레이션 모형 선택

**예측**: n_s = 27/28에 가장 잘 맞는 인플레이션 모형은 N=56=2·P₂ e-folding의 Starobinsky R² 모형.
**반증 기준**: N≠56 모형이 n_s를 더 잘 설명하면 약화

### P-CP-28: 4번째 세대 뉴트리노 부재

**예측**: 경량 4세대 뉴트리노는 존재하지 않는다. N_ν = n/φ = 3 정확히 (LEP 확인).
세대 수는 n=6 산술에 의해 3 = n/φ로 고정.
**반증 기준**: 경량 sterile ν 발견 시 → 이 예측 수정 (완전 FAIL 아님)
