## 1. 불가능성 정리 목록

### 불가능성 1: 시공간 차원은 반드시 τ(6)=4

**정리**: 안정한 행성 궤도와 원자가 존재하려면 정확히 3+1=4 차원이 필요하다.

```
  증명:
  공간 d차원에서 중력 포텐셜: V(r) ∝ r^{-(d-1)}
  
  d=2: V ∝ ln(r) → 궤도 불안정 (탈출 불가)
  d=3: V ∝ 1/r → 안정 궤도 (케플러 법칙)
  d≥4: V ∝ 1/r^{d-1} → 궤도 불안정 (나선형 추락)
  
  수소 원자 (슈뢰딩거 방정식):
  d≥4: 바닥 상태 에너지 → -∞ (원자 붕괴)
  d=3: 안정 바닥 상태 존재
  
  시간 차원:
  t=0: 동역학 없음 (정적 우주)
  t=1: 인과율 + 파동 전파
  t≥2: 닫힌 시간곡선 → 인과율 위반
  
  ∴ 유일한 안정 시공간 = 3+1 = τ(6) = 4
  □
```

### 불가능성 2: 쿼크는 반드시 n=6종, n/φ=3 세대

**정리**: 관측 가능한 우주에서 쿼크 종류는 정확히 6개다.

```
  증명:
  LEP 실험 (Z 보손 폭 측정):
  Γ_Z = Γ_had + N_ν · Γ_ν + Γ_lep
  
  측정: N_ν = 2.9840 ± 0.0082 → 3 경질량 중성미자
  → 렙톤 세대 = n/φ = 3 (실험적 확정)
  → 쿼크 세대 = n/φ = 3 (렙톤-쿼크 대응)
  → 쿼크 종류 = (n/φ) × φ = n = 6
  
  4세대: Z 보손 폭 데이터와 모순
  2세대: CP 위반 설명 불가 (CKM 행렬 3×3 필요)
  
  ∴ 세대 수 = n/φ = 3은 실험적 + 이론적 필연
  □
```

### 불가능성 3: 쿼크 색 수는 반드시 n/φ=3

**정리**: QCD의 색전하는 정확히 3개여야 한다.

```
  증명:
  점근적 자유도: β₀ = 11N_c/3 - 2N_f/3
  
  N_c = 3, N_f = 6: β₀ = 11 - 4 = 7 > 0 → 점근적 자유 (QCD 핵심!)
  N_c = 2: β₀ = 22/3 - 4 = 10/3 > 0 (가능하나 confinement 약함)
  N_c = 4: β₀ = 44/3 - 4 = 32/3 > 0 (가능하나 N_f 제한 강함)
  
  그러나 R-비율 (e⁺e⁻ → hadrons):
  R = N_c · Σ Q_i² = 3 · (4/9+1/9+1/9+4/9+1/9+1/9) = 3 · 12/9 = 4
  실험: R ≈ 4 (← N_c = 3 확정)
  
  N_c = 3 = n/φ 는 실험적으로 확정된 값.
  □
```

### 불가능성 4: 글루온 수는 반드시 σ-τ=8

**정리**: SU(3) 색대칭의 생성원 수 = N_c² - 1 = 8이다.

```
  증명:
  SU(N) 리 군의 생성원 수 = N² - 1
  N_c = n/φ = 3 (불가능성 3에서 증명)
  
  글루온 수 = 3² - 1 = 8 = σ - τ = 12 - 4
  
  이것은 리 군론의 수학적 필연이며, 물리학의 선택이 아님.
  □
```

### 불가능성 5: BBN 경원소는 반드시 τ=4종

**정리**: 빅뱅 핵합성으로 생성되는 경원소는 정확히 4종이다.

```
  증명:
  BBN 조건: T ~ 10⁹ K, t ~ 3분, 바리온/광자 비 ~ 6×10⁻¹⁰
  
  A=1 (H): 가장 단순, 잔여 양성자
  A=2 (D): p+n → D+γ (첫 번째 반응)
  A=3 (He-3): D+p → He-3+γ
  A=4 (He-4): 가장 안정 (결합에너지/핵자 최대)
  
  A=5: 질량갭 (He-5, Li-5 불안정, 10⁻²² s)
  A=8: 질량갭 (Be-8 불안정, 6.7×10⁻¹⁷ s)
  
  → A=5, 8 질량갭이 핵합성을 τ=4종에서 차단
  → Li-7은 미량 (~10⁻¹⁰), 주요 원소가 아님
  
  ∴ BBN 경원소 = τ(6) = 4종
  □
```
