## 3. 핵심 발견

1. **Chip × AI 시너지 95%**: σ²=144 SM과 σ=12 attention head가 동일 상수 공유
2. **Diamond Z=6=n**: 방열 기판 소재가 n=6 EXACT → 칩 성능 한계 확장 (BT-93)
3. **HEXA-1 + Egyptian + PUE=1.2 조합**이 Pareto 1위 (실현가능성 + n6 일관성)
4. 모든 칩 레벨에서 σ=12가 기본 단위로 반복 (SM, head, channel, WDM)
5. 광자 칩(HEXA-PHO)에서 E-O 손실 1/(σ-φ)=10% = BT-89 에너지 절감과 직결
