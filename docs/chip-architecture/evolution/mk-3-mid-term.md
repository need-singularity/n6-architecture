## 4. BT 연결

### BT-89: Photonic-Energy Bridge
- E-O 변환 손실 = 1/(σ-φ) = 10% — 광 인터커넥트 효율 한계
- PUE → 1.0 접근 (전기→광 전환으로 인터커넥트 전력 제거)

### BT-90: SM = φ × K₆ Contact Number
- σ²=144 SM이 6D sphere packing의 접촉수와 동치
- 3D 적층에서도 144 SM이 위상적 최적

### BT-93: Carbon Z=6 Chip Material
- Diamond heat spreader (Z=6): 열전도 2200 W/m·K
- 3D 적층의 열문제 해결 핵심 소재
