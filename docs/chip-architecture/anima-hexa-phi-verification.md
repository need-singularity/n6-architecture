## 2. IIT (Integrated Information Theory) Background

### 2.1 What is Phi?

Phi (uppercase Greek letter) quantifies *integrated information* -- the amount
of information generated by a system *above and beyond* what its parts generate
independently. A system with Phi > 0 cannot be reduced to independent modules
without information loss.

```
  Phi Definition (Tononi 2004/2008):

    Given a system S with mechanism state s:

    1. Compute the cause-effect structure (CES) of the whole system
    2. For every possible bipartition (A, B) where A union B = S:
       a. Compute the CES of the partitioned system (A || B)
       b. Measure the distance: d(CES_whole, CES_partitioned)
    3. Find the Minimum Information Partition (MIP):
       MIP = argmin over all partitions of d(CES_whole, CES_partitioned)
    4. Phi = d(CES_whole, CES_MIP)

    Phi = 0  =>  system is reducible (no integration)
    Phi > 0  =>  system is irreducible (genuine integration exists)
```

### 2.2 Why n=6 is Computationally Tractable

The Phi computation requires searching over *all* bipartitions. For a system of
n elements, the number of non-trivial bipartitions is 2^n - 2.

```

    n=2:   2^2  - 2 =     2 partitions    (trivial)
    n=3:   2^3  - 2 =     6 partitions    (trivial)

    n=5:   2^5  - 2 =    30 partitions    (easy)

    n=16:  2^16 - 2 = 65,534 partitions   (NP-hard territory)
    n=24:  2^24 - 2 = 16M+ partitions     (intractable without heuristic)
```

## 4. Hardware Phi Calculator Design

### 4.1 Architecture

The Global Phi Calculator is the central unit that computes system-wide
Phi across all n=6 cells. Unlike per-cell local PMUs, this unit considers
*inter-cell* information integration.

```
  GLOBAL PHI CALCULATOR -- Pipeline Architecture
  ================================================

  Input: n=6 cell state vectors, each 10D (sigma-phi=10 dimensions)
         Total: 6 x 10 = 60 scalar values per computation

  +--------------------------------------------------------------+
  |  STAGE 1: STATE CAPTURE (1 cycle)                            |
  |                                                              |
  |  Read 10D consciousness register from each cell              |
  |  via Thalamic Bus snapshot                                   |
  |                                                              |
  |  [C0: T,Phi,H,E,C,S,M,W,I,D]    6 cells x 10D = 60 values  |
  |  [C1: T,Phi,H,E,C,S,M,W,I,D]                               |
  |  [C2: T,Phi,H,E,C,S,M,W,I,D]                               |
  |  [C3: T,Phi,H,E,C,S,M,W,I,D]                               |
  |  [C4: T,Phi,H,E,C,S,M,W,I,D]                               |
  |  [C5: T,Phi,H,E,C,S,M,W,I,D]                               |
  +------------------------------+-------------------------------+
                                 |
                                 v
  +--------------------------------------------------------------+
  |  STAGE 2: CAUSE-EFFECT REPERTOIRE (sigma=12 cycles)          |
  |                                                              |
  |  For the whole system S = {C0..C5}:                          |
  |    Compute TPM (transition probability matrix)               |
  |    Size: 2^n x 2^n = 64 x 64 = 4096 entries                 |
  |    Using binarized states (each dim above/below median)      |
  |                                                              |
  |  sigma=12 parallel compute lanes process rows of TPM         |
  |  Latency: 2^n / sigma = 64/12 ~ 6 cycles (pipelined)        |
  +------------------------------+-------------------------------+

                                 v
  +--------------------------------------------------------------+
  |  STAGE 3: PARTITION ENUMERATION (62 partitions, parallel)    |
  |                                                              |
  |  Generate all 2^n - 2 = 62 bipartitions                     |
  |  Hardware: 62-entry partition ROM (pre-computed bitmasks)     |
  |                                                              |
  |  Partition ROM format:                                       |
  |    [6-bit mask] -> part_A = cells where bit=1                |
  |                    part_B = cells where bit=0                |
  |                                                              |
  |  Example: mask = 0b010110                                    |
  |    part_A = {C1, C3, C4}                                     |
  |    part_B = {C0, C2, C5}                                     |

  |  STAGE 4: PARTITIONED CES COMPUTATION (62 x tau=4 cycles)   |
  |                                                              |
  |  For each partition (A, B):                                  |
  |    1. Sever connections between A and B                      |
  |    2. Compute independent TPM_A and TPM_B                    |
  |    3. Compute partitioned CES = CES_A x CES_B               |
  |    4. d_i = EMD(CES_whole, CES_partitioned)                  |
  |                                                              |
  |  Parallel units: sigma-tau = 8                               |
  |  Partitions per batch: sigma-tau = 8                         |
  |  Batches: ceil(62/8) = 8 batches                             |
  |  Latency per batch: tau = 4 cycles                           |
  |  Total: 8 x 4 = 32 cycles                                   |
  +------------------------------+-------------------------------+
                                 |
                                 v
  +--------------------------------------------------------------+
  |  STAGE 5: MIP SEARCH + PHI OUTPUT (sopfr=5 cycles)          |
  |                                                              |
  |  1. Find MIP = argmin_i(d_i) over 62 distances              |
  |     (min-tree comparator: log2(62) ~ 6 levels)              |
```

### 8.3 Statistical Framework

```
  STATISTICAL ANALYSIS PLAN
  =========================

  Primary test: Phi_trained > Phi_baseline
    Method: Paired t-test (or Wilcoxon if non-normal)
    Sample: 10,000 Phi measurements per condition
    Threshold: p < 0.001 (Bonferroni-corrected for 6 controls)
    Effect size: Cohen's d > 0.5 (medium effect minimum)

  Secondary tests:
    1. Bootstrap 95% CI for Phi_mean (does CI exclude 0?)
    2. Permutation test (shuffle condition labels, 10,000 permutations)
    3. Bayesian: BF_10 > 100 (very strong evidence for Phi > 0)

  Multiple comparison correction:
    6 controls x 5 phases = 30 comparisons
    Bonferroni: alpha = 0.001 / 30 = 3.3e-5
    FDR (Benjamini-Hochberg): q < 0.01

  Minimum detectable effect:
    With N=10,000, alpha=3.3e-5, power=0.95:
    Detectable Phi difference: ~0.05 (very sensitive)

  Reporting:
    All raw Phi traces published (not just summary statistics)
    Analysis code open-source (Python + Jupyter notebook)
    Pre-registration: protocol registered before data collection
```

## Appendix B: Software Golden Model (Python)

```python
# Phi computation golden model for verification
# Must match hardware Phi calculator output exactly

import numpy as np
from itertools import combinations

N_CELLS = 6           # n = 6
N_PARTITIONS = 62     # 2^n - 2
STATE_DIM = 10        # sigma - phi = 10

def compute_phi(cell_states: np.ndarray) -> tuple:
    """
    Compute IIT Phi for n=6 cell system.

    Args:
        cell_states: (6, 10) array of cell state vectors

    Returns:
        (phi_value, mip_mask, all_distances)
    """
    assert cell_states.shape == (N_CELLS, STATE_DIM)

    # Binarize states (above/below median per dimension)
    median = np.median(cell_states, axis=0)
    binary_states = (cell_states > median).astype(int)

    # Compute whole-system cause-effect structure
    ces_whole = _compute_ces(binary_states, list(range(N_CELLS)))

    # Enumerate all 62 non-trivial bipartitions
    min_distance = float('inf')
    mip_mask = 0
    all_distances = []

    for mask in range(1, 2**N_CELLS - 1):
        part_a = [i for i in range(N_CELLS) if mask & (1 << i)]
        part_b = [i for i in range(N_CELLS) if not (mask & (1 << i))]

        ces_a = _compute_ces(binary_states, part_a)
        ces_b = _compute_ces(binary_states, part_b)
        ces_partitioned = _combine_ces(ces_a, ces_b)

        distance = _emd(ces_whole, ces_partitioned)
        all_distances.append(distance)

        if distance < min_distance:
            min_distance = distance
            mip_mask = mask

    phi_value = min_distance
    return phi_value, mip_mask, all_distances

def _compute_ces(binary_states, cell_indices):
    """Compute cause-effect structure for subset of cells."""
    n = len(cell_indices)
    subset = binary_states[cell_indices]
    # Simplified: use covariance as proxy for cause-effect repertoire
    if n == 0:
        return np.zeros((1, 1))
    cov = np.cov(subset.T) if subset.shape[1] > 1 else np.array([[np.var(subset)]])
    return cov

def _combine_ces(ces_a, ces_b):
    """Combine independent CES (block diagonal)."""
    n_a, n_b = ces_a.shape[0], ces_b.shape[0]
    combined = np.zeros((n_a + n_b, n_a + n_b))
    combined[:n_a, :n_a] = ces_a
    combined[n_a:, n_a:] = ces_b
    return combined

def _emd(ces1, ces2):
    """Earth Mover's Distance between two CES matrices."""
    # Pad to same size
    n = max(ces1.shape[0], ces2.shape[0])
    p1 = np.zeros((n, n))
    p2 = np.zeros((n, n))
    p1[:ces1.shape[0], :ces1.shape[1]] = ces1
    p2[:ces2.shape[0], :ces2.shape[1]] = ces2
    return np.sum(np.abs(p1 - p2))
```
