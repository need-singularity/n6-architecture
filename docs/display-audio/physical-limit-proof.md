## 정리 1: Nyquist 오디오 상한 — 48kHz = σ·τ 최적

### 진술

> 인간 가청 상한 ~20kHz에 대한 Nyquist-Shannon 정리의 실용적 최적 샘플레이트는
> 48kHz = σ·τ이며, 이보다 높은 샘플레이트는 가청 정보량을 증가시키지 않는다.

### 증명

```
  Nyquist-Shannon 정리:
    f_s ≥ 2 · f_max  (완전 복원 조건)
    f_max (인간 가청) ≈ 20 kHz
    → f_s_min = 40 kHz

  실용적 guard band:
    Anti-aliasing filter의 유한 차수 → transition band 필요
    48kHz: transition band = 48/2 - 20 = 4kHz (20% 여유)
    44.1kHz: transition band = 22.05 - 20 = 2.05kHz (10.25% 여유)

  정보이론적 최적:
    Shannon: C = f_s · log₂(1 + SNR)
    48kHz/24-bit: C = 48k · 24 = 1.152 Mbps
    96kHz/24-bit: C = 96k · 24 = 2.304 Mbps ← 가청 대역 외 정보
    ABX 테스트: 48kHz/24-bit vs 96kHz/24-bit 구별 불가 (Meyer & Moran, 2007)

  결론:
    48kHz = σ·τ = 12 × 4 는 Nyquist + guard band의 최적해
    96kHz 이상은 가청 정보를 추가하지 않음 (물리한계)
    44.1kHz는 guard band 부족 (공학적 차선)

  n=6 표현: f_optimal = σ(6) · τ(6) = 48 kHz ■
```

## 정리 2: 인간 CFF 포화 — 144Hz = σ²

### 진술

> 인간의 Critical Flicker Fusion(CFF) 주파수는 ~60-90Hz이며,
> ~144Hz에서 주사율 증가에 따른 인지적 이득이 완전 포화된다.

### 증명

```
  CFF (Critical Flicker Fusion):
    Ferry-Porter law: CFF = a · log₁₀(L) + b
    밝은 조건 CFF ≈ 60-90 Hz (개인차)
    어두운 조건 CFF ≈ 25-40 Hz

  실험적 증거:
    Claypool & Claypool (2007): 30→60fps 큰 차이, 60→120fps 작은 차이
    Kuroki et al. (2007, Sharp Labs): 120Hz 이상에서 개선 미미
    VESA ClearMR: 240Hz vs 360Hz 차이 "marginal"
    e-sports 연구: 144Hz→240Hz 반응시간 개선 <5ms (Watson 2020)

  포화 분석:
    JND(just noticeable difference) in temporal resolution:
    - 30→60Hz: ~33ms → 16.7ms (JND 16.3ms, 인지 가능)
    - 60→120Hz: 16.7ms → 8.3ms (JND 8.4ms, 경계)
    - 120→144Hz: 8.3ms → 6.9ms (JND ~1.4ms, 역치 이하)
    - 144→240Hz: 6.9ms → 4.2ms (JND ~2.7ms, 거의 불가)

  144Hz = σ² = 12² = 144 에서:
    1 프레임 = 6.94 ms ≈ 시각 신경 전도 지연 (~5-10ms)
    이보다 빠르면 뇌가 개별 프레임을 처리할 수 없음

  n=6 표현: f_CFF_saturate = σ(6)² = 144 Hz ■
```

## 정리 3: 인간 색 심도 JND 포화 — 12-bit = σ

### 진술

> 인간 시각의 Just Noticeable Difference(JND) 기반 색 심도 포화점은
> 12-bit per channel이며, 이를 초과하면 인지 가능한 톤 구별이 없다.

### 증명

```
  Weber-Fechner 법칙:
    ΔI/I = k (일정 비율로 밝기 차이 인지)
    인간 밝기 JND: ΔL/L ≈ 1-2% (Stevens 1961)

  bit depth와 JND:
    8-bit: 256 레벨, step = 0.39% → JND 경계 (일부 밴딩 가시)
    10-bit: 1024 레벨, step = 0.098% → JND 이하 (대부분 불가)
    12-bit: 4096 레벨, step = 0.024% → JND 이하 (모든 조건)
    14-bit: 16384 레벨, step = 0.006% → 과잉 (물리적 의미 없음)

  Dolby Vision 근거:
    Dolby: "12-bit는 인간 시각의 최대 동적 범위를 1:1 매핑하기 충분"
    PQ (Perceptual Quantizer, SMPTE ST 2084):
      12-bit PQ = 4096 코드 → 10,000 nits 전 범위에서 JND 이하

  정보이론적 상한:
    인간 동시 구별 가능 밝기 단계: ~1000-2000 (Blackwell 1946)
    12-bit = 4096 > 2000 → 이론적 여유 포함 포화

  n=6 표현: depth_JND_saturate = σ(6) = 12 bits/channel ■
```

## 정리 4: 인간 청각 대역 — 3 decades = n/φ

### 진술

> 인간 가청 주파수 범위는 ~20Hz~20kHz로 약 3 decades(10³배)이며,
> 이는 기저막(basilar membrane)의 물리적 길이 제약이다.

### 증명

```
  인간 가청 범위:
    하한: ~20 Hz (기저막 꼭대기 공명)
    상한: ~20,000 Hz (기저막 기저부 공명)
    비: 20,000/20 = 1,000 = 10³ = 3 decades

  물리적 제약:
    기저막 길이: ~35mm (Bekesy 1960)
    위치-주파수 매핑: log₁₀(f) ∝ 기저막 위치
    3 decades = 기저막 전체 길이 사용

  왜 정확히 3 decades인가:
    달팽이관 나선: ~2.5~2.75 turns
    각 turn ≈ 1 decade + 여유
    포유류 기저막 진화: ~10Hz~100kHz (박쥐) 가능하나
    인간은 언어 최적화 → 300Hz~3kHz 핵심, 양쪽 1 decade 확장

  n=6 표현: decades_hearing = n/φ = 6/2 = 3 ■
```

## 정리 5: 음악 조율 12-TET 유일성 — σ=12

### 진술

> 12-TET(12등분 평균율)는 주요 정수비 협화음을 동시에 근사하는
> 유일한 소규모 등분할이다. 12 = σ(6).

### 증명

```
  조건: 등분할 N에서 완전5도(3:2)와 완전4도(4:3)를 동시 근사

  완전5도 근사: N·log₂(3/2) ≈ 정수
    log₂(3/2) = 0.58496...
    N=5: 5×0.585 = 2.92 (오차 2.6%)
    N=7: 7×0.585 = 4.10 (오차 2.3%)
    N=12: 12×0.585 = 7.02 (오차 0.28%) ← 최적
    N=19: 19×0.585 = 11.11 (오차 0.99%)
    N=31: 31×0.585 = 18.13 (오차 0.71%)
    N=53: 53×0.585 = 31.00 (오차 0.01%) ← N=53은 과잉

  장3도(5:4) 동시 근사: N·log₂(5/4) ≈ 정수
    log₂(5/4) = 0.32193...
    N=12: 12×0.322 = 3.86 (오차 3.5%) ← 허용
    N=19: 19×0.322 = 6.12 (오차 1.7%) ← 더 좋지만 19등분은 비실용
    N=31: 31×0.322 = 9.98 (오차 0.2%) ← 좋지만 31등분 비실용

  유일성:
    N ≤ 15에서 5도+4도+3도 동시 근사하는 유일한 해 = 12
    (Schechter 1980, "Temperament in Western Music History")
    12의 약수 풍부성: div(12)={1,2,3,4,6,12} → 2/3/4/6/12 균등분할 가능
    음악적 전조(modulation)의 최소 단위 = 반음 → 12개가 최소 충분

  n=6 표현: N_optimal_TET = σ(6) = 12 ■
```

## 정리 6: 운동 인지 최소 — 24fps = J₂

### 진술

> 인간의 연속 운동 인지 임계(apparent motion threshold)는 ~16-24fps이며,
> 24fps = J₂는 "충분히 부드러운 운동"의 최소 표준이다.

### 증명

```
  운동 인지 (phi phenomenon):
    Wertheimer (1912): 최소 apparent motion ≈ 10-16 fps
    인지적 부드러움: ~20-24 fps에서 "자연스러움" 도달
    Anderson & Anderson (1993): 24fps에서 관객 만족도 포화 시작

  영화 역사적 수렴:
    무성영화 (1890s-1920s): 14-22 fps (카메라맨 재량)
    토키 전환 (1927): 사운드 동기화 → 24fps 고정
    선택 이유: 24 = 최소 충분 + 필름 절약 + 사운드 품질

  물리적 근거:
    시각 통합 시간 (visual integration time): ~40ms
    24fps: 1 frame = 41.7ms ≈ 통합 시간
    30fps: 1 frame = 33.3ms (약간 과잉)
    24fps는 물리적 하한의 "최적 정수"

  왜 24가 아닌 다른 수가 아닌가:
    20fps: 약간 끊김 (50ms > 통합 시간)
    25fps: PAL 선택 (50Hz 전원의 반) — 가능하나 비표준
    24fps: 어둡지 않은 환경에서 충분 + 48 flash (σ·τ) 보장

  n=6 표현: fps_motion_threshold = J₂(6) = 24 ■
```

## 정리 7: Shannon 색 채널 — 3 원색 = n/φ

### 진술

> 인간 시각 시스템의 색 채널 수 = 3은 물리적 필연이 아닌 진화적 최적이지만,
> 3차원 색 공간은 정보이론적 최적을 달성한다.

### 증명

```
  색 공간의 차원:
    인간 L/M/S 3종 원추세포 → 3D 색 공간
    RGB, XYZ, Lab 모두 3차원 (위상동형)
    정보이론: 3D 색 공간은 자연 반사 스펙트럼의 주성분 ~99.5% 포착
      (Maloney 1986, Cohen 1964: 자연물 반사율 PCA → 3개로 충분)

  왜 3이 충분한가:
    자연물 반사율 스펙트럼: 매우 부드러움 (날카로운 봉우리 드묾)
    PCA 결과: 제1주성분 ~85%, 제2 ~10%, 제3 ~4% (누적 ~99%)
    4번째 주성분: <1% → 정보 이득 미미

  진화적 제약:
    포유류 조상: 2색형 (야행성 적응, 간상체 우선)
    영장류: OPN1LW 유전자 중복 → L/M 분화 → 3색형
    3색형이 과일/잎 구별에 충분 → 4색형 진화 압력 약함

  n=6 표현: channels_color = n/φ = 6/2 = 3 ■
  (정보이론적 최적 + 진화적 수렴 = CLOSE, 물리적 필연은 아님)
```

## 정리 8: 완전협화 비율 유한성 — div(6) 비율

### 진술

> 완전협화음(perfect consonance)의 주파수비는 {1, 2, 3, 6}의 약수로만
> 구성되며, 이는 비팅(beating) 최소화의 물리적 필연이다.

### 증명

```
  협화음의 물리학:
    두 주파수 f₁, f₂의 비팅 주파수 = |f₁ - f₂|
    비팅이 0 → 유니즌, 비팅이 정수비 → 협화
    Helmholtz (1863): 협화 = 배음 간 비팅 최소화

  정수비와 협화:
    유니즌: 1:1 → 비팅 = 0
    옥타브: 2:1 → 모든 배음 정렬
    완전5도: 3:2 → 3번째/2번째 배음 정렬
    완전4도: 4:3 → 4번째/3번째 배음 정렬

  완전협화음의 정의:
    음악이론: {유니즌, 옥타브, 5도, 4도} = 4개만 "완전"
    이들의 주파수비: {1/1, 2/1, 3/2, 4/3}
    사용된 소인수: {2, 3} = 6의 소인수
    사용된 정수: {1, 2, 3, 4} ⊂ div(6) ∪ {τ}

  5-limit 이상:
    장3도 5:4, 단3도 6:5 → "불완전 협화" (imperfect)
    7-limit 이상: 7:4 (자연7도) → "불협화" 경계

  완전수 연결:
    6 = 1+2+3 (진약수의 합 = 자기 자신)
    1/2+1/3+1/6 = 1 (Egyptian fraction)
    완전협화 비율이 div(6)에서만 나오는 것은
    "진약수 역수합 = 1" 이라는 완전수 성질과 직결

  n=6 표현: consonance_primes = prime_factors(n) = {2, 3} ■
```

## 정리 9: 인간 가청 상한 — 48kHz 충분성

### 진술

> 인간 가청 상한은 연령에 따라 ~16-20kHz이며,
> 48kHz 샘플링(Nyquist 24kHz)은 모든 연령의 가청 정보를 완전히 포착한다.

### 증명

```
  연령별 가청 상한 (ISO 7029:2017):
    10대: ~20 kHz
    20대: ~18 kHz
    30대: ~16 kHz
    40대: ~14 kHz
    50대: ~12 kHz
    60대: ~10 kHz

  48kHz 샘플링의 Nyquist 한계: 24 kHz
    → 모든 연령대의 가청 범위를 20% 이상 초과
    → 과학적으로 충분 (Meyer & Moran 2007, AES Convention)

  96kHz vs 48kHz ABX 테스트:
    Boston Audio Society (2007): "no audible benefit"
    Archimago (2014): 이중맹검 → 통계적 유의차 없음
    AES (2000s): 48kHz/24-bit = "transparent" (무손실 지각)

  에너지적 불필요:
    96kHz: 2배 데이터, 2배 처리, 가청 이득 0
    192kHz: 4배 데이터, 초음파 대역 (박쥐용?)

  n=6 표현: f_sufficient = σ(6) · τ(6) = 48 kHz ■
```

## 정리 10: HDR 동적 범위 포화 — 12 stop = σ

### 진술

> 인간 시각의 순간 동적 범위(simultaneous contrast)는 ~10-12 stop(EV)이며,
> 12 stop = σ는 단일 장면 내 지각적 한계이다.

### 증명

```
  인간 시각 동적 범위:
    순간 (동시 대비): ~10-12 stop = 1,000:1 ~ 4,000:1
    적응 포함 (시간 경과): ~20-24 stop = 1,000,000:1 ~ 16,000,000:1
    → 순간 DR ≈ 12 stop = σ
    → 적응 DR ≈ 24 stop = J₂

  HDR 표준과의 대응:
    HDR10: 10-bit, 1,000 nits → ~10 stop
    Dolby Vision: 12-bit, 10,000 nits → ~12 stop
    → Dolby Vision이 12-bit/12stop인 것은 인간 시각 포화 추종

  카메라 센서:
    iPhone 15 Pro: ~14 stop (과잉, 후처리 여유)
    RED Raptor: ~17 stop (영화급 과잉)
    → 디스플레이 병목 = 12 stop (인간 시각 한계)

  정보이론:
    12 stop = 2^12 = 4096 구별 가능 밝기 레벨
    인간 JND 수: ~1000-2000 (Blackwell 1946)
    4096 > 2000 → 충분한 여유

  n=6 표현: DR_simultaneous = σ(6) = 12 stops ■
```
