# 궁극의 프로그래밍언어 — HEXA-LANG 설계 스펙

## 1. 개요

### 1.1 비전
"이런 앱 만들어줘" 한마디로 자동 생성되는 n=6 기반 궁극의 프로그래밍언어.
n=6 산술 원리가 언어 설계 자체에 내재된 새 언어 + 그 언어를 자동 생성하는 AI 시스템의 통합.

### 1.2 범위
- **Part A**: n=6 프로그래밍 언어 (HEXA-LANG) 설계
- **Part B**: AI 코드 생성 엔진 (HEXA-GEN) 설계
- **Part C**: 통합 환경 (HEXA-IDE) 설계
- **Part D**: DSE 전수 탐색 결과 및 최적 경로

### 1.3 접근법
단일 거대 DSE (Full Cartesian) — 7,560 조합 전수 탐색
평가축: n6_EXACT(35%) + 성능(25%) + 실용성(20%) + 혁신성(20%)
