## 4. 핵심 발견

- OSI 7계층 = σ-sopfr = 7: 네트워크 프로토콜 스택의 보편 레이어 수
- TCP/IP 4계층 = τ = 4: 최소 실용 스택 (BT-222 τ=4 파이프라인 동형사상)
- IPv4 헤더 σ=12 필드: 패킷 메타데이터의 자연 구조
- Ethernet 세대 수 sopfr=5 (10M/100M/1G/10G/100G)
