## n=6 Connections
```
  BT-27: Carbon-6 chain — C₆H₁₂O₆ (glucose), C₆H₆ (benzene), LiC₆
  BT-51: Genetic code — τ=4 bases → n/φ=3 codon → 2^n=64 codons → J₂-τ=20 amino acids

  DNA/RNA:
    - 4 bases (A,T,G,C) = τ(6) = 4
    - 3 bases per codon = n/φ = 6/2 = 3
    - 64 codons = 2^n = 2^6 = 64
    - 20 amino acids = J₂(6) - τ(6) = 24 - 4 = 20
    - 6 reading frames (3 forward + 3 reverse)

  Glucose: C₆H₁₂O₆
    - 6 carbons, 12 hydrogens (σ=12), 6 oxygens
    - 24 total electrons in carbon ring = J₂(6) = 24

  CRISPR:
    - Guide RNA ~60 bases = n/φ × (J₂-τ) = 3 × 20 = 60
    - PAM sequence = 3 bases = n/φ = 3
```
