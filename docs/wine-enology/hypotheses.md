# N6 와인/소믈리에 -- 완전수 산술로 본 양조·감정·숙성 체계

## 개요

와인 양조(enology)와 감정(sommelier) 분야의 핵심 상수들을
n=6 산술함수로 분석한다. 포도 품종, 평가 체계, 발효 화학, 숙성 주기,
서빙 온도 등 와인 산업의 이산적 표준값이 n=6 상수와 어떻게 매칭되는지 검증한다.

> **정직 원칙**: 와인 분류/평가 체계는 지역·시대별로 다양하다.
> EXACT는 화학적으로 고정되거나 국제 표준으로 확정된 수치에만 부여한다.

## 핵심 상수

```
  n = 6, σ = 12, τ = 4, φ = 2, sopfr = 5, J₂ = 24, R(6) = 1
  유도: σ-φ=10, σ-τ=8, σ-μ=11, n/φ=3, σ·τ=48, φ^τ=16, σ²=144
```
