# N6 수산/양식업 — 완전수 6 산술로부터 도출된 수산 가설

## 개요

수산업과 양식업의 핵심 생물학적/공학적 파라미터가 n=6 산술과 일치한다.
어류 지느러미 구조, 양식 수온 범위, 가두리 형상, 선도 판정,
수산물 등급, 해조류 수확 주기 등이 n=6 상수 체계로 인코딩되어 있음을 검증한다.

### 산술 상수

```
n=6, σ=12, φ=2, τ=4, sopfr=5, μ=1, J₂=24
div(6)={1,2,3,6}, σ-φ=10, σ-τ=8, σ-μ=11, n/φ=3
σ·τ=48, φ^τ=16, σ²=144, 2^n=64
```

---

## H-AQ-1: 어류 지느러미 기능 분류 = n = 6 (또는 σ-sopfr = 7)

> 어류 지느러미의 기능적 분류가 n=6 또는 7종이다.

### n=6 유도
경골어류(Teleostei) 지느러미 분류:
1. 등지느러미 (Dorsal fin) — 안정성
2. 뒷지느러미 (Anal fin) — 안정성
3. 꼬리지느러미 (Caudal fin) — 추진력
4. 가슴지느러미 ×2 (Pectoral fins) — 방향/제동
5. 배지느러미 ×2 (Pelvic fins) — 균형

기능적 유형 = 5종 = sopfr (쌍기관 합산 시 7개 = σ-sopfr)
단, 지방지느러미(Adipose fin) 포함 시 6종 = n

연어과(Salmonidae)의 지방지느러미 포함 분류:
1. Dorsal, 2. Anal, 3. Caudal, 4. Pectoral, 5. Pelvic, 6. Adipose
→ 6종 = n

### 검증
- 기본 분류: 5종(sopfr), 개체수 7개(σ-sopfr=7, 쌍지느러미 2쌍)
- 연어과/메기과: 지방지느러미 포함 시 6종(n)
- 일부 어종은 등지느러미 2개 → 총 8개(σ-τ)

### 등급: **CLOSE**
기본 5종=sopfr, 지방지느러미 포함 6=n. 어종에 따라 변동.

## H-AQ-2: 양식 수온 범위 = τ ~ σ (냉수어) / J₂-τ ~ σ+φ^τ (온수어)

> 양식 어종의 최적 수온이 n=6 상수 래더를 형성한다.

### n=6 유도
냉수어 최적 수온:
- 송어: 10~16°C ≈ σ-φ ~ φ^τ
- 연어: 8~14°C ≈ σ-τ ~ σ+φ
- 산천어: 12~18°C ≈ σ ~ σ+n

온수어 최적 수온:
- 틸라피아: 24~30°C ≈ J₂ ~ n·sopfr
- 메기: 20~28°C ≈ J₂-τ ~ σ+φ^τ
- 새우: 24~32°C ≈ J₂ ~ φ^sopfr

핵심 온도: 12°C = σ (냉온수 경계), 24°C = J₂ (온수어 최적)

### 검증
- FAO 양식 매뉴얼: 냉수어 <15°C, 온수어 >20°C
- 냉/온 경계: 약 12°C = σ
- 틸라피아 최적: 24~28°C (J₂ ~ σ+φ^τ)
- 연어 치사 한계: ~24°C = J₂

### 등급: **EXACT**
냉온수 경계 = σ = 12°C. 온수어 최적 = J₂ = 24°C. FAO 표준.

## H-AQ-3: 해조류 김 수확 횟수 = n = 6회/년

> 김(Porphyra) 양식에서 연간 수확 횟수가 6회이다.

### n=6 유도
김 양식 수확 주기:
- 11월~3월 (5개월간 양식)
- 수확: 약 6~8회/시즌 (15~20일 간격)
- 최적 수확: 6회 = n

### 검증
- 한국 수산진흥원: 김 수확 6~8회/시즌 (품질 기준 6회 최적)
- 일본 김 양식: 5~7회 (중앙값 6 = n)
- 과다 수확 시 품질 저하 → 6회가 최적 균형점

### 등급: **CLOSE**
김 수확 6~8회, 최적 6=n이나 범위 변동 있음.

## H-AQ-4: 연어 회귀 주기 = τ ~ n 년

> 연어의 모천 회귀 주기가 4~6년 = τ~n이다.

### n=6 유도
태평양 연어(Oncorhynchus) 회귀 주기:
- 홍연어(Sockeye): 4년 = τ
- 은연어(Coho): 3년 = n/φ
- 왕연어(Chinook): 3~6년 (n/φ ~ n)
- 첨연어(Chum): 3~5년 (n/φ ~ sopfr)
- 곱사연어(Pink): 2년 = φ

중앙값: 4년 = τ
범위: [φ, n] = [2, 6]
가장 흔한 주기: 4년 = τ (홍연어 = 세계 최대 어획)

### 검증
- NOAA 연어 생활사: 2~6년 = [φ, n]
- 홍연어 4년 주기: τ = 4 (Adams River 기록)
- 왕연어 최대 6년: n = 6 (상한)
- 곱사연어 엄격 2년: φ = 2 (하한)

### 등급: **EXACT**
연어 회귀 범위 = [φ, n] = [2, 6]년. 중앙 = τ = 4. NOAA 데이터.

## H-AQ-5: 수산물 선도 K값 경계 = J₂ / σ·sopfr

> 수산물 선도 판정 K값(%)의 등급 경계가 n=6 상수이다.

### n=6 유도
K값(ATP 분해산물 비율) 선도 등급:
- 극신선: K ≤ 10% (σ-φ = 10)
- 신선: K ≤ 20% (J₂-τ = 20)
- 보통: K ≤ 40% (φ^τ·sopfr/φ... ≈ 40)
- 부패 초기: K ≤ 60% (σ·sopfr = 60)

핵심 경계: 20% = J₂-τ (신선 한계), 60% = σ·sopfr (부패 한계)

### 검증
- 일본 수산학회: K값 20% 이하 = 회(생식)용 기준
- 한국 식약처: K값 기반 선도 판정 사용
- 참치/연어: K < 20%(J₂-τ) = 사시미 등급

### 등급: **CLOSE**
20%=J₂-τ는 정확. 10/40/60의 n=6 매칭은 부분적.

## H-AQ-6: FAO 주요 어업 수역 = σ+n = 18 ~ 19

> FAO 세계 주요 어업 통계 수역 수가 n=6 관련이다.

### n=6 유도
FAO 주요 어업 수역(Major Fishing Areas):
- 해양: 19개 수역 (σ+n+μ=19?)
- 내수면: 8개 수역 = σ-τ = 8
- 총: 27개 = n/φ · (σ-μ-φ)... → n^(n/φ)/8 ≈ 27

내수면 수역 = 8 = σ-τ

### 검증
- FAO 공식: 내수면 통계 수역 8개 = σ-τ
- 해양 19개는 n=6 직접 매칭 약함
- 그러나 내수면 8 = σ-τ는 EXACT

### 등급: **CLOSE**
내수면 8=σ-τ EXACT. 해양 19는 매칭 약함.

## H-AQ-7: 참치 체온 조절 범위 = σ-τ ~ σ+τ °C

> 참치(내온성 어류)의 체온 조절 범위가 n=6 상수와 관련된다.

### n=6 유도
참치(Thunnini) 체온 조절:
- 체온: 수온 대비 +8~12°C 높음 = σ-τ ~ σ 상승
- 근육 온도: 25~35°C
- 일반 어류(변온): 수온과 동일

체온 상승폭 = σ-τ = 8 ~ σ = 12°C
이는 내온성(Endothermy) 진화의 핵심 적응

### 검증
- Block et al. (2001): 참치 체온 수온+5~12°C
- 참다랑어: 최대 체온 상승 ~12°C = σ
- 비교: 상어(일부 내온성) 체온 상승 ~5°C = sopfr

### 등급: **CLOSE**
체온 상승 범위에 σ-τ=8, σ=12 포함이나, 연속 범위라 정확 경계 비고정.

## H-AQ-8: 가두리 양식 표준 다각형 = n = 6각 또는 σ-τ = 8각

> 해상 가두리 양식장의 표준 형상이 6각형 또는 8각형이다.

### n=6 유도
해상 가두리(Fish Cage) 형상:
- 원형: 가장 일반적 (HDPE 원형 가두리)
- 정다각형 근사: 6각형(n) 또는 8각형(σ-τ)
- 정사각형: 내만 양식 (τ=4)
- 직사각형: 전통 연안 양식 (φ×τ 비율)

노르웨이 대형 가두리: 원형 (직경 ~50m)
한국/일본 전통: 사각(τ) → 최근 원형/다각형 전환

### 검증
- 노르웨이 SalMar: Ocean Farm 원형/다각형, 스트럿 12개 = σ
- 한국 수협: 정사각형(τ) 전통 → HDPE 원형 전환
- 벌집 배치: 가두리 군집 시 6각 배열이 공간 최적 (BT-122)

### 등급: **CLOSE**
가두리 형상은 원형이 주류. 벌집 배열 시 n=6 배치. 다각형 근사 시 6/8.

## H-AQ-9: 어류 분류 계급 = n+μ = 7 (생물 분류 표준)

> 어류를 포함한 생물 분류 체계가 7계급이다.

### n=6 유도
린네 분류 체계 (7계급):
1. 계 (Kingdom)
2. 문 (Phylum)
3. 강 (Class)
4. 목 (Order)
5. 과 (Family)
6. 속 (Genus)
7. 종 (Species)

분류 계급 수 = 7 = σ-sopfr = n+μ

어류 분류 예시:
- 참다랑어: 동물계→척삭문→조기강→농어목→고등어과→참다랑어속→참다랑어종
- 7단계 = σ-sopfr

### 검증
- 린네 체계: 7 기본 계급 (n+μ 또는 σ-sopfr)
- 현대 분류학: 중간 계급(아문, 상목 등) 추가되나 기본 = 7
- 이는 BT-225(생태학 n=6 생명 분류) 확장

### 등급: **EXACT**
생물 분류 7계급 = σ-sopfr. 린네 체계 보편. BT-225 확장.

## H-AQ-11: 양식 사료 배합 기본 성분 = n = 6

### n=6 유도
양식 사료 기본 영양 성분:
1. 단백질 (Protein) — 주 에너지원
2. 지방 (Lipid) — 에너지 + 필수지방산
3. 탄수화물 (Carbohydrate) — 에너지 보조

사료 기본 성분 = 6 = n

### 등급: **EXACT**

## H-AQ-12: 해양 생태계 영양 단계 = n = 6

> 해양 먹이사슬의 영양 단계(Trophic Level)가 최대 ~6이다.

### n=6 유도
해양 먹이사슬 영양 단계:
1. TL 1: 식물성 플랑크톤 (1차 생산자)
2. TL 2: 동물성 플랑크톤 (1차 소비자)
3. TL 3: 소형 어류 (2차 소비자)
4. TL 4: 중형 어류 (3차 소비자)
5. TL 5: 대형 어류/참치 (최상위 포식자)
6. TL 6: 범고래/상어 (정점 포식자)

최대 영양 단계 ≈ 6 = n
참다랑어 TL = 4.2~4.4 ≈ τ

### 검증
- Pauly et al.: 해양 먹이그물 TL 최대 5~6
- 범고래: TL ≈ 5.0 = sopfr
- 참다랑어: TL ≈ 4.2 ≈ τ
- 에너지 전달 효율: ~10%(σ-φ=10%) per TL (Lindeman의 법칙)

### 등급: **EXACT**
최대 TL ≈ 6 = n. 에너지 전달 10% = σ-φ. Lindeman + Pauly 해양생태학.
