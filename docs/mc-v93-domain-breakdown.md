## 해석

- 도메인별 독립 Monte Carlo는 노드 수가 다른 영역 간 비교를 가능하게 한다.
- z >= 5: n=6 특이성 확정 (극도 유의)
- 3 <= z < 5: 통계적으로 유의
- z < 3: 표본 과소 또는 도메인 내 n=6 정렬이 부족 → 보강 대상
