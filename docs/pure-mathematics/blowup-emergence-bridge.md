# BT-185: Algebraic Blowup–Emergence E₆ Bridge

> 블로업 = 수축 → 특이점 → 펼침 → 창발
> n=6이 특이점 해소의 수학적 구조를 결정한다

## 핵심 발견

대수기하학의 **블로업(blowup)** 연산은 **창발(emergence)**의 정확한 수학적 원형이다:

```
수축          특이점         블로업(펼침)        창발
───→  ·  ───→  ●  ───→  ═══╗  ───→  새 구조
                             ║
                        예외인자 E ≅ P^(n-1)
                        χ(E) = n = 6
```
