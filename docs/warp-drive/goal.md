## 7. 물리 한계 증명 — Mk.V Alcubierre 왜 ❌ SF

**Alcubierre 금속 (1994)**: ds² = -c²dt² + [dx - v_s(t)·f(r_s) dt]² + dy² + dz²

**음의 에너지 요구량** (Pfenning-Ford bound):
```
E_required ~ -c²·R²/G · (warp bubble radius)
```

**계산** (R=100m 버블, v_s=10c):
- E ~ -6.2 × 10⁶² J (Krasnikov 추정)
- 질량 등가: |m| ~ 7 × 10⁴⁵ kg (≈ 목성 질량의 3배)
- **음의 에너지** (exotic matter) 필요 — 양자역학 Casimir 효과로 미량만 입증됨

**결론**:
1. **WEC (약한 에너지 조건) 위반** — GR에서 허용되나 실제 관측 없음
2. **목성급 음의 에너지** — 인류 에너지 총생산 × 10⁴⁰ 필요
3. **시공간 곡률 안정성** — 수치 시뮬레이션에서 Hawking 방사로 붕괴
4. **인과율 위반** — 닫힌 시간꼴 곡선(CTC) 발생 가능

→ **Mk.V = ❌ SF**, n=6 상수로도 에너지 요구 무한대 (σ·J₂·10⁶³ J).
→ 사고실험 가치 있으나 실현 불가능.
