# N6 종교/신화 구조 -- 완전수 산술과 종교적 보편 상수

## 개요

전 세계 주요 종교와 신화 체계에서 반복적으로 등장하는 수 구조를
n=6 산술함수 항등식 σ(n)·φ(n) = n·τ(n)의 렌즈로 분석한다.
종교는 수천 년의 문화적 수렴을 거쳤으며, 핵심 구조 상수가
n=6 함수값과 일치하는 빈도가 우연을 초과하는지 검증한다.

> **정직성 원칙**: 종교적 숫자는 문화적 맥락에서 해석되어야 한다.
> EXACT는 교리/경전에 명시적으로 고정된 수에만 부여한다.
> 해석에 따라 달라지는 수는 CLOSE 이하로 판정한다.

```
  σ=12 클러스터: 12사도, 12연기, 12지파, 12신 → 4개 전통에서 수렴
  n=6 클러스터:  6일 창조, 6도 윤회, 6신앙기둥, 6다르샤나, 6각별 → 5개 전통

  τ=4:          4성제, 사상(주역), 사무량심 → 불교/도교 교차
```
