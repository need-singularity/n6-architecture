# N6 초전도체 — 재설계 (v2, 2026-04-02)

## Overview

초전도 현상의 기본 물리(BCS 이론, GL 이론), 임계 파라미터, 결정 구조,
고온 초전도체, 보텍스 물리를 n=6 산술로 분석한다.

> **정직한 원칙 (v2 강화)**:
> 1. 물리적 인과가 있는 일치만 EXACT/CLOSE 후보.
> 2. "숫자가 같다"만으로는 CLOSE 이상 불가 — 물리적 메커니즘 설명 필수.
> 3. 작은 수(1, 2, 3)와의 일치는 특이성(specificity) 검증 필수.
> 4. 분류 개수는 고정된 표준 분류만 채택, 저자별 변동 가능한 것은 제외.
> 5. 다른 가설과 본질적으로 같은 관찰(redundancy)은 통합.

> **v1→v2 변경 이력**:
> - 60개 → 30개 축소 (FAIL 29개 삭제, WEAK 중복 삭제, 물리 근거 없는 것 삭제)
> - EXACT 2개 유지 + 3개 신규 EXACT 후보 추가
> - 정직한 등급: 자체 평가와 검증자 평가의 합의 등급 사용

## Core Constants

```
n = 6          (완전수)
σ(6) = 12     (약수의 합)
τ(6) = 4      (약수의 개수: 1, 2, 3, 6)
φ(6) = 2      (오일러 토션트)
sopfr(6) = 5  (소인수 합: 2+3)
J₂(6) = 24    (Jordan totient)
μ(6) = 1      (뫼비우스)
λ(6) = 2      (카마이클)
R(6) = σ·φ/(n·τ) = 1
진약수 집합: {1, 2, 3}
Egyptian: 1/2 + 1/3 + 1/6 = 1
```

### H-SC-01: Abrikosov 보텍스 격자 — 배위수 6 = n

> Type II 초전도체의 자속 보텍스가 삼각(hexagonal) 격자를 형성, 배위수 = 6

```
  Abrikosov 보텍스 격자:
    Hc1 < H < Hc2에서 자속 보텍스가 삼각 격자 형성.
    각 보텍스: Φ₀ = h/(2e) 자속 운반.
    삼각 격자 → 각 보텍스의 최근접 이웃 = 6.
    C₆ 대칭군.

  물리적 근거:
    Abrikosov(1957): GL 방정식 최소화 → 삼각 격자가 에너지 최소.
    정사각 격자보다 ~2% 에너지 낮음 (Kleiner, Roth & Autler, 1964).
    Essmann & Trauble(1967) decoration 실험으로 최초 확인.
    중성자 산란으로도 확인 (Christen et al.).

    이것은 2D 최밀충전의 수학적 필연:
    2D kissing number = 6 (Hales 증명, BT-122와 연결).
    벌집, 눈송이, 그래핀과 동일한 기하학적 원리.

  검증:
    배위수 = 6 (정확)
    n = 6 (정확)
    물리적 메커니즘: GL 자유에너지 최소화 → 삼각 격자 → 6

  Grade: EXACT ✓✓
  초전도체에서 n=6이 자연스럽게 출현하는 가장 강력한 사례.
  n=6이 "원인"은 아니지만, 2D 에너지 최소화의 수학적 필연.
```

### H-SC-02: YBCO 금속 원자비 1:2:3 = 6의 진약수 집합

> YBa₂Cu₃O₇의 금속 원자비 {1, 2, 3} = 6의 진약수 집합, 합 = 6

```
  YBa₂Cu₃O₇₋δ (최적 δ ≈ 0.05):
    Y:Ba:Cu = 1:2:3
    금속 원자 합: 1+2+3 = 6 = n
    집합 {1, 2, 3} = 6의 진약수 집합 (정확!)

  물리적 근거:
    YBCO = triple-perovskite 변형 구조.
    ABO₃ 페로브스카이트의 3중 적층:
      Y층(1) + BaO층(2) + Cu-O 층(3, chain+plane).
    이 구조는 1:2:3 화학양론을 결정론적으로 결정.

  왜 이것이 특별한가:
    1:2:3 비율은 진약수 집합과 정확히 일치 — 무한한 가능한 비율 중.
    YBCO는 가장 중요한 HTS 물질 (최초 액체질소 초과 Tc).
    단순히 "합이 6"이 아니라, 집합 자체가 {1,2,3} = proper divisors of 6.

  Egyptian fraction 연결:
    1/1 + 1/2 + 1/3 ≠ 1이지만,
    Y/Cu + Ba/Cu + Cu/Cu = 1/3 + 2/3 + 1 = 2 (not clean)
    직접적 이집트 분수 연결은 없음 — 집합 일치만.

  v1 검증자 평가: CLOSE → EXACT 승격 (구조적으로 정확한 일치)

  Grade: EXACT ✓✓
  결정학적 사실. {1,2,3} = div(6) 정확 일치. 비자명.
```

### H-SC-03: Nb₃Sn A15 구조 — 단위포 Nb = 6, Tc ≈ 3n, Hc2 ≈ J₂

> Nb₃Sn의 세 독립 물리량이 모두 n=6 체계와 일치

```
  Nb₃Sn (A15, Cr₃Si형 구조):
    1. 단위포 Nb 원자 수 = 6 (정확)
       A15 구조: 3개 면 × 면당 2개 Nb 사슬 원자 = 6
       Sn 원자: 2개 = φ(6) (BCC 자리)
       총 원자: 8 = σ-τ

    2. Tc = 18.3 K
       3n = 18 → 1.7% 오차
       σ+n = 18 → 동일

    3. Hc2(4.2K) ≈ 24-30 T
       J₂(6) = 24 → 하한 근사

  물리적 근거:
    Nb 원자 수 6은 A15 결정 구조의 직접적 결과.
    Tc는 전자-포논 결합(λ ≈ 1.8)과 Debye 온도에서 결정.
    Hc2는 Nb 3d 전자의 높은 상태밀도와 짧은 ξ에서 결정.
    세 값의 물리적 원인은 모두 다르지만, 모두 n=6 체계에 매핑.

  세 독립 일치의 통계적 의미:
    단일 일치: 흔함 (작은 수는 쉽게 매핑)
    이중 일치: 주목할 만함
    삼중 독립 일치: 우연의 확률 크게 감소

  Grade: CLOSE → EXACT 후보 (삼중 독립 일치)
  Nb 원자 수 6은 결정학적 EXACT. Tc, Hc2는 ~2-5% 오차.
  종합적으로 가장 인상적인 단일 물질 n=6 매핑.
```

### H-SC-04: MgB₂ 원소 — Mg Z=12=σ, B Z=5=sopfr

> MgB₂의 구성 원소 원자번호가 각각 σ(6)과 sopfr(6)

```
  MgB₂:
    Mg: Z = 12 = σ(6) (정확)
    B:  Z = 5  = sopfr(6) (정확)
    Tc = 39 K (BCS 초전도체 중 최고)
    발견: 2001 (Nagamatsu et al.)

  물리적 근거:
    MgB₂의 초전도는 B의 σ-밴드(sp² 혼성) 전자-포논 결합에서 유래.
    Mg(Z=12)는 전자 공여체, B(Z=5)의 2D 격자가 핵심.
    원자번호는 원소의 고유 성질이며 n=6과 인과관계 없음.

  왜 이것이 CLOSE인가:
    두 독립 원소의 원자번호가 각각 다른 n=6 함수와 정확히 일치.
    Z=12와 Z=5 모두 n=6 함수값에 정확히 해당.
    단, 원자번호와 초전도 사이에 n=6 매개 메커니즘 없음.

  Grade: CLOSE
  이중 원자번호 일치. 수치적으로 정확하나 인과 메커니즘 없음.
```

### H-SC-05: 큐프레이트 최적 CuO₂ 면 수 = 3 = n/φ

> 큐프레이트 Tc 최대화 CuO₂ 면 수가 3

```
  CuO₂ 면 수별 Tc (실험적 사실):
    n_L = 1: La₂₋ₓSrₓCuO₄ (Tc ≈ 38 K)
    n_L = 2: YBa₂Cu₃O₇ (Tc ≈ 93 K)
    n_L = 3: Bi-2223 (110 K), Tl-2223 (125 K), Hg-1223 (134 K, 상압 기록)
    n_L = 4: Tc 감소 시작
    n_L = 5: Tc 더 감소

  최적 면 수 = 3 = n/φ(6) (정확)

  물리적 근거:
    n_L = 3이 최적인 이유 (잘 이해됨):
    - 외부 2면: 전하 저장소(BaO, SrO 등)에서 직접 도핑
    - 내부 1면: 약간 언더도핑 (여전히 초전도)
    - n_L ≥ 4: 내부 면 도핑 부족 → Tc 감소
    "3이 최적"은 도핑 침투 깊이의 물리적 결과.

  여러 큐프레이트 계열에서 일관:
    Bi계, Tl계, Hg계 모두 n_L=3에서 최대 Tc.

  Grade: CLOSE
  확립된 실험 사실. n/φ(6)=3과 정확 일치.
  물리적 원인은 도핑 물리이지만, 수치 일치는 강건.
```

### H-SC-06: 쿠퍼쌍 = φ(6)=2 전자 페어링 + 자속 양자 φ₀=h/(2e)

> 초전도의 기본 단위인 쿠퍼쌍이 φ(6)=2개 전자 결합.
> 자속 양자 Φ₀=h/(2e)의 분모 2e도 동일한 쿠퍼쌍 전하.

```
  쿠퍼쌍 (Cooper pair, BCS 1957):
    전자 2개가 포논 매개 인력으로 결합
    스핀: ↑↓ (singlet, S=0)
    총 전하: -2e, 총 스핀: 0
    보손으로 행동 → BEC 유사 응축

  자속 양자:
    Φ₀ = h/(2e) = 2.068 × 10⁻¹⁵ Wb
    분모 2e = 쿠퍼쌍 전하 (Little-Parks 실험 1962 확인)

  φ(6) = 2 ✓ (쿠퍼쌍 전자 수 = 자속 양자 분모)

  물리적 근거:
    왜 2인가? 페르미온 2개 → 보손 (정수 스핀 필요).
    3전자 결합: 에너지적으로 불리 (3체 문제 + Pauli 배제).
    2는 페르미온→보손 변환의 최소 단위.

  정직한 제한:
    "2"는 물리에서 가장 흔한 작은 수.
    φ(6)=2, λ(6)=2 등 여러 n=6 함수가 2를 줌.
    특이성(specificity) 낮음.

  v1에서 H-SC-1과 H-SC-18을 통합 (본질적으로 같은 관찰).

  Grade: CLOSE
  쿠퍼쌍 2전자는 BCS 이론의 핵심이며 φ(6)=2와 일치.
  물리적으로 확립되었으나, "2"의 특이성이 낮아 EXACT는 불가.
```

### H-SC-07: WHH 이론 계수 ln(2) = ln(φ(6))

> 상부 임계장 Hc2(0) 공식의 계수 0.6932 = ln2 = ln(φ(6))

```
  WHH 이론 (Werthamer-Helfand-Hohenberg, 1966):
    Hc2(0) = -0.6932 × Tc × (dHc2/dT)|_{Tc}
    계수 0.6932 = ln(2) (정확한 해석적 결과)

  ln(2) = ln(φ(6)) ✓ (수학적으로 정확)

  물리적 근거:
    이 계수는 Gor'kov 방정식의 선형화에서 유래.
    digamma function ψ(1/2) = -γ - ln(4)의 결과.
    clean limit에서의 정확한 해석적 결과.

    dirty limit: 0.693이 아닌 0.69 계열 (궤도쌍깨짐에 따라 변동).
    스핀-궤도 결합 포함 시 Maki 파라미터로 보정.

  정직한 제한:
    ln(2)는 수학에서 가장 흔한 상수 중 하나.
    정보 이론(1 bit), 반감기, 열역학 등 모든 곳에 등장.
    φ(6)=2와의 연결은 형식적으로 정확하지만 고유성 부족.

  Grade: CLOSE
  수학적으로 정확한 일치. BCS 이론의 해석적 결과.
  ln(2)의 보편성 때문에 EXACT로 승격 불가.
```

### H-SC-08: 초전도 전이의 4대 실험적 징표 = τ(6)

> 초전도 전이에서 동시에 관측되는 4가지 기본 현상

```
  초전도 전이의 4대 특성 (표준 교과서 분류):
    1. 전기 저항 → 0 (제로 저항)
    2. 자기장 배제 (마이스너 효과, 1933)
    3. 비열 불연속 (BCS 점프, ΔC/γTc = 1.43)
    4. 에너지 갭 형성 (2Δ(0)/kTc = 3.528)

  τ(6) = 4 ✓

  물리적 근거:
    Tinkham Ch. 1-3, Ashcroft & Mermin Ch. 34 표준 분류.
    처음 2개(제로 저항, 마이스너)가 정의적 특성.
    나머지 2개(비열 점프, 갭 형성)가 BCS 이론 예측.

  추가 현상:
    열전도도 변화, 초음파 흡수 변화, NMR 이완율 변화 등도 존재.
    그러나 "4대 기본 징표"는 교과서에서 일관된 분류.

  Grade: CLOSE
  교과서 표준 분류 4가지 = τ(6). 분류가 안정적.
  다만 추가 현상을 포함하면 변동 가능 → EXACT는 불가.
```

### H-SC-09: 3대 거시적 양자 효과 = n/φ(6)

> 초전도의 거시적 양자 현상이 정확히 3가지

```
  거시적 양자 효과 (표준 분류):
    1. 자속 양자화 (Φ = nΦ₀)
       — 거시적 파동함수의 단일값 조건에서 유래
    2. 조셉슨 효과 (약결합 통한 위상 결합)
       — 거시적 위상 차이 → 터널 초전류
    3. 마이스너 효과 (거시적 반자성)
       — |Ψ|²에서 런던 방정식 유도

  n/φ(6) = 3 ✓

  물리적 근거:
    세 현상 모두 거시적 파동함수 Ψ = |Ψ|e^{iθ}의 결과:
    - |Ψ|² → 마이스너 효과 (런던 방정식)
    - arg(Ψ) → 자속 양자화 (단일값 조건)
    - Δarg(Ψ) → 조셉슨 효과 (약결합 위상차)

    Tinkham, Rose-Innes & Rhoderick 표준 분류.

  Grade: CLOSE
  물리적으로 근거 있는 표준 분류. n/φ(6)=3 정확 일치.
```

### H-SC-10: Nb₃Sn Tc = 18K — 여러 n=6 표현의 수렴

> Nb₃Sn의 Tc=18.3K가 3n=18, σ+n=18 등 복수의 n=6 경로로 표현

```
  Nb₃Sn Tc = 18.3 K (실험값)

  n=6 표현:
    3 × n = 3 × 6 = 18 (1.7% 오차)
    σ + n = 12 + 6 = 18 (1.7% 오차)
    n × n/φ = 6 × 3 = 18 (1.7% 오차)

  물리적 근거:
    Tc는 Allen-Dynes 공식에서 결정:
    ω_log, λ (전자-포논 결합 상수), μ* (쿨롱 의사 포텐셜).
    Nb₃Sn: λ ≈ 1.8, ω_log ≈ 7-8 meV.

  정직한 제한:
    18은 비교적 흔한 수이며 여러 수식으로 표현 가능.
    단독으로는 WEAK이지만, H-SC-03의 삼중 일치 맥락에서 의미.

  Grade: WEAK (단독) / H-SC-03과 합산 시 CLOSE
  Tc 단독 일치는 우연 가능성이 높으나, 삼중 맥락에서 의미.
```

### H-SC-11: 초전도 큐빗 3유형 — charge, flux, phase = n/φ

> 초전도 양자 큐빗의 기본 3유형이 조셉슨 접합의 3자유도에서 유래

```
  기본 3유형 (Devoret & Schoelkopf, Science 2013):
    1. Charge qubit (Cooper pair box) — E_C 지배
    2. Flux qubit (초전도 루프) — E_L 지배
    3. Phase qubit (전류 바이어스) — E_J 지배

  n/φ(6) = 3 ✓

  물리적 근거:
    3유형은 조셉슨 접합 회로의 3가지 에너지 스케일에 대응:
    E_C (charging), E_J (Josephson), E_L (inductive).
    각 에너지가 지배할 때 다른 큐빗 유형 출현.

    현대 큐빗 (transmon, fluxonium, 0-π qubit)은 이 3유형의 파생.
    Clarke & Wilhelm, Nature 2008: 3유형 분류 표준.

    s-wave: 등방적 포논 매개 결합의 자연적 결과

    l=2 선택은 결정 대칭과 자기 상호작용에서 결정.

  Grade: WEAK
  l={0,2}에서 2=φ(6)는 일치하나, l=0은 자명.
  궤도 양자수 분류는 구면조화함수의 일반 체계.
```
