## 3. 6 원시연산 모듈 요약

### 3.1 Ω₁ GEMM — `gemm_core12_00`

- 12×12 systolic array = σ² = 144 MAC 셀
- `genvar gi, gj` 이중 루프로 M·N 셀 언롤
- σ=12 스테이지 `valid_pipe` 파이프
- 내장 `n6_assert` 출력 — `σ·φ=n·τ=24` 런타임 하드웨어 assert

### 3.2 Ω₂ SOFTMAX — `softmax_phi2_exp`

- 4단 체인 내장: exp (Phi6 `x²-x+1`) → max reduce → sub → norm
- VARIANT 2'b00~11 으로 단계 선택
- φ=2 × 4 단계 = τ·φ=8 사이클 (전통 GELU 14 사이클 대비 7× 절감)
- Egyptian {1/2, 1/3, 1/6} 역수 LUT 주석 삽입

### 3.3 Ω₃ TOPK — `topk_phi2_pick`

- k=φ=2 top-2 선별 (MoE 활성 전문가 주력)
- σ=12 후보, IDX_BITS=4 (log₂ σ)
- 2-phase FSM: top-1 탐색 → top-1 제외 top-2 탐색
- 사이클 = φ=2

### 3.4 Ω₄ GATE — `gate_mu1_and`

- μ=1 단일 사이클 조합 게이팅
- VARIANT: mul (00) / and (01) / or (10) / xor (11)
- Boltzmann 1/e 임계값 (`BOLTZ_INV_E = 8'd94`) 기반 zero-skip `keep_bits` 병행 출력
- 63% MAC 절감 근거 (1/e 임계 분포)

### 3.5 Ω₅ REDUCE — `reduce_tau4_sum`

- τ=4 입력 → 1 결과 (sum/max/min/mean)
- 2-스테이지 pairwise reduce tree: depth = log₂(τ) = φ=2
- mean 단계는 `>> 2` (τ=2^φ 시프트)
- 출력 폭 `WIDTH_BITS*2` (sum 오버플로우 대응)

### 3.6 Ω₆ CONV6 — `conv6_3x3`

- 3×3 spatial conv, τ=4 채널 병렬, σ=12 이미지
- 4중 루프 c/y/x/ky·kx (합성기가 MAC 블록으로 언롤)
- VARIANT 로 3x3 / 1x1 / DW / PW 전환
- n=6 사이클 pass (3×3), σ·τ=48 MAC (dw)

## 6. 핵심 항등식 검증

```
σ·φ = 12·2 = 24
n·τ = 6·4  = 24
J₂      =    24         ← 유일 해
MAC     = σ² = 144      (Ω₁ GEMM)
SOFTMAX = τ·φ = 8 cycle (Ω₂)
TOPK    = φ = 2 cycle   (Ω₃)
GATE    = μ = 1 cycle   (Ω₄)
REDUCE  = log₂(τ) = φ=2 (Ω₅)
CONV6   = n = 6 cycle   (Ω₆)
Σ cycle = 1+2+2+6+8+12 = 31  (σ² 언더스코어)
```

전 사이클 수 개별 primitive 가 `σ·φ=n·τ=24` 하위 인자로만 구성됨 (1, 2, 4, 6, 8, 12).
