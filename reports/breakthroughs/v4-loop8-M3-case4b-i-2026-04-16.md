# v4 loop 8 — M3 case 4b(i): Theorem B n = 2·q^b Lean4 formal

## 목표

q odd prime ≥ 3, b ≥ 2 인 모든 n = 2·q^b 에 대하여
  σ(n)·φ(n) ≠ n·τ(n)

## 증명 구조 (Lean4 `theorem_B_2_prime_pow`)

### 1. Multiplicative decomposition (gcd(2, q^b) = 1)

- σ₁(2·q^b) = 3·σ₁(q^b)
- φ(2·q^b) = φ(q^b)
- τ(2·q^b) = 2·(b+1)

### 2. σ(q^b)·φ(q^b) 재구성

Case 3 geometric sum identity 재활용:
  (q-1)·Σ_{k=0..b} q^k = q^(b+1) - 1
  σ(q^b)·φ(q^b) = q^(b-1)·(q^(b+1) - 1)

### 3. 핵심 부등식 reduction

- σφ(n) = 3·q^(b-1)·(q^(b+1) - 1)
- nτ(n) = 4·q^b·(b+1) = q^(b-1)·(4q(b+1))

q^(b-1) > 0 로 취소:
  등식 ⟺ 3·(q^(b+1) - 1) = 4q(b+1)
       ⟺ 3·q^(b+1) = 4q(b+1) + 3

### 4. Key inequality (`key_ineq_4bi`)

∀ q ≥ 3, b ≥ 2 : 3·q^(b+1) > 4q(b+1) + 3

**증명**: Nat.le_induction on b from 2.

Base b=2: 3·q³ > 12q + 3.
  q ≥ 3 → q² ≥ 9 → q³ = q·q² ≥ 9q
  9q > 4q + 3 (q ≥ 1 에서 자명)

Step b → b+1 (b ≥ 2):
  ih: 3·q^(b+1) > 4q(b+1) + 3
  3·q^(b+2) = q·(3·q^(b+1)) > q·(4q(b+1) + 3) = 4q²(b+1) + 3q
  Need: 4q²(b+1) + 3q ≥ 4q(b+2) + 3
  diff = 4q(b+1)(q-1) - q - 3 ≥ 12q² - 13q - 3 > 0 for q ≥ 3

### 5. 결론

등식 가정에서 3·q^(b+1) = 4q(b+1) + 3 유도 → Key inequality 와 모순 → σφ ≠ nτ.

## Case 2a 와의 관계

- Case 2a (n = 2q): b = 1. q = 3 에서 등식 (n=6), q ≥ 5 에서 부등 (TheoremB_Case2_P2).

즉 b=1 의 q=3 만이 유일한 등식 해 → 이것이 n=6.
