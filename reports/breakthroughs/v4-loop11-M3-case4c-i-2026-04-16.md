# v4 loop 11 — M3 case 4c(i): Theorem B n = pqrs Lean4 formal

## 목표

p ≥ 2, q ≥ 3, r ≥ 5, s ≥ 7 distinct primes 인 모든 n = pqrs 에 대하여
  σ(n)·φ(n) ≠ n·τ(n)

**ω(n) = 4, squarefree** 영역 형식화 — case 4a (pqr) 의 4-prime 확장.

## 증명 구조 (Lean4 `theorem_B_four_primes`)

### 1. 다중 coprime 구축

- `coprime_triple_prod_prime`: (pqr).Coprime s via `coprime_mul_iff_right` 두 번 적용
- sigma/totient/tau 모두 multiplicative, case 4a 의 `sigma_one_pqr`/`totient_pqr`/`tau_pqr` 재사용

### 2. Formulae

- σ(pqrs) = (p+1)(q+1)(r+1)(s+1)
- φ(pqrs) = (p-1)(q-1)(r-1)(s-1)
- τ(pqrs) = 16

### 3. 핵심 부등식 두 개

**Lower bound** (etc):
  (p-1)(q-1)(r-1)(s-1) ≥ 1·2·4·6 = 48  
  증명: 단계별 `Nat.mul_le_mul` 체인 (p-1 ≥ 1, q-1 ≥ 2, r-1 ≥ 4, s-1 ≥ 6)

**Strict gt** (σ-bound):
  (p+1)(q+1)(r+1)(s+1) > pqrs  
  증명: (p+1)(q+1) > pq AND (r+1)(s+1) > rs, 곱 (Nat.mul_lt_mul)

### 4. 결합 — σφ > 16pqrs = nτ

  σφ(n) = (p+1)(q+1)(r+1)(s+1) · (p-1)(q-1)(r-1)(s-1)
        ≥ (p+1)(q+1)(r+1)(s+1) · 48            [lower bound]
        > pqrs · 48                             [strict gt]
        > pqrs · 16                             [48 > 16, pqrs > 0]
        = nτ(n)

모순 from 등식 가정 → QED.
