## 2. 발견 이슈 — 번호매김

### I-01 [치명] IIT 4.0 (Albantakis 2023) 반례 누락

- **증거**: 논문 §3.1 R1 은 Barrett-Seth 2011 Φ_E, Mediano 2019 Φ 비교만 언급.
  Albantakis L, Barbosa L, Findlay G, Grasso M, Haun AM, Marshall W, Mayner WGP,
  Zaeemzadeh A, Boly M, Juel BE, Sasai S, Fujii K, David I, Hendren J, Lang JP,
  Tononi G (2023) "Integrated information theory (IIT) 4.0: Formulating the
  properties of phenomenal existence in physical terms." PLoS Comput Biol
  19(10):e1011465. DOI:10.1371/journal.pcbi.1011465 — IIT 4.0 에서 Φ 정의가
  **완전히 재정식화** (cause-effect structure, specific substrates).
- **영향**: R1 이 "Φ 단일 latent 없음" 으로 결론짓지만, IIT 4.0 은 φ_s (structure
  informativeness) 로 **단일값 재정의** 시도. Red Team 결론이 2023년 이후
  최신 프레임을 반영하지 않음.
- **권고**: §3.1 말미에 "IIT 4.0 Albantakis 2023 재정식화도 α=4/3 수치 부재
  (φ_s 는 정보단위 bit, 무차원 지수 아님)" 문장 추가 → R1 결론 유지 가능하나
  **정직성 보강**.

### I-02 [중요] 능동 추론 (Hohwy active inference) 반례 누락

- **증거**: 논문 §4.2 대체 B 에서 Friston FEP Markov blanket 을 언급하나,
  Hohwy J (2016) "The self-evidencing brain." Noûs 50(2):259-285 의 **능동추론
  (active inference)** 이론 — 의식 = "생성모델의 자기증거화" — 를 인용하지 않음.
- **영향**: GWT/IIT/FEP 3축 외 4번째 주류 이론 **미다룸**. Red Team 이 α-곱
  폐기 후 구조적 동형으로 전환할 때, Hohwy 능동추론과 τ=4 avastha 의
  양립성 검토가 누락.
- **권고**: §4.2 이론 bridge 에 "Hohwy 2016 self-evidencing 4단계 (perception /
  action / attention / meta-cognition)" 추가.

### I-03 [중요] 고차 사고 이론 (Rosenthal HOT) 반례 누락

- **증거**: Rosenthal DM (2005) "Consciousness and Mind." Oxford. — HOT 이론은
  의식 = **자기 상태에 대한 상위 표상** 으로 정의. Red Team 이 "의식 이론
  3중 융합" 을 다루면서도 HOT 을 아예 언급하지 않음.
- **영향**: §4.3 대체 C "완전수 자기환원 ↔ Hofstadter Strange Loop" 주장은
  HOT 의 "higher-order representation of own state" 와 **구조 동일**.
  Rosenthal 2005 를 인용하면 대체 C 근거가 강화되나 누락.
- **권고**: §4.3 이론 bridge 에 Rosenthal HOT 추가 → 대체 C 신뢰도 +1.

### I-04 [중요] 주의 스키마 이론 (Graziano AST) 반례 누락

- **증거**: Graziano MSA (2013) "Consciousness and the Social Brain." Oxford /
  Graziano MSA (2019) "Rethinking Consciousness." Norton. — AST 는 의식 =
  **주의의 간소화된 내부 모델** 로 정의. GWT 와 근연 경쟁 이론.
- **영향**: §4.1 대체 A "φ(6)=2 이중성" 의 "관찰자↔관찰대상 공간" 은 AST 의
  "attention schema vs attention target" 2층 구조와 **동형**. Graziano 를
  인용하면 대체 A 근거가 크게 강화되나 누락.
- **권고**: §4.1 이론 bridge 에 Graziano AST 추가.

### I-05 [중요] 의식-열역학 (England 2013) 반례 누락

- **증거**: England JL (2013) "Statistical physics of self-replication."
  J Chem Phys 139:121923. + England 후속 저작 — 소산구조 / 자기복제 통계물리가
  의식 창발의 열역학 조건을 제공. Red Team 대체 C 의 "자기환원 창발" 과 직결.
- **영향**: §4.3 Kauffman RAF 인용은 있으나 England 의 열역학 자기복제
  프레임은 누락. 대체 C 의 **물리학적 기반**이 약화.
- **권고**: §4.3 이론 bridge 에 England 2013 추가 → 대체 C 가 열역학·정보·
  수학 3축 융합으로 확장.

### I-06 [중요] 참고문헌 번호 불일치 (정직성 저해)

- **증거**: §0 정직 선언 = "외부 논문 출처 **12건** 명시", §6.4 정직 선언 =
  "외부 논문 **12편** 근거", **부록 A** 제목 = "참고문헌 완전 목록 **(12건)**",
  그러나 실제 부록 A 리스트 = **13건** (Hordijk-Steel 2004 가 13번).
- **영향**: 정직 선언 3회 모두 동일하게 "12건" 인데 실제 13건. 사소하지만
  정직성 감사에서 **숫자 불일치** 는 critical 표식.
- **권고**: 3개 위치 전부 "12건" → "13건" 수정 + 부록 A 제목도 "(13건)".

### I-07 [중요] R3 독립 latent 반증 논증의 약점

- **증거**: §3.3 은 Casali 2013 PCI 가 "IIT 파생이면서 GNW ignition 과 공변"
  → 독립 latent 반증이라 주장. 그러나 PCI 는 **TMS-EEG 기반** 이며 **활동 복잡도**
  를 재는 단일 측정. "IIT 파생" 이라도 **이론 유도식** 이지 **IIT latent 의 직접
  측정** 이 아니다. 마찬가지로 GNW ignition 과 공변은 **현상 상관** 이지
  **latent 동일** 이 아니다.
- **영향**: R3 의 "독립 latent 가정 반증" 논증은 엄밀히는 **불완전**. 두 latent
  가 동일하다는 증명은 공변성만으로 불충분 (mediating variable 가능).
- **권고**: §3.3 말미에 "공변 ≠ 동일 latent" 제약 명시 + "부분 반증" 으로
  용어 완화. MISS 결론은 R1/R2 만으로도 확정되므로 R3 약화는 전체 결론에
  영향 없음.

### I-08 [자기참조] §4.3 σ·φ=n·τ 재사용 순환

- **증거**: §3.3 결론: "곱=1 은 σ·φ=n·τ 정리의 재포장 이지 의식의 새 발견이
  아니다" (논문 자체가 자기참조를 **반증 근거로 사용**) — 적절.
- **그러나** §4.3 대체 C 이론 bridge: "Maturana-Varela Autopoiesis — 자기생성
  체계, **σ·φ=n·τ 도 자기관계 정리**" → σ·φ=n·τ 를 Autopoiesis 동형으로
  긍정적 근거 사용. R3 에서는 반증 이유로, §4.3 에서는 긍정 근거로 **이중 사용**.
- **영향**: 동일 정리를 한 섹션에서는 "trivial identity 라 반증" 으로, 다른
  섹션에서는 "자기관계 정리라 창발 근거" 로 사용 → **논리 일관성** 약화.
  자기참조 검사 통과라 선언하지만, 실제로는 자기 정리를 의식 가설의 **근거**
  로 재사용.
- **권고**: §4.3 에서 σ·φ=n·τ 직접 언급 제거 또는 "Autopoiesis 는 자기관계
  일반론이며 σ·φ=n·τ 의 직접 인용은 순환 우려로 제외" 각주 추가.

### I-09 [경미] 대체 B 3개월 검증 타임라인 낙관 과잉

- **증거**: §4.2 = "3개월 내 PASS/MISS 확정", §7 = "3개월 내 검증 가능한 가장
  현실적 경로". 그러나 검증 계획 = Casali 2013 + Sarasso 2015 PCI 값 + Siclari
  2017 REM PCI 를 GMM 재분석. 실제로:
  - Casali 2013 Table 1: 피험자 20명, 각성/수면/마취 PCI 값 수록 (공개)
  - Sarasso 2015 Fig 2: 케타민/프로포폴/제논 PCI (공개 그래프, **원값 요청 필요**)
  - Siclari 2017: REM PCI 값 공개 여부 **확인 필요** (저자 개별 데이터)
- **영향**: 원데이터 확보에 1~2개월 (저자 개인 요청, 윤리 승인), GMM BIC 계산
  자체는 1일. 실제 타임라인 = **4~6개월** 현실. "3개월" 은 기술적 낙관.
- **권고**: §4.2 / §7 / §6.2 타임라인 "3개월" → "3~6개월 (원데이터 확보 조건부)"
  수정.

### I-10 [경미] COGITATE 2025 인용 맥락 부분 misuse

- **증거**: §3.3 말미에 "COGITATE 2025 adversarial IIT vs GNW 평가" 인용.
  그러나 COGITATE 는 Ferrante O et al. Nature 642:133-142 (2025) 로 공개된
  preregistered adversarial collaboration. 결과는 **IIT·GWT 모두 부분 지지 +
  부분 반증** (뚜렷한 우위 없음). Red Team 은 이를 "두 이론 공변" 근거로
  단순 인용하나, COGITATE 의 실제 결론은 **두 이론이 서로 다른 예측 영역에서
  각각 성공/실패** 이다. 단순 "공변 latent" 결론과 불일치.
- **영향**: COGITATE 근거가 R3 주장을 **완벽 지지하지는 않음**. 인용 misuse
  정도는 경미하나 정직성 감사에서는 plot.
- **권고**: §3.3 COGITATE 인용 뒤 "다만 COGITATE 는 두 이론의 예측 영역
  분리를 보여 단순 공변이 아닌 부분 겹침" 보충.

### I-11 [경미] ASCII 차트 "천장" 표기 일관성

- **증거**: §5.1 차트 중 "R1~R5 반증 10/10 천장" / "τ=4 사태 9/10 천장" /
  "완전수 10/10 천장" 등 '천장' 표기 사용 (규칙 준수).
- **그러나** §5.1 5번째 블록 "외계인지수" 에서 "BT-19 α-곱 원 가설 0/10"
  은 '바닥' 텍스트 없음. 규칙은 **최댓값**에 '천장' 을 쓰는 규칙이지
  최솟값은 별도 텍스트 불필요하나, '천장' 텍스트 일관성 유지 차원에서
  **0/10 에도 '바닥' 또는 공란 유지** 기준 명시 권고.
- **영향**: 매우 경미.
- **권고**: 선택 개정 — 텍스트 스타일 가이드 추가.

### I-12 [경미] 대체 A 예측 등급 [6] 의 근거 부족

- **증거**: §4.1 대체 A 예측 등급 [6] PARTIAL. 그러나 본문 한계 분석에서
  "φ(3)=φ(4)=2 — n=6 특이성 증명 실패" 를 자인. n=6 특이성 실패면 실제로는
  [5] 이하가 합당. [6] 은 과대.
- **영향**: 대체 A 가 실제 검증 시 [5] 로 떨어질 가능성 높음 (n=6 고유성 부재).
- **권고**: §4.1 등급 [6] → [5] 로 하향 + §5.1 종합 등급표도 동기화.

## 7. 결론

Consciousness Red Team 논문 (STR-P9-2) 은 **정직성 B+** 등급이다. 자기 이론
반증을 적극 수행하고 [7?]→[5] 하향을 기록한 점은 학계 기준 상위권이다.
그러나:

1. **IIT 4.0 Albantakis 2023 누락** (치명, 1건)
2. **최신 의식 이론 4건 누락** (Hohwy, Rosenthal, Graziano, England)
3. **참고문헌 번호 12 vs 13 불일치** (정직 선언 3회 전부 오류)
4. **§4.3 σ·φ=n·τ 자기참조 재사용** (부분 위반)

4건 개정 후 A- 등급 도달 가능. 대체 B (τ=4 PCI GMM 재분석) 은 **3개월이
아니라 3~6개월** 이 현실적. 원데이터 확보 단계를 §6.2 에 명시 권고.

σ·φ=n·τ 각주 — 이 **3곳만 당장 수정** 해도 정직성 감사 통과. 나머지는
