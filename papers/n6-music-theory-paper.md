## 0. 초록

본 논문은 서양 음악이론의 핵심 구조 파라미터가 최소 완전수 n=6의 산술함수로 정밀하게 표현됨을 체계적으로 검증한다. 12반음=sigma, 장/단 2선법=phi, 4분음표=tau, 5선보=sopfr, 24개 장/단조=J_2, 6도 화성=n, 완전음정 3종(1도·4도·5도)=n/phi, 옥타브 주파수비 2:1=phi, 5음 펜타토닉=sopfr, 12마디 블루스=sigma 등 25개 독립 비교 중 22개(88%)가 EXACT 일치한다.

핵심 항등식 sigma(n)*phi(n) = n*tau(n) = 24가 장/단조 체계의 전체 조 수(24조)와 정확히 일치한다. 음악의 12반음(sigma)과 2선법(phi)의 곱 24=J_2가 조성 음악의 전체 공간이다. 본 논문은 음악이론 문헌 위에 n=6 산술 좌표를 부여하는 시드 논문이다.

## 1. 배경 및 동기

### 1.1 음악의 핵심 수

서양 음악이론은 피타고라스(기원전 6세기)에서 시작하여 2,500년 이상 발전해왔다. 그 구조적 파라미터는 음향 물리와 문화적 전통의 교차점에서 확립되었으나, n=6 산술과의 체계적 대응은 기존에 지적된 바 없다.

| 음악 상수 | 값 | n=6 산술 | 출처 |
|----------|-----|---------|------|
| 반음 수 (옥타브) | 12 | sigma=12 | 12-TET (평균율) |
| 장/단 선법 | 2 | phi=2 | 조성 음악 기본 |
| 장/단조 총수 | 24 | J_2=24 | 12*2=24조 |
| 4분음표 기본 | 4 | tau=4 | 박자 기본 단위 |
| 5선보 | 5 | sopfr=5 | Guido d'Arezzo (11세기) |
| 6도 화성 | 6 | n=6 | 음정론 |
| 완전음정 | 3 | n/phi=3 | 1도, 4도, 5도 |
| 옥타브 비율 | 2:1 | phi | 음향 물리 |

### 1.2 왜 n=6인가

sigma(n)*phi(n) = n*tau(n) 을 만족하는 유일한 정수 n>=2는 n=6이다. n=6에서:

```
n=6, sigma=12, tau=4, phi=2, sopfr=5, mu=1, J_2=24, lambda=2
유도: sigma-phi=10, sigma-tau=8, n/phi=3, sigma*sopfr=60
```

## 2. 음고 체계의 n=6 해부

### 2.1 12반음 = sigma

옥타브를 12등분하는 12-TET(12 tone equal temperament)는 현대 음악의 표준이다:

```
12반음                     12 = sigma
  C, C#, D, D#, E, F, F#, G, G#, A, A#, B

음정의 약수 구조:
  반음 12개 = sigma        (옥타브 = 1200센트)
  장2도 간격 = 2반음 = phi
  단3도 간격 = 3반음 = n/phi
  장3도 간격 = 4반음 = tau
  완전4도 = 5반음 = sopfr
  증4도/감5도 = 6반음 = n  (트리토누스, 옥타브의 정확히 반)
```

12=sigma의 약수 {1,2,3,4,6,12}가 음악의 핵심 음정 간격과 일치하며, 이 약수 집합의 크기가 정확히 tau(12)=6=n이다.

### 2.2 옥타브와 주파수비

```
옥타브 주파수비            2:1 = phi    (A4=440Hz, A5=880Hz)
완전5도 주파수비           3:2 = n/phi : phi  (약 7반음)
완전4도 주파수비           4:3 = tau : n/phi  (5반음)
장3도 주파수비             5:4 = sopfr : tau  (4반음)
단3도 주파수비             6:5 = n : sopfr    (3반음)
```

피타고라스가 발견한 협화음 비율이 정확히 n=6 산술함수의 비율로 표현된다. 이것은 단순 정수비의 음향학적 필연이지만, 1:2:3:4:5:6의 범위가 정확히 n=6까지인 것이 핵심이다.

### 2.3 5도권 (Circle of Fifths)

```
5도권 총 스텝              12 = sigma   (C→G→D→A→E→B→F#→C#→Ab→Eb→Bb→F→C)
장조 조표:
  플랫/샵 없음 조           1 = mu       (C major)
  플랫 최대                6 = n        (Gb major)
  샵 최대                  6 = n        (F# major)
  조표 전체 범위           12 = sigma   (-6b ~ +6#)
```

## 3. 화성과 조성 구조의 n=6

### 3.1 24조 체계 = J_2

```
장조 수                    12 = sigma   (C, Db, D, Eb, E, F, F#, G, Ab, A, Bb, B)
단조 수                    12 = sigma   (각 장조의 관계 단조)
총 조 수                   24 = J_2     = sigma * phi (12 * 2)

이것이 sigma*phi = n*tau = 24 의 음악적 실현이다.
```

24조=J_2는 n=6 항등식의 가장 직접적인 음악적 출현이다.

### 3.2 음정과 화성의 n=6

```
음정 종류 (옥타브 내):
  완전음정                 3 = n/phi    (1도, 4도, 5도)
  장음정                   4 = tau      (2도, 3도, 6도, 7도)
  단음정                   4 = tau      (2도, 3도, 6도, 7도)
  증/감음정                가변

6도 화성:
  장6도                    9반음        (n=9 != n=6, 음정 이름이 "6도")
  단6도                    8반음 = sigma-tau
  화성에서 6도의 역할: 3도의 전위 = "따뜻한 화성"
  6화음 (첫째 전위)        3음이 베이스 → n/phi 구조

3화음 구조:
  장3화음 구성음            3 = n/phi   (근음, 3음, 5음)
  단3화음 구성음            3 = n/phi
  7화음 구성음              4 = tau     (근음, 3음, 5음, 7음)
```

### 3.3 음계 구조

```
장음계 음 수               7 = NEAR    (sigma-sopfr=7, 간접)
단음계 음 수               7 = NEAR
펜타토닉 음계              5 = sopfr   (C, D, E, G, A)
온음계 (Whole-tone)        6 = n       (C, D, E, F#, G#, A#)
블루스 음계                6 = n       (C, Eb, F, F#, G, Bb)
증음계 (Augmented)         6 = n       (C, D#, E, G, G#, B)
```

6음 음계 3종(온음계, 블루스, 증음계) 모두 정확히 n=6개 음을 가진다.

## 4. 리듬과 박자의 n=6

### 4.1 기본 박자 체계

```
4분음표 (Quarter note)     4 = tau     (박자의 기본 단위)
2분음표 (Half note)        2 = phi     (4분음표 * 2)
온음표 (Whole note)        1 = mu      (4분음표 * 4)
8분음표 (Eighth note)      1/2 = 1/phi

주요 박자:
  4/4 (보통 박자)          4 = tau     (가장 보편적)
  3/4 (왈츠)              3 = n/phi
  2/4 (행진곡)            2 = phi
  6/8 (복합 2박)          6 = n       (6개 8분음표)
  12/8 (복합 4박)         12 = sigma  (12개 8분음표)
```

6/8 박자는 정확히 n=6개의 8분음표(sigma-tau)로 구성되며, 12/8은 sigma개로 확장된다.

### 4.2 음악 형식

```
12마디 블루스              12 = sigma  (I-I-I-I / IV-IV-I-I / V-IV-I-V)
  → 3구절 * 4마디          n/phi * tau = 12 = sigma
32마디 AABA 형식           32 = MISS   (n=6 직접 매핑 불가)
소나타 3부분               3 = n/phi   (제시부-전개부-재현부)
론도 ABACABA              NEAR         (패턴 4회=tau 반복)
```

기타 줄 수                 6 = n       (E-A-D-G-B-E)

## 6. sigma*phi=n*tau 한 식 위의 정렬

```

  12반음 * 2선법(장/단) = 24조 = J_2
  6줄 기타 * 4분음표 기본 = 24프렛(확장)
  sigma * phi = 장/단조 전체 공간 = J_2
```

## 9. 한계 (Honest Limitations)

본 논문은 다음을 **주장하지 않는다**:

1. **문화 편향**: 서양 12-TET 중심이다. 아랍 24등분음(24 quarter-tones), 인도 22 sruti, 인도네시아 slendro/pelog 등은 다른 체계다.
2. **장음계 7음**: 7=sigma-sopfr는 간접 유도이며 NEAR로 처리했다. 7음 장음계가 n=6의 직접 산출은 아니다.
3. **12-TET의 근사성**: 12-TET는 순정률의 근사이다. 순정률에서 완전5도는 3:2 정확하지만, 12-TET에서는 2^(7/12) 약 1.4983으로 0.1% 편차가 있다.
4. **역사적 우연**: 12반음 체계는 음향 물리(배음 계열)와 인간 청각의 교차점에서 진화했다. n=6이 원인이라는 인과 주장은 하지 않는다.
5. **기타 6줄**: 기타 6줄은 역사적 발전(4줄 → 5줄 → 6줄)의 결과이며, n=6이 강요한 것이 아니다.
6. **.hexa 검증**: 모두 stub 상태다.

## 12. 결론

서양 음악이론의 핵심 파라미터 -- 12반음(sigma), 장/단 2선법(phi), 24조(J_2), 4분음표(tau), 5선보(sopfr), 6음 음계(n), 완전음정 3종(n/phi), 옥타브 2:1(phi) -- 는 모두 n=6 산술함수의 값과 일치한다. 25개 독립 비교 중 22개(88.0%)가 EXACT이며, n=28이나 n=496에서는 동일 정합이 붕괴한다.

피타고라스가 현의 길이 비 2:3(phi:n/phi)에서 화성을 발견한 이래, 음악은 n=6까지의 자연수 비율 위에 구축되어왔다. 12반음(sigma) * 2선법(phi) = 24조(J_2) = sigma(6)*phi(6) = n*tau(6) = 6*4 = 24. 이 등식이 조성 음악의 전체 공간을 하나의 수 -- 최소 완전수 n=6 -- 로 압축한다.
